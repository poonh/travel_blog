3.囲爐裡慶聖誕 - 遊記

日期：2021年12月17-18日

地點：下蒲刈島（梶が浜的囲爐裏），上蒲刈島（七國見山），大崎下島（御手洗）

引言：今次用的是廣島人專用優惠やっぱ広島じゃ割。每人付10325日元，可以住梶ヶ浜コテージ，帶導遊的自行車服務及出張料理人的本格囲爐裏飯。我們2020年初在這裏租自行車時我就知道這個コテージ，當時就很想玩囲爐裏。所以前幾個月，在這個計劃上來之前我就和raku老師說12月去梶ヶ浜コテージ玩囲爐裏，當時是想自己做飯，沒想到這個計劃上來了，就給了我們一個機會喫好玩好的。

行程開支：
 やっぱ広島じゃ割套餐：10325
車費：700
早餐等雜費：500
午飯：770
合共：~12000

 去咖啡店要走個小坡，車不能上的，上坡前的大路有停車空地，有兩個車位，從御手洗開車約20分鐘。

我們的午飯，770日元。

 

 

最後我在人家的日式園林拍照。我選擇來這裏喫飯的一個原因也是因爲環境和我的衣服配。

中午1點，我們就到梶ヶ浜的自行車店了。導遊已經準備好等我們，最後我們換衣服去廁所等搞到差不多1點半纔出發...

自行車店，上次招待我們的老爺爺也在，他是負責維修的。


我們的助動車，2020年入手，16萬一輛。

今天因爲大風，臨時改了線路，在下蒲刈島的橘子園之間騎，要爬坡，不過助動車給力。這條路線風景很贊，風也沒有海邊那麼大，而且路上完全沒車。

我們首先去了一個看羊（ 菅原オレンジ農場＆広島ミニヤギ牧場 ）的地方。

 

裏面大概有10只羊，還有快要當媽媽的，我用草餵它們。它們也不是什麼草都喫，喜歡喫大塊的。

 帥氣的導遊

來到這裏，我跟大家說一下蒲刈名字的由來。話說2000年前，公主在這裏出遊，在下圖，位於上蒲刈島，長滿蒲的山丟了梳子。爲了把梳子找回來，公主就下令把蒲割掉（"刈"是割掉的意思），所以島就叫蒲刈了。蒲刈的發音是かまがり。我之前一直都記不住，聽了這個故事後就記得了。

下面紅圈就是公主丟梳子的山

我們來看看路上風景照

深秋落葉

橘子園，騎在橘子樹下我多次想把橘子給順了...據導遊說，橘子園是100多年前開發的，現在如果是要回到野生，橘子會活不了的。另外，二戰前橘子是高檔水果，有錢人才喫得起。後來才大量種植，現在就成爲大衆水果了。島上的車路都是爲了運送橘子修的。

 


 

 

 翻過山後，我們就下來了，到了海濱路(天神鼻公園)。90年代政府想發展這裏旅遊業，修了很多地方，不過後來沒發展起來。

幾百年前，還沒有機動船的時候，下蒲刈島是旅客休息的地方，因爲這裏是最西側，外地的船都首先經過這裏。這一帶也很繁華。但後來有了機動船，旅客就去御手洗那較大的碼頭，這個島就慢慢衰落了。這裏幾個島，每個都有自己的故事的。

 

 最後，我們就去白雪樓喝抹茶了。400元門票，提供茶點及博物館參觀，但拿着廣島的學生證免費。

白雪樓原來是在京都的，後來搬了過來。日本建築很多木做，把木板拆了重建很容易，所以建築可以遷移。

我們快喫完的時候，Shino就開始洗碗。 他問我們喫不完的東西他幫我們處理還是我們自己處理，不過我們說我們能喫完，果然是的~東西的確挺多的，不過能喫完，但又不覺得撐。

今晚非常豐富，我們也喝酒了，特別滿足。各位晚安了~今次的旅行套餐，喫住玩我都很滿意，給個滿分。

海邊另一側是山景。


晨運完我們要出發登山去了~

 這次七國見山是我們登山旅程前所未有之旅，經歷過艱苦然後又平安無事，回頭一看，當時覺得很困難，過後又好小的事。

我們在蒲刈町ウオーキングセンター 出發，那裏有大概5個停車車位。當時除了我們還有一輛，不過路上見不到人。

當時センター裏面沒有職員，但廁所隨便上（注意，週一週二不開門的，想上廁所要開車10分鐘去縣民之濱）。

  大家先參觀一下里面。

 

非常難得這裏提供登山杖，贊一個！（提示：這個山用登山杖很有必要）

 

出發照，我們大概10點開始登山。

開始的路還挺容易的，而且有海景看，也有不同植物。

 

路上還有岩石景


 

很快我們就到了第一個景點 西泊公園，那是有1200年曆史，詳見此。


這裏有廁所，有水洗手，不過不能沖水，排出的東西是直接到底的（用過這種廁所的人就能明白我的意思）。

看完倒胃口的東西看看美景中和一下吧~

 

 

看完我們就繼續上路了~ 這裏開始就是比較容易走的樓梯了。

 

 很快我們就到第二個景點，這裏可以敲鐘。

我們到山頂了，耶！花了大概兩個多小時，比較慢的速度。

 

 

 我來給大家科普一下，爲什麼這山叫七國見山？因爲在古代這裏可以看到七個國，分別是 安芸、備後、讃岐、伊予、豊後、周防、長門，所以就叫七國見山。

 

 

這裏有個亭子，不過當時不開，我們就坐在石頭上喫午飯。



愛美的HY又抓緊機會補妝~

 

我們要下山了，沒想到這是困難的開始~

一開始也沒什麼的，路也好走。



 raku老師還在路上檢麻花喫。 

 

路上標誌也算清晰。樹上有標記。這個地方頭上好多烏鴉，離我們很近，我不敢抬頭，怕一抬頭就中招...這裏樹蔭還很多 ，多的把太陽遮掉了大半，我還以爲是陰天快下雨，其實不是。

 

 但過了不久，嚴峻的挑戰就開始了~

我們遇到了鎖場。

 

今天HY一直在前頭爲我們開路



 後來越來越困難，我放棄了，直接滑下去。
 

 

 

 

下午兩點半，我們去到一個走投無路的地方，那裏有很大的石礦場。

我們之前看到樹上的紅色標記，現在也找不到了。雖然下來的路很艱難，但我們還是決定遠路返回了。這個時候小Y的狀態不太好，臉色有點蒼白，但大家同心協力，克服困難，過了45分鐘，我們就返回原地，之後原路下山。這個時候小Y由走的最慢變了走的最快那位。

下山時竟然還看到富士山。

 

 經過平安鍾，我們每人都敲了一下以示平安。

 


 

我們目標是天黑前歸來，最後果然成了，回到原點大概4:45.今天這個山玩了快7小時。原計劃還去縣民之濱喝咖啡摘薄荷，現在改計劃去便利店喝。

平安歸來照，大家都笑的特別燦爛。

看看我玩過滑梯的褲子，有一點點磨破，但還能穿。還好這個布料能把泥洗淨。

我們連臉上都沾了泥，實在太拼了，不過我們戰勝了，一切都是值得的。

 回來路上直売所買橘子，100元一袋，一袋有10幾個。我們買了10袋。橘子皮薄又甜。

自助服務

我們的橘子 

 今天的壯舉，太值得慶祝了。回去正好是晚飯時候，我們去好喫的中餐館。照片自己說明了～

我順便記錄一下，raku老師原來一直說晚上要去教會，最後聽到去中餐館喫飯就馬上放棄了...





昨天西條颳大風下大雨，聽說有不少意外。我們正好去了南面避難。今次囲爐裏慶聖誕，穿浴衣逛古鎮，瀨戶內海橘子園騎行，出張料理人本格囲爐裏飯，歷盡艱辛的登山，最後豐盛的中餐，是一次難忘之旅～
