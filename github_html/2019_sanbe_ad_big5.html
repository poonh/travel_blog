1.深秋三瓶石見紅葉之旅 - 廣告

日期：2019年11月（2天1夜）

D1: 西條 - 石見銀山 - 北の原キャンプ場 - さひめ野溫泉 - 火鍋晚餐 - 觀星

早餐後出發，到島根世界遺產石見銀山。此乃戰國到江戶時代日本最大銀山，高峯期產量達到世界三分一。現在景區街道當時就保留下來。 我們先於當地午飯，逛各個景點，然後租用電動自行車騎森林小道到龍源寺礦道，深入感受礦工當時開採的艱難。
保留當時景色的石見銀山街道
林間騎車
還能看到當時徒手挖掘痕跡的龍源寺礦道
下午我們將入住三瓶山腳下的北の原キャンプ場最大的木屋，然後到さひめ野溫泉泡溫泉。晚飯木屋喫火鍋，飯後觀星。
我們住的森林木屋
D2:早餐後前往三瓶山登山。登頂需時約兩小時。山頂360度美景，一側是日本海，另外三側是無窮無盡的中國山地，非常壯觀。
成本估計：
車費～1200
石見銀山電動車*龍源寺門票：700+200（外國人優惠，原價400）
住宿：3000
溫泉：500
必要開支 ～ 5600，另一早二午一晚，合共～10000
