2.毕业安芸滩骑行 - 相关行程信息

(所有价格2024年底更新)

地理简介：
看地图这里从西到东坐船是总共5个岛，跨越广岛及爱媛县,我们骑到第三个岛豊島就回来。从本土过去，经过安芸滩大桥。这几个小岛都没有便利店的，只有豊島一个超市A-Coop。

住宿：
県民の浜(县民之滨)

属于吴市政府观光产业科。里面有三种不同住宿，我们住的コテージ(cottage),是要稍微爬点坡的小木屋，看风景更加漂亮。平时18000，周末20000。10个人的平时27000，周末30000。官网，JTB可以预约。厨房有烤盘可以烧烤（这里厨房想不到的齐全，游记再说)，有洗衣干衣机。

5人间的话，建议住513蜜柑の館。10个人建议512。这两间正面对着县民之滨，风景最好。另外的木屋看恋ヶ浜，也是海景的。另外还有仅靠海边的小别墅（2-4人用），也有一泊两食的酒店房间（輝ぎの館）。

这里玩的非常多，有室内运动场（600元/小时），网球场(750元/小时)，室外足球/迷你哥儿夫球场（免费，自己带工具），温泉(600元)，网球()，海滩，皮筏艇,SUP(4000元)，夏天开放游泳池（25米，200日元）。要是想搞学习活动有教室可以租。

这里还有民营天文台，搞各样活动，周六晚开放，平时要预约，200日元一个人。想体验的话有古法造盐体验 (5个人6000)，不体验也可以参观造盐设施。

这是一个步行距离就可以玩一天的好地方，也是修学旅行的好去处。

由于是吴市政府公共设施，根据政府政策，每隔5年要公开管理者。根据政府报告2010-2015年间，酒店平均每年入住人数约1万（我们住的小木屋3000），所有设施加起来使用人次年间约14，5万。整个景区年间总收入2亿多，支出基本持平。时有盈余（最多500万）时有赤字（最多2百多万）。2021年疫情期间，人数大幅减少，政府给4百万补助金，总收入和疫情前差不多。

県民の浜的指定管理者自2005年以来一直都是本地企业株式会社県民の浜，直到2023年发生了一起政治事件。

故事要从历史说起。県民の浜1988年开设，当时上蒲刈島还没合并到吴市，是县政府的设施，由当地直接营运。2005年和吴市合并后引入指定管理制度，当时岛上112民岛民（合并时候岛上居民约2000多名。）出资52%，蒲刈町出资48%成立第三方公司株式会社県民の浜，由合并前町长柴崎龍雄当会社代表者（他当时已经连续25年当町长）。会社注册地址也在県民の浜内 ，除了管理県民の浜并无其他业务。会社80%员工是本地人。

直到2023年更新指定管理者发生变化，当时管理隔壁海滩梶ヶ浜コテージ的管理公司ビルックス株式会社加入竞争。他们的方案大幅削减员工工资。前市议员谷本誠一为此事进行过街头演讲，对于新公司的工资削减，质疑工资低的不合法。株式会社県民の浜当时代表者平尾俊正经共产党议员奥田和夫介绍，上书请愿吴市议会议长,提出新管理公司的问题，希望他们慎重考虑。株式会社県民の浜也有员工在change发起签名运动，希望市政府公正判断。广岛县劳连发起署名抗议，并且在市政府表决前共55人到市庁舎抗议。最后，市政府以1票反对，30票赞成支持改变管理者。ビルックス株式会社成功打败了本地人。株式会社県民の浜随即解散。

我看政府公开的候补者选定文件，评选准则关于经费预算一项，株式会社県民の浜评分较高。新公司评分高的地方是业务计划内容。只是我看其他资料感觉是开支问题。具体评审内容我也不了解。我上面提到的两名议员，进行街头演讲的谷本誠一和帮忙上书请愿的奥田和夫在2022年底定例会議案提出质疑，包括两家管理公司工资预算差异及为什么不公开選定委員会的審議内容。


梶ヶ浜コテージ自行车租贷
2017开始的自行车租贷业务，可以异地还车。本店租用最便宜，8小时2000（非电动车），3000（电动车），可网上预约。吴市川尻駅前支店也有分店，费用高2000。从川尻駅骑两三公里可以到岛上。我们骑的とびしま海道官网地图在此。 骑行路上的风景点参考此。这条路线平时车不多。过桥时上下坡很陡，我们都要推车（我们骑非电动车)。其他线路平坦。有行人路的路段耶挺多。如果你对骑行不是很有经验的话，还是结伴，平时来较好。

饮食：
这个不用想了，乡下人少的地方就不是以餐厅为卖点的，不过路上也遇到高档海鲜店，看我的google地图，不过我们骑行时的午饭倒不错。这几个岛连便利店都没，只有好像中国那种杂货店，超市也就只有一家。吃的地方不多，这个网站都有介绍。
https://setouchiskip.jp/

但我超级推荐我们骑行那天，在丰岛东侧的小店しまcafe。 他们有自家产的濑户内海礼品，带柠檬皮的柠檬冰淇淋及橘子酒蛋糕尤其好吃呢。他家的牛筋咖喱也很赞。他们有网点的哦。后来我还买了好些东西呢。
http://www.kitatani-cc.com/shimacafe.html

安芸滩大桥回数券：
这是链接本土的桥，路费730日元。但现在办复兴，你只要在几个岛上消费满1000日元（可以拼单），然后在指定地点凭消费小票加来的时候的过桥小票，就可以换回数券，返程时出示回数券就免费过桥，意思是你只需要单程路费。实际上我们换回数券时，也不用给消费小票，只有过桥的小票就行。
