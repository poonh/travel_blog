今天行程：
早上吃完早餐到左营站坐8:30的屏东客运9127D(票价61元，klook预订)到东琉线码头，计划坐10:10蓝白航运到小琉球，船程约20分钟（最后坐了9:45的，往返船票450元，klook预订）。11:00-12:30玩水行侠的水上自行车(1200元，klook预订)，随后午餐吃简餐。14:30-15:10坐探索拉美半潜艇看海龟(380元，klook预订 )。最后坐16:00的船返程（实际坐了15:30）。回到东港(即东琉线码头)于码头隔壁的华侨市场（当地海鲜市场）吃当造肥美的黑鲔鱼（蓝琼吞拿，4-6月是他们洄游产卵季）及买手信。最后包车回酒店(1000元)。详细信息看行程相关资料。

今天出海是行程重点～

行程开始:
 酒店包早餐，不是自助的。有6款选择，四款是汉堡，另外两款分别是粥及肉臊饭。每款汉堡用的面包都不一样，例如竹炭，巧克力，荞麦等。每款早餐卖相都很漂亮，而且很丰富，根本吃不完。汉堡餐有些炸物，不是新鲜炸的，不过除此之外都很好，沙拉酱汁尤其好吃。有一点特别不好是由于分量太多，我要求职员减量，她以怕乱为理由拒绝。这就逼我浪费食物（实际上我看其他客人也吃不完）。

这里是文创酒店，很多漂亮的餐具

吃完早饭，由于我早一天探好路，算好我们7:50分出发（约8:10到左营大巴站等8:30的大巴）。最后我们7:47离开酒店，8:10到达目的地。在左营捷运先上厕所，然后在捷运1出口（高铁方向）坐扶手电梯上去，上去后是捷运和高铁的半露天连接处，在连接处往左方看就看到大巴站，到最尽头的1号站台就是我们上车处。从下捷运到大巴站台大概2分钟。

话说，在我们坐扶梯上去大巴站台时，"少"突然说肚子痛，要上厕所。最后我们时间刚刚好，8:26上车。出远门真的要提早点。如果你万一错过了车，这里也有拼车兜客的，参考价200/人。

我们第一站东琉线码头到站，路上没怎么堵车。我在ibus app观察了几天(网页版在此,我主要看首班车8:00的），平日早上这段时间基本55分钟到。

我们买完东西，就慢慢走去水行侠那边玩水上自行车了。这边的街道明显和码头那边不一样。
水行侠s那边的街道


到了职员说海况不太好，问我们能不能下午1点玩。但我们14:30要玩半潜艇，时间无法改。后来我发现他们10点半左右发过邮件问我能不能改时间，不过我没看到。其实我之前Line联系过他们的，不过他们不知道那个line是我。临时联系用邮件很不方便，他们大可以预约时让客人留下line。虽然我看当时风平浪静但也可以海况不好临时改时间，预约的时候最好就说明。沟通上我觉得他们有改善空间。当时我们准时到，除了海况原因教练也没回来，等到11:20左右我们出发。期间“少”看到店里的狗又发狂，今次闯祸了，被狗咬。她摸狗的脸，人家不喜欢。我突然听到狗叫了一声，人家大概是反抗。狗狗好像叫阿东，其实很乖的，我刚进门时不小心踩到它也没事，职员还说它不咬人，但咬了少后，之后有小朋友想摸它，职员就说它咬人了...

为此，后来搞了很多事，主要是老特别担心。老说"少"出海前还很开心，回来不一样。我没有她敏感没看得出。我自己本身对狂犬病没了解过，现在也了解到一点。被狗咬一定要马上用水及洗手液冲洗10-15分钟，然后消毒。万一中了病毒，死亡率百分百。不过越落后的地方狂犬病越多，台湾2013年从菲律宾引入病毒，在此之前已经50年没有人染病。台湾的传染病统计资料查询系统有台湾狂犬病数据，参考此。基本上，在台湾感染到狂犬病的概率很低。

回到我们的水上自行车，虽然今天很热，但玩的时候有海风，一点都不觉得热。穿的救生衣有兜，可以带一小瓶水（200ml的能放下，500ml不可以）。不过当时情况全程不喝也行，我喝了几口。

这时大概一点。我们准备午饭。水行侠位于码头东侧，吃的码头西侧多。码头附近很多是租机车或卖商品，没见到有啥吃。这点实在不太方便。之前看地图知道小琉球魅力商圈那带有吃（靠近花瓶岩，从码头走约十分钟，到达前有几十米斜坡），我们往那边走。这一带看上去好点的有海鲜餐厅，也有一些小店，好些没冷气的...

我们在小琉球魅力商圈的7-11岁便买了点吃的，还有麻花（两袋120元，麻花店内也卖鱿鱼丝，麻花是烘焙的，非油炸，比起油炸的硬点），然后去码头免税店那边坐。当日早餐吃的很饱午餐不觉得饿。

码头附近很多租机车的地方，我见过有家说没机车牌照也能租。没有预定水上活动的话也可以现场预定。

下船后我们就直奔华侨市场吃黑鲔鱼了。由于怕热，我们不考虑热炒。这里生鱼片店很多，由于货源都一样，我相信每家都差不多。我们去的守席。吃了1）5片金三角（黑鲔鱼最贵的部位）400元，2）两碗海鲜当（400一碗），3）星鳗当，忘记多少钱，好像400左右，4）三个不同部位黑鲔鱼生鱼片，好像500， 5）熟虾寿司及 6）生墨鱼片。4个人吃，吃的很饱，总共2400多。另外，吃当饭送海鲜味增汤（非常鲜美），可以续。这里用的米很好吃，环境主要问题是苍蝇太多。温度倒是没有想象中热，能接受。比起昨天的龙华市场好。

另外，这边早上五六点就是拍卖鱼的时候，如果你想看可以这个时候来看。我在youtube看过视频，还可以直击宰鱼过程，但毕竟是生鱼片，切了就吃，普通人能近距离看觉得有点不卫生...

在市场期间，突然下了很大的雨，可能超过1小时。当时特别凉快，在市场内好像开冷气一样。等到雨差不多停了，我们也逛完，我们就去泰富等候室坐，等18:15的包车，当时大概17:45。

话说，我之前在youtube看到有人在华侨市场吃海胆，连壳一起吃的，不过没见到有卖，不知道现在不是时候还是去晚了。

我们沿着裕诚路往东走，首先看到一家炸鸡店，但里面没人不敢去吃。但我后来查google评价，吃的人很多的。后来我们去了富民路那边一家看上去挺上档次的韩国料理店，里面装修也很好，不过低销要每人350元，当时我们根本吃不了那么多（台湾很多地方有低销一说），然后就去别的地方了。最后去了一家主打鸡汤的店 - 龙涎居鸡膳食坊。当时其实也没人吃，不过这店感觉不错我们就去了。
