白木山简介
yamaquest广岛县排名第四。攀升800多米，我们原路往返，走短的路线，上下都是两小时。这座山楼梯挺多，但不全是。不算最安全的路，但也不危险，对体能有要求，我们下来都很累，大家的腿也疼了好几天，最好带登山杖。

白木山登山线路及记录参见yamap。

我原来规划的登山线路如下（来自yamap)，但后来只走了黑色有圈的那段，从白木山駅附近有正式登山口上。另外，如果你坐电车，有条线路下山往上深川駅。

停车场：登山口附近没看见正式停车场，我们停在大概1公里外的Shoji超市。
厕所：登山路上没厕所的，还有标志禁止随地上，要上就在超市上。
