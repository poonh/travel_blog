2.初春島根悠閒遊 - 相關行程信息

(所有價格2024年底更新)

我們的行程地圖(線路取自navitime ,包括我在這章介紹但我們沒去的地方)：
(繞湖的是自行車線路)
(紅:住宿   藍:景點    綠:喫飯    橙:超市  啡:加油站    黃:廁所  紫:停車場)


今次旅程去的地方都靠的比較近，開車時間不太長。下面介紹我研究過在雲南市及奧出雲一帶的地方。

奧出雲旅遊官網
一貫島根旅遊官網風格，喫住玩看都有詳細信息並附有網址。這官網還特別介紹農泊，裏面還有一位老奶奶的airbnb。我打開airbnb看，聯繫人不是老奶奶自己，而是在奧出雲町観光協會下。他們收錄了不到町內10所住宿，除了個人的還有民宿，甚至政府自己的住宿設施。介紹說除了宿泊諮詢還提供旅遊諮詢。這樣的管理方式實在給旅客帶來很大方便。

住宿：
健康の森
政府宿泊設施，我們住這裏,電話預定。官網有郵件不知道能否郵件預定，4人棟9,540円，沒獨立臥室,可以租燒烤工具，有網球場也有球拍和球出租。去超市(みしまや 雲南三刀屋店,大超市)及溫泉(おろち湯ったり館)開車10多分鐘就到。可以睡帳篷及森林小木屋，屬於荒山野嶺，自然有蟲子。這裏比我們以前住過的小木屋便宜。房子條件比我們之前住的差一點，房間不太大，但也不擁擠。沒有微波爐，洗髮液及沐浴露，只有洗手液及洗衣液。要自己帶筷子。廚具不太多但夠用。枕頭是用蕎麥還是塑料的，比較硬。暖爐,空調,垃圾要另外收費。要自己帶垃圾袋，不提供的（還好我事前在谷歌評價看到了，帶了我用來做時尚透明桌布的45L塑料袋過來）。有牀單及枕頭套，但沒有被罩。房子是很乾淨的。泡溫泉的話記得去前臺拿優惠券（我們當時8折）。

這裏有遊步道，由於位於山上上早上可以看雲海（雖然山上，但開車沒有翻山，很好開的），看雲海去創作研修棟該較好。

這裏覺得規矩沒有一般營地/酒店那麼死，雖然說4點入住，但你早點去也行。打網球也是，也沒明確規定早上幾點開始，你想早點也行的。這裏職員不多，老人爲主，我們入住退宿時都和我們聊。 由於職員不多，辦公時間打電話來也不一定有人接的，他們可能去哪幹活去了。

話說網站還是做的不錯的，最棒是小木屋有360照片，房子多大，佈局怎樣一看就清楚。

かみくの桃源郷
政府宿泊設施，這裏的小木屋更便宜，4人棟5500。但因爲去超市溫泉較遠，我就沒考慮了。

奧出雲多根自然博物館(酒店房及農家住宿)
號稱全國唯一能住宿的博物館，酒店房有爲小朋友而設的恐龍房。農家住宿是奧出雲百姓塾。看照片很大，4人住1棟24000，能燒烤還有各種體驗活動。

這裏附近有個溫泉長者の湯，吸引我的是這個溫泉鹼度特高（pH9.8)。網站說用“美肌測定器“測量入浴後皮膚柔軟度，水分都變好。

其他住宿看官網。

遊玩：
觀光列車あめつち
這趟列車有三條路線，這邊的是木次線，米子到出雲橫田，全程3小時，車資約3000。木次到出雲橫田1小時,賣點山陰景色。奧出雲原來還有很出名的觀光列車奧出雲おろち號，賣點是全國最高（700多米），短時間內攀升。我看視頻還有預定中間站的蕎麥麪，在開放式車廂送貨。島根旅遊官網還有一個旅遊產品是把這兩趟觀光列車結合在一起，中間喫和牛泡溫泉，松江出發最後又返回松江。不過原來這趟列車已於2023年年底結束。

奧出雲練鐵
在古代出口全國8成鐵的奧出雲，現在還保存他們的文化遺產保存得很好。當時的遺蹟鉄穴殘丘還能看到。我感興趣的事練鐵生成的循環型農業的科學原理，我在這介紹一下。

Tatara鍊鐵需要很重要的原料 - 砂鐵，藏於花崗岩，開採技術叫“鐵穴流”（かんなながし），要先將山體切割崩解，然後砂鐵需要加入適當比例的水才能提取。所以開採地方需要水渠和蓄水池（這爲了日後農業灌溉準備）。山體被開採後並沒荒廢，而是被改造成梯田。但含花崗岩的土地都很貧瘠，所以需要通過種香菇(しいたけ)和蕎麥及牛糞堆肥,提升土壤肥力，並且用之前練鐵建造的供水系統（那是含有豐富礦物質的花崗岩水），由此孕育出本地出名的“仁多米”(獲過全國金獎)。當時牛用來運鐵，品種經過改良，今天島根和牛全國有名(島根和牛在2022年第12回全國和牛能力共進中，榮獲肉質評価1位的殊榮，現在已經被政府用作旅遊宣傳），還有肉厚香菇，加蕎麥殼的蕎麥麪，因爲練鐵造成一大堆特產。源自鐵穴流的水利系統網絡，還孕育了多種多樣的生物和生態系統，形成了豐富的自然環境。1991年奧出雲的tatara鍊鐵資源再利用農業被認定爲中國地區首個日本農業遺產。

棉打公園是現在重要的梯田遺蹟，每年有活動可以夜遊。我們在這附近喫飯的店鉄師卜藏家 鈩の舎以前就是用來鍊鐵的。店外還保留着以前鍊鐵的儲水池。棉打公園梯田沒看到專門停車場，我們停車在這裏的(注意，這帶喫飯很少，要事前查好，此店也不天天開)。

想參觀tatara練鐵有關博物館有一下：

菅谷たたら山內
たたら角爐伝承館(免門票,在廣島去鬼之舌震的路上)

奧出雲其餘看梯田的地方：
福頼棚田展望臺
這裏景色可能更好，但看谷歌評價說開車的路很窄，我直接屏蔽掉... 

大原新田
這是“日本棚田百選に選定”。但看谷歌評價說展望臺看不了全景。綜合考慮到開車及綿打公園附近有飯喫，所以我就決定去綿打公園了 。

出雲橫田
車站出雲橫田駅本身是景點，仿製出雲大社的，另外鍊鐵博物館(奧出雲たたらと刀剣館)在南面一點，還有名建築橫田相愛教會也在附近。

鬼の舌震
 島根著名的景點，秋天紅葉勝地。主要是看流水和石頭，景色很好，有吊橋。遊步道單程約2公里，木製棧道很好走，老人小孩都可以。雖然路不長，今次我們又沒人拍照，不知道爲什麼往返竟然走了2小時。

這裏有家喫飯的地方，叫鬼蕎麥，麪條很筋道很有顆粒感。週末午飯去晚了可能要等。

島根自行車線路官網
我去過的縣發現原來只有廣島沒自行車旅行官網，其餘的我覺得島根的做的最好。不但有不同版本的電子地圖線路下載，地圖標註要注意的地方，還有短視頻顯示海拔。

奧出雲町サイクリングターミナル(自行車租貸)
 位於出雲三成，我計劃的自行車租車點就在這。起碼提前三天預約(只能電話)，4小時內510日元，只去水庫的話時間是夠的了（我計劃線路是全程20公里）。 
我們原計劃騎的接近官網尾原ダム路線，不一樣的是的地方官網不是繞水庫一週而是從起點到木次。我計劃的線路是繞水庫一週，參考了比賽線路的。我還做了個文件標註了線路要注意的地方。

奧出雲葡萄園
從住的地方過去開車十多分鐘就到，在上坡上，可以俯視葡萄園，那裏有餐廳及室外咖啡店，情侶約會首選的地方。

尾原ダム（大壩）
開車的話，可以去西側大壩，展望臺拍照，那裏有拍照點，之後可以開車去山上看遠景，中間經過牧場可以看牛。如果你湊夠5個人，可以申請見學，帶着頭盔在大壩裏面看。

這是一個活動非常豐富的地方。大衆化的每年3月第一個週六有放流活動(イベント)，當中有見學，爬樓梯比賽（只有當天樓梯纔開放，完成可能有證書），還有很多喫的，看到我都想參加。秋天有湖祭 ，有三種不同水上活動，還有其他節目。除了這兩個大型的，還有SUP體驗,越野跑,攝影比賽,遠足,花火大會,鐵人三項,野外定向等等，詳情看他們活動專頁。

這裏的自行車賽道可是B級公認的,想辦比賽提供配套設施。

最後，如果你想看大壩的政府規劃，參考《尾原ダム 水源地域ビジョン》（2013年）。從報告看到，大壩除了儲水功能外，還有連同地域團體帶動觀光，推動企業社會責任，和地方大學合作研究，環境學習，農業體驗，發展體育活動等等。另外，日本基本所有大壩都是風景區，尾原是日本比較大的一個，設施活動多些。
