1.瀨戶貴賓列車尾道休閒一天遊 - 廣告

日期: 2023年2月24日(週五)

需要用品:水，防曬，運動衣服，浴巾(如去溫泉)

行程: 早8點多西條出發，乘坐廣島站9:32瀨戶內海貴賓專列エトセトラ(etsetora)三小時暢遊瀨戶內海。 我們定好了靠近海側位置，帶上小點心可以一邊享受瀨戶內海美景一邊品嚐美食。

中午12:32到尾道，先午飯。想看風景可以到千光寺附近,可以遙望瀨戶內海景色的cafe holoholo午飯，或者尾道駅一代很多 其他選擇。隨後步行到千光寺公園。千光寺山海拔100多米，乃當地風景名聲，頂上可以俯視 瀨戶內海景色。

備選行程: 下山後看時間有兩個選擇
1.經多部電影拍攝地-千光寺新道返回車站。此道自大正時代，以狹長石板道聞名，乃尾道標 志。附近有多家特色咖啡店，可以享受下午茶後返回。

2.當地人氣溫泉平原溫泉ぽっぽの湯。除了室內外風呂還有女士專享的漢方藥草桑拿。
備註: 從車站坐バス過去，約20分鐘。

成本估計:
Jr: 1520+990 (西條-廣島-尾道-西條)
エトセトラ グーリン券(etsetora乘車券): 1000
午飯: ~1000
溫泉門票:860
溫泉往返バス:560
合共: 〜6000(不去溫泉的話~5000)
