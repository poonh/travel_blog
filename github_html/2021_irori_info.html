2.相关行程信息

我们的行程地图（线路取自navitime）： 


梶ヶ浜コテージhttps://kajigahama.jp/

吴市农林水产课管理，除了コテージ还有营地，造盐设施，海边游步道外还有一些景点。コテージ是古民家风格，房子一室一厅+独立厨房，总共五六十平米。外面可以烧柴做饭，コテージ可以提供鲷鱼。更详细介绍看游记。
房子17300一栋，可以住4个人，另外加钱可以最多加两个床位。 另外，垃圾要自己带走。

补充一下，广岛县另一个可以玩囲炉裏的营地是三次江の川カヌー公園さくぎ，价格也差不多。

出张料理人Shino
 http://kure-kokoro.com/about_shino/
https://www.facebook.com/shino.evolving/
https://www.youtube.com/channel/UCTuMUDE3d5R2UG178UwxPdQ

一位帅气的厨司，广岛县唯一的出张料理人。他有自己的生意，料理教室及提供上门做饭服务(包洗碗)。全广岛县都能出张，费用都是每人4000+出张费（按距离算）。他会做西餐，去梶ヶ浜コテージ就专门做囲炉裏饭。来之前几天他会联系你，问你想吃什么及有什么过敏。

日本全国一些古民家有囲炉裏，但能提供上门做本格囲炉裏饭可能就只此一家。我吃完之后觉得自己玩囲炉裏不会做，还是请人来做好，真是觉得好高级。

这顿饭如果单独付费要每人6000的。

自転車観光案内 ぽたりん（自行车导游服务）
 https://npo-scm.wixsite.com/setouchi/自転車観光案内ぽたりん
https://www.facebook.com/takao.saito.5099 (导游fb)

我们的导游å斉藤先生就是做旅游策划的,拿了"广岛通"资格。とびしま海道有官方自行车线路（我们上次骑的那条），而他们自己开发的和官网线路没有重叠。我们参加的是《島まるごと庭園！歴史とおもてなし体験ツアー》，深入山上橘子园，看海景，喝抹茶及听历史故事，尽情享受日本小岛风情。

他们的团都是3小时，每人7000元。虽然是贵，不过专业导游提供的服务和自己玩是很不一样的。他们可是获过ベストガイド(best guide)賞的。

七国见山
特别要说一下七国见山。虽然只有457米，但下山的物見岩ルート非常不容易，是野路，不是登山高手不要试。我们找不到出路，走了一半之后原路折返。后来看日本人的游记，这条路后半更难走。

上山的路很好，是志愿者帮忙搭建的，参考此。

かまがり天体観測館: http://kamaten.net/
离梶ヶ浜10公里的県民の浜，科普天文台（全日本很多这种天文台），初了周六都要预约，门票200。当时我们计划去的，不过同一时间被团体预约了。
