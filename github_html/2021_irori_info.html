梶ヶ浜コテージ
 我们住，吃本格囲炉裡飯，以及骑车的地方。属于吴市农林水产课管理， 位于下蒲刈岛（过了安芸滩大桥第一个小岛）的海滩。除了コテージ还有营地，造盐设施，海边游步道,疫情下还开了优美的コワーキングスペース(coworking space，参见吴市2021年12期广报)。此外还有一些景点。コテージ是古民家(不是真的)风格，房子一室一厅+独立厨房，总共五六十平米。外面可以烧柴做饭(かまど)，コテージ可以提供鲷鱼或其他美食套餐(烧烤火锅都行，不用自己买菜。
房子17300一栋，可以住4个人，另外加钱可以最多加两个床位，只能电话预约。 另外，垃圾要自己带走。 补充一下，广岛县另一个可以玩囲炉裏的营地是三次江の川カヌー公園さくぎ，价格也差不多。

出张料理人Shino
 个人主页
facebook
Youtube 这频道是英语的，他教料理制作。如果你不会日语请他来做饭也没有问题的。

Shino是一位帅气的厨司，广岛县唯一的出张料理人。他有自己的生意，料理教室及提供上门做饭服务(包洗碗)。全广岛县都能出张，费用都是每人4000+出张费（按距离算）。他会做西餐，去梶ヶ浜コテージ就专门做囲炉裏饭。来之前几天他会联系你，问你想吃什么及有什么过敏。

日本全国有囲炉裏的古民家很多，但能吃正宗囲炉裡飯不多。而且能提供上门做本格囲炉裏饭可能就只此一家。我吃完之后觉得自己玩囲炉裏不会做，还是请人来做好，真是觉得好高级。如果是外国游客想深度体验日本，我推荐这个体验。Shino自己有店铺，也有料理教室，梶ヶ浜コテージ把他引入是这里一大特色。

这顿饭如果单独付费要每人6000的(4人算，详见官网)。

自転車観光案内 ぽたりん（自行车导游服务）
 官网
Facebook
营运团体是NPO法人　瀬戸内サイクルメディア。看法人网站简介及政府的NPO法人ポータルサイト，他们活动包括自行车教室，小轮车体验，和其他团体办比赛，帮忙宣传活动等,成立目的是振兴文化，地域观光，自行车运动及改善围绕自行车的交通环境做出贡献(吴市也的确有自行车推进计划)。 我们的导游斉藤孝穂是NPO的法人，广岛县观光大使，2021年加入豊町协力队(协力队是日本政府的移住政策，招纳城市年轻人到乡村从事几年地域振兴工作)，做旅游策划的。他拿了"广岛通"资格。とびしま海道有官方自行车线路（参见《毕业安芸滩骑行》，自己租车2000），而他们自己开发的和官网线路没有重叠。我们参加的是《島まるごと庭園！歴史とおもてなし体験ツアー》，深入山上橘子园，看海景，喝抹茶及听历史故事，尽情享受日本小岛风情。

他们的团都是3小时，每人7000元。虽然是贵，不过专业导游提供的服务和自己玩是很不一样的(提醒一下，我们当时参加政府旅游计划，这个自行车导游团算是送的)。他们可是获过ベストガイド(best guide)賞的。

可网上预约，网站有英语但不确定是否真的有英语服务。

广岛县另一条自行车道很出名 - しまなみ海道（县东侧福山那边）。他们官网也很棒，预约租车骑行线路各种信息一条龙、とびしま海道没专门自行车官网(不过相比起岛根，我还是觉得岛根自行车官网最好，他们提供电子地图线路，还有视频显示高低落差）。とびしま自行车租贷业务由梶ヶ浜コテージ的管理公司ビルックス株式会社于2017年引入。而梶ヶ浜コテージ负责人高岛俊思是とびしま自行车旅行的发起人。他爱媛出生，2015年参加地域协力队到下蒲刈島，之后选择留下来，加入了管理公司管理梶ヶ浜コテージ，并且开始了自行车业务,从开始9台车到2022年45台。

从这个故事，我看到日本政府的地域发展政策，协力队首先可以吸引城市人才。然后你有好的旅游计划，市政府也给力帮你宣传。日本运动发达，除了全国众多自行车道，海上水库河流玩皮筏艇也很多。那里来的运动人才？我相信和日本部活的教育政策有关。部活是日本初中开始选择的课外活动，差不多要天天练,分文化和体育，广岛大学的スポーツセンター(体育中心)下属的体育社团超过40种运动。振兴地域，从人才培养，移住政策，到地方政府和市民合作有一连串关系。

七国见山
特别要说一下七国见山。虽然只有457米，但下山的物見岩ルート非常不容易，是野路，不是登山高手不要试。我们找不到出路，走了一半之后原路折返。后来看日本人的游记，这条路后半更难走。

上山的路很好，是志愿者帮忙搭建的，参考此。

かまがり天体観測館
离梶ヶ浜10公里的県民の浜，科普天文台（全日本很多这种天文台），初了周六都要预约，门票200。当时我们计划去的，不过同一时间被团体预约了。
