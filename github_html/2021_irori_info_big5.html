梶ヶ浜コテージ
 我們住，喫本格囲爐裡飯，以及騎車的地方。屬於吳市農林水產課管理， 位於下蒲刈島（過了安芸灘大橋第一個小島）的海灘。除了コテージ還有營地，造鹽設施，海邊遊步道,疫情下還開了優美的コワーキングスペース(coworking space，參見吳市2021年12期廣報)。此外還有一些景點。コテージ是古民家(不是真的)風格，房子一室一廳+獨立廚房，總共五六十平米。外面可以燒柴做飯(かまど)，コテージ可以提供鯛魚或其他美食套餐(燒烤火鍋都行，不用自己買菜。
房子17300一棟，可以住4個人，另外加錢可以最多加兩個牀位，只能電話預約。 另外，垃圾要自己帶走。 補充一下，廣島縣另一個可以玩囲爐裏的營地是三次江の川カヌー公園さくぎ，價格也差不多。

出張料理人Shino
 個人主頁
facebook
Youtube 這頻道是英語的，他教料理製作。如果你不會日語請他來做飯也沒有問題的。

Shino是一位帥氣的廚司，廣島縣唯一的出張料理人。他有自己的生意，料理教室及提供上門做飯服務(包洗碗)。全廣島縣都能出張，費用都是每人4000+出張費（按距離算）。他會做西餐，去梶ヶ浜コテージ就專門做囲爐裏飯。來之前幾天他會聯繫你，問你想喫什麼及有什麼過敏。

日本全國有囲爐裏的古民家很多，但能喫正宗囲爐裡飯不多。而且能提供上門做本格囲爐裏飯可能就只此一家。我喫完之後覺得自己玩囲爐裏不會做，還是請人來做好，真是覺得好高級。如果是外國遊客想深度體驗日本，我推薦這個體驗。Shino自己有店鋪，也有料理教室，梶ヶ浜コテージ把他引入是這裏一大特色。

這頓飯如果單獨付費要每人6000的(4人算，詳見官網)。

自転車観光案內 ぽたりん（自行車導遊服務）
 官網
Facebook
營運團體是NPO法人　瀬戸內サイクルメディア。看法人網站簡介及政府的NPO法人ポータルサイト，他們活動包括自行車教室，小輪車體驗，和其他團體辦比賽，幫忙宣傳活動等,成立目的是振興文化，地域觀光，自行車運動及改善圍繞自行車的交通環境做出貢獻(吳市也的確有自行車推進計劃)。 我們的導遊斉藤孝穂是NPO的法人，廣島縣觀光大使，2021年加入豊町協力隊(協力隊是日本政府的移住政策，招納城市年輕人到鄉村從事幾年地域振興工作)，做旅遊策劃的。他拿了"廣島通"資格。とびしま海道有官方自行車線路（參見《畢業安芸灘騎行》，自己租車2000），而他們自己開發的和官網線路沒有重疊。我們參加的是《島まるごと庭園！歴史とおもてなし體験ツアー》，深入山上橘子園，看海景，喝抹茶及聽歷史故事，盡情享受日本小島風情。

他們的團都是3小時，每人7000元。雖然是貴，不過專業導遊提供的服務和自己玩是很不一樣的(提醒一下，我們當時參加政府旅遊計劃，這個自行車導遊團算是送的)。他們可是獲過ベストガイド(best guide)賞的。

可網上預約，網站有英語但不確定是否真的有英語服務。

廣島縣另一條自行車道很出名 - しまなみ海道（縣東側福山那邊）。他們官網也很棒，預約租車騎行線路各種信息一條龍、とびしま海道沒專門自行車官網(不過相比起島根，我還是覺得島根自行車官網最好，他們提供電子地圖線路，還有視頻顯示高低落差）。とびしま自行車租貸業務由梶ヶ浜コテージ的管理公司ビルックス株式會社於2017年引入。而梶ヶ浜コテージ負責人高島俊思是とびしま自行車旅行的發起人。他愛媛出生，2015年參加地域協力隊到下蒲刈島，之後選擇留下來，加入了管理公司管理梶ヶ浜コテージ，並且開始了自行車業務,從開始9臺車到2022年45臺。

從這個故事，我看到日本政府的地域發展政策，協力隊首先可以吸引城市人才。然後你有好的旅遊計劃，市政府也給力幫你宣傳。日本運動發達，除了全國衆多自行車道，海上水庫河流玩皮筏艇也很多。那裏來的運動人才？我相信和日本部活的教育政策有關。部活是日本初中開始選擇的課外活動，差不多要天天練,分文化和體育，廣島大學的スポーツセンター(體育中心)下屬的體育社團超過40種運動。振興地域，從人才培養，移住政策，到地方政府和市民合作有一連串關係。

七國見山
特別要說一下七國見山。雖然只有457米，但下山的物見巖ルート非常不容易，是野路，不是登山高手不要試。我們找不到出路，走了一半之後原路折返。後來看日本人的遊記，這條路後半更難走。

上山的路很好，是志願者幫忙搭建的，參考此。

かまがり天體観測館
離梶ヶ浜10公里的県民の浜，科普天文臺（全日本很多這種天文臺），初了週六都要預約，門票200。當時我們計劃去的，不過同一時間被團體預約了。
