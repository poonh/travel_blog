2.盛夏暢遊日本 - 相關行程信息

(所有價格2024年底更新)

(紅:住宿   藍:景點    綠:喫飯    橙:超市  啡:加油站    黃:廁所  紫:停車場)


出雲旅遊官網

可能是我見過日本最好的官方旅遊網站，信息出奇詳細。大小景點，旅館，餐廳都有介紹。

細看政府網站，出雲市甚至島根縣的旅遊政策非常給力。在《初春島根之旅》 我介紹過我看過最好的自行車觀光網站。島根縣又稱爲"美肌縣"。政府有專門針對女性的美肌旅遊網站及冊子《ご縁旅x美肌旅》。 島根自稱美肌縣是有根據的(《ご縁旅x美肌旅》p5，有研究引用)。山陰地區本身日照時間短，紫外線吸收減少，島根溼度高，而且空氣潔淨對皮膚少刺激。抓住這點就針對女性遊客搞美肌之旅,看《ご縁旅x美肌旅)(p10),島根溫泉被稱爲“美肌溫泉”，裏面介紹的每個溫泉都有自己特點，例如放射，巖化，含鐵等，每種特點都對皮膚有不同幫助，實在非常會宣傳。針對“美肌”，還有“美肌宿”。

要發展美肌旅遊，各方旅遊事業者要給力。政府爲了支持，從2020年開始每年都給推動美肌旅遊補助金《美肌県しまね推進事業補助金募集》(補貼上限150萬)。從網站看到，採用的事業內容包括草藥，藥膳，發酵茶，特殊入浴方法，坐禪等等，總之把美肌及旅遊關聯。

縣級另一個推動旅遊政策補助金關於自然公園活用，個人也能申請，補貼上限500萬，有興趣參看《自然公園を活用した誘客促進補助金》。

爲了方便外國遊客，島根縣還設立網站幫助縣內食肆翻譯菜譜。根據觀光廳《宿泊旅行統計調查》(p6)，島根縣在2023年乃全國外國人住宿人數最低的縣。

更多縣級補助參考島根縣観光振興課。對縣內旅遊數據(縣內各地觀光人數，旅客國籍，觀光類型等)有興趣的可看島根県観光動態調査結果。出雲市政府也有自己的《観光動態調査》。

講完縣級到市級出雲市，他們自己也很多旅遊政策。出雲市政府2024年中開始舉辦「出雲市観光戦略會議」，每次都有非常多詳細報告。從 《R5年度からの新たな観光戦略の取組狀況》(p3)得知， 訪日外國人旅行者每次旅遊支出約15萬，大概爲國內旅行宿泊者3倍。整個島根縣，好些景點門票給外國人打折，例如穴道湖乘船，石見銀山，足立美術館等。坐大巴從廣島市到松江或出雲外國人有乘車優惠，單程票1000日元(原價4000左右)。 針對外國人優惠政策，有日本人向縣政府広聴広報課県民対話室提出建議，認爲同行的日本人也應該給優惠呢。

對於旅行團，只要出雲留宿，市政府直接給現金補助，每人給2000/天，具體參考《出雲市で宿泊される訪日団體旅行への支援》。

キララグループ(kirara group)

屬於出雲市政府的一個集團，經營kirara海灘附近的設施，包括酒店，營地小木屋，いちじく溫泉，餐廳等(參看官網地圖)。我只在本文詳述我去過的。

這裏不是大景點，喫飯的地方不多，便利店超市都只有一家的。

這個地方難得附近有車站(小田駅）。住酒店的話不開車也可以但我們住的地方在小山坡，還是有車好。

住宿
1.キララコテージ(kirara cottage)

我們住的地方，由於在小山坡陽臺風景特別好。離游泳的地方開車3分鐘。除了營地，室內住宿有多種選擇，最便宜的不帶獨立衛浴。有兩種獨立別墅，分別是コテージ(4或6人，沒桑拿)及ログハウス(log house，4，6或8人，含桑拿,4人用超旺季(暑假黃金週)一棟22,900円，非超旺季週末19,400円，平時17,000円)。這兩種獨立別墅即使淡季都特別受歡迎。我們超旺季中的淡季平時去(在8月底，當時海灘沒有小賣部救生員但還有廁所浴室，游泳的人已經少很多)，提前兩個月預約。租用燒烤場一桌2000（含所有燒烤工具）。按季節有不同的宿泊套餐。除了官網，樂天,jalan也能預約住房。但宿泊套餐和燒烤場只能電話預定。夏天有游泳圈(浮き輪)出租。

如果你住這裏,去集團下的いちじく溫泉或酒店游泳池有優惠的，記得問前臺。另外，這裏可以用信用卡(我們住過一些只能用現金)。整個園區及道の駅(游泳地方哪裏)都是北歐風格，所以小木屋有桑拿，不知道是不是因爲出雲和芬蘭Kalajoki結爲友好都市。

整個集團指定給株式會社多伎振興管理。管理公司老闆是島根JA(農業協同組合),JF（漁業協同組合）及出雲市商工局。我們住過不少政府的營地，這次這家管理水平是最高的，除了前臺很長的服務時間（早7至晚10，以前我們住的不是5點就是6點前就要入住），而且網站介紹設施非常詳細的，廚房用品就連一個削皮刀都寫了，燒烤廚房用具都很新。 如果你看2022年政府對管理者的報告《事業実施狀況の評価結果》的見晴らしの丘公園(即kirara cottage)，2018-2022每年掙大概幾百到接近1000萬。人數方面，全年利用人數兩三萬。據出雲市政府報告,此集團可能未來幾年轉爲民間經營。

2.マリンタラソ酒店(Marine Thalasso)

集團下的酒店，主打海水泳池。室外池可以看日落，室內各種按摩池特別多。非宿泊者費用1080日元。餐廳有午晚餐，菜式不特別多。

キララビーチ(kirara海灘)

日本海灘有"海開""海閉"的說法。一般7月初海開(海開き日)。海開後廁所浴室(一般投幣式收費，kirara是三百元三分鐘)等設施才能用，還有小賣店救生員等服務。お盆（日本8月盆節）後可能沒有小賣店救生員，但廁所浴室還開，直到海閉，一般在8月底，具體時間要看官網。但也有一些海灘例外，例如《長濱燒烤暢泳之旅》的廣島江田島的長濱就全年開放。

另外，如果遇到高浪，海灘會關閉。日本海可能容易遇到這個情況,瀨戶內海可能較好。我們原計劃第一天去游泳但沒開（當天風速每秒五六米，第二天開放時兩三米）。如果你一天往返，可以去之前打電話確定。過夜的話，計劃行程時最好彈性點，今天不遊可以明天遊。當然kirara海灘遊不了可以去酒店海水泳池。kirara官網有攝像頭，你可以看看有沒有人游泳，或者看天氣預報的風速也可以估計能不能遊。

道の駅正面面對海灘游泳部分(整個海灘很大，開放一小部分)。kirara cottage 位於海灘東側幾百米的山丘上。兩者中間有兩個風車，你游泳時往沙灘一側一看，以爲自己去了歐洲，因爲上述兩個建築都是歐風，美的不得了。再加上這裏是清澈的日本海，比起我們的瀨戶內海確實不一樣。

這個海灘的水質是連續6年最高的AA級(截止2021)，具體定義參考島根縣 令和3年度海水浴場の水質調査結果 。 日本政府部門環境省 水・大気環境局每年海開前都做海水浴場海水質量調查，水質爲最高級AA的海灘佔調查一半以上。 2024年全國調查報告在 《令和6年度 水浴場(開設前)水質調査結果》，裏面有全國個別海灘檢查結果，調查內容爲"ふん便性大腸菌羣數、油膜の有無、化學的酸素要求量(COD)、透明度 (水素イオン濃度、病原性大腸菌O-157"。

附近其他海灘

石見海濱公園也挺受歡迎，也有住的。小木屋只要每人1000左右，但裏面什麼都沒有，沒廚房也沒廁所，牀鋪也要自己帶或者租。想享受還是推薦kirara。

立久恵峽

郊遊景點，路線簡單，可以帶老人，秋天可以看紅葉,遊步道2.3公里，官網給的時間是40分鐘。此類型景點整個島根很多，詳見旅遊官網。特別提醒去島根這種地方事前查一下政府網站這頁《國立公園・國定公園・県立公園の通行(使用)できない區間等》。這些地方很多時候有落石，還有颱風暴雨等車路會不通的。

這裏有能邊泡溫泉邊看紅葉的溫泉旅館八光園及御所覧場，都是小溫泉不設散客。不知道有沒有午飯提供，看官網，酒店有鮎（あゆ）魚料理。這是日本一種河魚，不便宜的，raku老師喫過野生的，有一種特殊的香味。我一直很想喫，後來終於在 《絕景梯田湯來百年古民家之旅》喫過獲獎的。

另外，這裏有純營地立久恵峽 わかあゆの裏。

多岐特產

無花果(日語いちじく，即kirara附近溫泉的名字），我們去的時候是當做季節。道の駅有很多，但後來發現超市（只有一家超市叫ラピタ）隔壁的無花果園能直接買，便宜很多。
