三原旅游官网（岩海所在地）
政府旅游报告：(2019)三原市観光戦略プラン(観光課)

西条早餐店
西条除了コメダ(komeda)，早餐店我去过的有Ncafe(只限周末)及hana cafe，都在国道375的御薗宇东侧。两家都不错，我喜欢Ncafe多点，我们这次也在这里吃。

久井岩海
日本最大规模的岩海，我们去的假日不过人很少。当时那里厕所没水，蜘蛛也很多。

岩海那边附近没啥吃的，要提前找好。我在地图标了两家(绿色)。想吃好点可以去北面的世罗町。

公园高尔夫球(パークゴロフ)
有9个洞，每个洞打的距离不到100米。没打过的话挺新鲜的玩意。我们当时想去的白龍湖観光農園 因为开错车没去成。这门票1000，租用球具不知道多少。另外东广岛福富有政府场地（道の駅湖畔の里那里）- 東広島市福富パークゴルフ場(西条以北，开车约半小时),是公認コース,门票1040，另租用球具310。一圈9个洞,再玩要再给钱。你不懂得话职员会教你规矩。分数是谁打更少的杆进洞就赢，有计分表提供。这个地方经常有比赛，去的话最好先打电话。
