1.疫情下上山下海遊愛媛 - 廣告

日期：2020年7月（4天3夜）

行程：

D1:西條 - 來島海峽大橋 - 久萬高原町

中午西條出發，經連接四國及本土的來島海峽大橋欣賞風景。到達愛媛先買菜，然後入住久萬高原ふるさと旅行村小木屋準備晚飯。那裏位於森林高原，天氣稍涼。有羊駝及東京愛的故事物品看。天晴晚上可以觀星。

D2: 久萬高原町 - 八釜甌穴羣 - 四國喀斯特 - 明浜海邊大木屋

八釜甌穴羣乃特殊地理形成多個甌穴，最震撼莫過於響亮流水聲，是要看也要聽的景點。
四國喀斯特乃當地著名景點，全國三大喀斯特之一。我們在當地午飯。由於在高原1400米，其行車道被稱爲“天空之道”，邊開車看喀斯特景色邊看高原牛。逛完喀斯特喫高原冰淇淋，隨後下海入住海邊100平米大木屋(あけはまオートキャンプ場，晚飯喫燒烤。
四國喀斯特
一百平米大木屋
大木屋面向海灘

D3: 佐田岬半島-龜池溫泉

早餐後出發往愛媛西邊狹長半島佐田岬，路上經過高科技設計音樂公路(メロディー道路)。只要車速合適就能聽到悅耳音樂。午飯當地餐廳喫當地海鮮，順便預定伊勢海老套餐下週末家裏喫。隨後到浪漫滿分四國最西端的燈塔徒步(1.8公里)，觀看戰跡。
浪漫燈塔
炮臺
爲了解除幾天疲勞，下午將泡當地人氣龜池溫泉併入住那裏，晚飯也在龜池溫泉。龜池溫泉米乃仿古建築，裏面各種池特別多，泉水來自地下1500米。
人氣龜池溫泉

D4:龜池散步 - 白豬瀑布 - 返程

早餐後到有各種生物的龜池散步，觀賞螃蟹，隨後到東溫繡球花盛開的県立自然公園,落差100米的白豬瀑布。午飯後返程西條。
到處繡球花的白豬瀑布

白豬瀑布

成本估算：
高速+油費：5500
三天住宿：12000/4 + 16500/4 + 2500 = 9625
燒烤用品：450
必要開支 ～16000
另外加 3早，3午，3晚，總成本合共約 ～ 26000
