2.瀨戶貴賓列車尾道休閒一天遊 - 相關行程信息

(所有價格2024年底更新)

我們的行程地圖：
(紅:登山口   藍:景點    綠:喫飯   )


尾道旅遊官網(一切你需要關於尾到喫住玩信息)
政府旅遊報告：令和5年尾道市観光客統計について(観光課)

車票購買攻略

售票機買票操作：みどりの券売機で観光列車etSETOra（エトセトラ） 広島→尾道のきっぷを買う （我看過纔去買的）

這趟列車只有兩節車廂，週末挺不好買的，所以我們平時去。當日車上有空坐，不過海側位置滿了。提前一個月買票，我們2月24日的票，車站售票機是1月25日開始售賣。我選了兩個最好的座位(預先看好官網座位表)，是第1節車廂10A及10D。我買的グリーン(green)券，即額外坐這車需要的票，1000日元。另外加上普通廣島-尾道車票（1520日元,可以坐車時纔買）。坐這趟車總成本2520日元。

綠色是1000元事前買的グリーン(green)券

午飯
 這裏半山喫午飯，可以看瀨戶內海的地方有兩個：1)>cafe holoholo，網上看是開門時間不過當時我們去沒開。2)cafe holoholo附近的酒店-尾道ビュウホテルセイザン喫泰國菜，只有週末開。 尾道駅附近的商店街也很多喫的。

千光寺公園
尾道駅北口出來走一百米左右就到登山口(公園西側)，有好幾個的。攀升100多米，半小時內到頂，全程有帶扶手的樓梯，很簡單的,然後從東邊下來經過貓之細道及千光寺新道。

咖啡店下午茶
原計劃去評價很好的高檔咖啡店satie,不過又沒開，最後去了尾道商店街的coyote，順便逛尾道商店街及尾道商工會議所參觀。尾道特色咖啡店很多。

平原ぽっぽの湯(溫泉)
位於車站東北3公里的。這個溫泉看網站玩的很多，特點是女士有漢方藥草桑拿。去這需要在車站坐公交車，詳情查詢navitime或おのみちバス-->路線バス時刻表各路線-->ＪＡ尾道総合病院線、市民病院線。

溫泉平日860円・土日祝日1000円

上面google地圖都標了上面的點及其他信息。

注意事項: 這裏不是大城市，小店不靠譜的。這次經歷了兩家店網上說開，實際都不開。下次去這種店最好先打個電話問問。
