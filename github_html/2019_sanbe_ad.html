1.深秋三瓶石见红叶之旅 - 广告

日期：2019年11月（2天1夜）

D1: 西条 - 石见银山 - 北の原キャンプ場 - さひめ野温泉 - 火锅晚餐 - 观星

早餐后出发，到岛根世界遗产石见银山。此乃战国到江户时代日本最大银山，高峰期产量达到世界三分一。现在景区街道当时就s保留下来。 我们先于当地午饭，在当地逛各个景点，然后租用电动自行车到骑森林小道到当时矿道龙源寺，深入感受矿工当时开采的艰难。
保留当时景色的石见银山街道
林间骑车
还能看到当时徒手挖掘痕迹的龙源寺矿道
下午我们将入住三瓶山脚下的北の原キャンプ場最大的木屋，然后到さひめ野温泉泡温泉。晚饭木屋吃火锅，饭后观星。
我们住的森林木屋
D2:早餐后前往三瓶山登山。登顶需时约两小时。山顶360度美景，一侧是日本海，另外三侧是无穷无尽的中国山地，非常壮观。
成本估计：
车费～1200
石见银山电动车*龙源寺门票：700+200（外国人优惠，原价400）
住宿：3000
温泉：500
必要开支 ～ 5600，另一早二午一晚，合共～10000
