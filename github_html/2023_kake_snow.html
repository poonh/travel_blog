活动记录 - 留学生活动之加计雪体验

日期：2023年2月10日（周五）

这是广岛国际中心（広島国際センター）的官方活动。该中心其中一个工作就是给留学生办活动，让他们体验本地。今次参加的是去安芸太田一所高中（加計高芸北分校生）和学生交流，内容包括雪体验，看学生表演神乐，体验神乐服装及留学生表演跳舞。这次活动还有中国新闻来采访（神楽や雪合戦で留学生らと交流　広島・加計高芸北分校生）

早上于东广岛出发，大概10：30到校。今天是上好的日子，去年这个活动因为没有雪变了室内活动，但去之前一天刚好下过雪，大家玩的很开心。

这个学校海拔较高，我们开车经过森林，一路是雪景。

 

今天因为有客人，他们不上课。到了他们就在门口欢迎我们，跟我们挥手说hello,之后校长致辞，我们就开始。

我们人数大概三十人，分成约7人一组，之后分组活动。这样的小组形式，更容易和陌生人熟悉，毕竟里面好些人是一个人参加活动的。

我们开始雪体验了。这里的操场全是雪，大概有几十里面米厚。我组首先进行雪战，学生给我们弄好雪球了。我不小心被打到脸，而很痛！

厚厚的积雪
雪球
准备作战了
进攻！

雪战完毕，我们开始玩雪人了。我学了这个日语ごろごろ，意思就是咕噜咕噜，让雪球在地上滚大。首先用一个小雪球，压实，然后就ごろごろ。

ごろごろ
我们的雪人
大家看到几个雪人？
大家随后玩起雪战来
他们还给我们挖了冰道，用来拍照呢
很好的拍照效果，好像去了北极

第三个雪体验就是雪滑梯了。我们每人都玩了好几轮，十分刺激呢。

 
冲啊！
大合照
到了中午时候，我们就在学校吃便当。由于有伊斯兰，便当都是清真的。 学生们还给我们做了用本地苹果榨的苹果汁和团子（だんご）呢。看到这些我感受到他们的爱心
大家一起开心吃午餐

下午活动就是神乐表演了。我一直很想看，之前有个机会但因为新冠肺炎没看成。这次非但有的看，还可以随便拍照，更棒的是还可以试穿衣服。他们的表演真很有水平，这个龙舞特别好看。

神乐乐队
演出开始
很精彩的龙舞
观众都看得很投入

完了大家试穿龙舞的衣服，原来很重的，所以龙都是男生。他们练了一年，一周练5天，很不容易的。

学生帮我们穿衣服
音乐学习

随后就是留学生的舞蹈表演，有孟加拉，尼日尼亚，还有几位印尼妹妹教大家跳maurere dance。

孟加拉学生开始唱歌时，日本学生就马上一同鼓掌
尼日尼亚舞蹈
大家一起跳印尼舞
最后就是学生们和花田舞太郎（当地吉祥物）一起跳舞。当舞台郎不容易，上下楼梯都要人搀扶的。当时所有人都参加了，玩的很开心。
大家一起来跳舞
很多人都和舞太郎合照
最后大合照
临走前和学生合照

由于今天有体力劳动，大家回程都很累，不过真的非常开心。丰富的雪体验，和本地学生交流，认识新朋友及尝试不同国家的舞蹈。非常棒的一天！
