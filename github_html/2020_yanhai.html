3. 初秋东广岛岩海小资一天游 - 游记

 日期：2020年9月21日

原来打算上午去久井岩海，下午去打哥儿夫球（Park Golf,不是正式那种，每人只要1000日元左右） 。不过由于路上经过机场，那里开车极其混乱，最后时间不够，只去了岩海(我会在这章补充福富的公园哥儿夫)。不过我们也很开心呢。这是洗碗王子离开西条前最后一次出门，趁着今天假期，天气也凉快，好好享受初秋之旅。

为了小资一把，早上8点出发，去N cafe吃早餐。很喜欢这个店呢，我吃了丰盛的800日元早餐，咖啡还是带拉花的，大家都好享受好满足。

餐厅在2楼，要上个楼梯
 
里面装修很别致
 这份是650的
这份是800的
 

9:20左右就启程出发，疯狂绕错路开始....不过还好最后到了，不小心上了告诉都差不多回到西条，多花了超过半小时。久井岩海有停车场，里面也有厕所，不过没水。所以去之前在7-11或超市上好（地图有标记）。最后去到岩海我记得11点多了....

开车路上的景也很不错呢
 
我们终于到了，门口来个合影
 
跑题，停车场看到一辆跑车，不过不是法拉利
 
进去首先要走一条短的森林路
 
之后就可以看到岩海，不过我们首先科普一下
科普的东西看了我也不太记得了，反正这里是日本最大级的岩海。整个景区分开几块，都叫“金钱龟壳”，大的中的小的，还有一个水音峡。进去第一个是最大的，几百米长，最壮观。整个景区不大，我们看看玩玩又拍照，大概1个多小时。如果你不拍照，不细看，半个小时左右也可以走完。另外，这里的蜘蛛也是很好看的呢，都是五颜六色的花蜘蛛，甚至看能就在你的眼前，不知不觉间就冲破了（我就是这样）。这里植物种类丰富，还有灵芝，木耳等。

有了经验，今次洗碗王子带了muhi来（日本防虫药）以便有虫咬用。上次爱媛之旅第一天他被咬，现在还有痕迹留在肩上呢。
 我们现在只看看景吧。
这是水音峡，走近的确听到流水的声音，虽然照片看不见

看完岩海，我们现在看看其他地方。
这里还有很多不同植物，我们看到灵芝和木耳
断的很整齐
路上见到一个特大的蘑菇，好像面包，我以为是假的
于是我很邪恶的去搞它，把它 挑起来，看见根了，顿时觉得自己很邪恶.....
好好的超特大蘑菇就被我破坏了。

我们现在看看那边的动物呢

这只螳螂向我们说:“万岁万岁万万岁!”

蜘蛛是最精彩的了，我第一次看到蜘蛛在吃蝴蝶，还看到被吸干了的蝴蝶，只剩下一个空壳

这张正在吸食（这是蝴蝶，不是树叶)
这个蜘蛛网就挂在天上
还有这个不知道是什么...
考眼力，蜘蛛在哪？中间偏上
蝴蝶
这个好像用来抓什么的

逛完岩海，我们去吃饭了，原来想吃的日式午餐今天关门，就去对面的西餐。今天假期，等了一会。

这个餐厅的木柱子好像我们之前在明浜住的大木屋。
这家午餐比较贵，很多1000多，由于现在已经晚了(2点左右)，今晚又早吃饭 ，我们就点了便宜点的简单午餐。
今天又是开心的一天
吃完饭我还喝了杯冰咖啡才走。由于来不及打哥儿夫了，就直接回家吧。晚上 五点左右开始吃火锅。

为了欢送洗碗王子，我给他准备了礼物 - glycolic acid去痘印套装。做 glycolic acid换肤不是那么简单的，我这是根据文献的做法，把一切需要的都 一站式给他了。
皮肤好了好多的洗碗王子，不过俏脸不能公开。
好了，到公园高尔夫了。我打过两次パークゴロフ（park golf）。我是无意中在地图上看到这个地方，然后就想去玩。第一次平时早上去，都是爷爷奶奶，很热闹。另一次周末下午，反而没人，就两个家庭。那里环境优美，这个玩意新奇有趣。第一次和onion学长去玩，他竟然玩的比我好（唯一玩的比我好的运动）。

应该是2021年秋天。平日早上很热闹。
我来了
进洞了！
和raku老师在冬末打过，她学过哥尔夫，打的很好
