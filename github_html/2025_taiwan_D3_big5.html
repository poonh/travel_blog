今天行程:
早餐後出發，先坐3站捷運到高雄站，然後轉乘台鐵到屏東的慈鳳宮，看完打車到1公裡外的屏東西市場午飯。慈鳳宮位於屏東站附近，有400多年歷史(明朝)，富麗堂皇。去這裡主要是因為原定第一個行程去屏東西市場午飯，然後早飯吃完馬上午飯吃不下，我看慈鳳宮在車站走三分鐘就到，所以加插這個行程。

至於為什麼去屏東西市場呢？這不是遊客區。話說不知道那裡看到我們D1去的龍華市場沒冷氣（我之前沒有想過的，因為在香港到處都有冷氣），我就查D2去的華僑市場到底有沒有冷氣。我搜的關鍵詞是「屏東 海鮮市場 冷氣」，然後搜出來這個去年落成，因為有冷氣有很多新聞介紹的屏東西市場（這是屏東縣首個有冷氣的市場）。我一開始也不知道原來不是我要查的地方，後來發現這個市場和今天的行程順路，看介紹環境好，一樓買東西二樓可以吃飯，廁所也乾淨。最重要是舒服，我就加入行程了。去完之後，如果你想吃台灣市場小吃，希望坐的舒服又有冷氣，尤其想買台灣本地咖啡，我十分推薦這個市場。這也是我們行程吃的舒服便宜又好的一頓。

午飯後去百年歷史的三和瓦窯。這是個瓦窯廠，可以DIY。這位於大樹鐵橋濕地，兩個地方可以同時去。最後去台灣鳳梨工場，是日佔時期唯一還保留的鳳梨工場。推薦導遊團（我參加的16:30那場），可以學到日佔時期歷史，台灣鳳梨史及這邊的地域歷史。

車站有個廣場

廣場的黑鮪魚攤

高雄車站和日本的一樣，同一個月台的車都是往同一個方向的（意思是我們去屏東，直接去1號月台等車就可以，其他月台肯定不對的），不過你在台鐵網站查不到那個月台上車，日本的鐵路網站可以。在日本，即使是東京多個月台的超大車站也可以做到同一個月台都同一個行車方向，這其實很方便。對於遊客來說，能事前查好上車月台可以大大減少坐錯車概率。但中國的則不是，得快到站是才知道什麼月台。在中國，不知道困難點在哪。

16:16，我們打車去台灣鳳梨工場，2公里多，90元，10分鐘到。到了我正好趕上16:30的導遊團。導遊團需時40分鐘，內容覆蓋日佔時期歷史，屏東簡史及台灣鳳梨史。我喜歡參加導遊團，帶著知識去看印象更深刻。這裡現在三個建築（都有空調），一個關於台灣鳳梨歷史，一個關於大樹文史，還有咖啡廳。如果你只是自己看，拍拍照，半小時內可看完，然後可以享用這裡的下午茶。

從三和瓦窯到鳳梨工場這段路有回到日本感覺

鳳梨田，看了這個田才知道原來鳳梨種地上，不是樹上...這個鳳梨田種了台灣不同品種的鳳梨，台農1號到差不多20號。導遊對每個品種都介紹了。牛奶鳳梨果肉是白色的，沒牛奶味。還有蘋果鳳梨，芒果鳳梨，有能直接剝皮的鳳梨。有些品種農夫不願意種，因為很快爛掉或者葉子太扎手。最普遍的金鑽鳳梨是因為容易打理，所以農夫願意種，但其實甜度不特別高。日佔時期時，台灣引入了夏威夷一些品種，即「土鳳梨」。我們買的鳳梨酥就是土鳳梨的。

導遊介紹

今次參加導遊團，我學會了「台農」產品。我查到農業部農業試驗所研發了很多產品 。吃的有草莓，土豆，大米，花生等。網站都有每個產品簡介。日本農業技術也很發達，我在日本的遊記也介紹過一些農業產品的比賽。看到這些我不禁疑惑，去中國的超市怎麼都沒見過農業技術研發出來的產品？

台農4號剝皮鳳梨

日佔時期台灣有很多鳳梨工場，現在只留下這家。原因是這裡曾經是眷村，有人住就保留下來了。這就是當時浴室的牆。

當時的窗戶都很大，為的是通風。窗戶往上抬不會掉下來的，為什麼？

其實是有科學原理的，導遊指的部分有個砝碼，用槓桿原理窗戶就不掉下來。現在台灣有日佔時期建築可以看到裡面的解構。
