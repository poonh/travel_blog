早上七點起來喫早飯，喫的深海魚皮炒飯，非常香。第一次在日本喫魚皮。這個深海魚的皮很筋道，很好喫。

黑色的是魚皮，還用了之前一天晚飯生魚片的配菜洋蔥炒的。

 


兩大碗我們6個人喫，都喫光了。

感謝洗碗公主爲我們服務

 

 

8點過，我們準備去釣魚了。今天要釣的是オヤビッチャ ，色彩斑斕的斑馬魚。這是我們的午餐。

 

先去倉庫取漁具

 

 先和行李車合個照。


然後就在這裏釣魚了。一邊釣，一邊聽海鷗的聲音。


準備就緒

 

 

開始了!西內先教我們如何釣。


然後我們自己來


  西內協助我們釣魚


 洗碗公主今日獨佔鰲頭，一個人釣了3條，成了魚殺手。

好興奮啊！

 

 我釣了河豚


 

河豚特別乖，在手上不會掙扎的。我玩了一會就放回去海里了。


最後，今天釣魚榜的冠軍（右，3條）及亞軍（左，兩條）合照。

 

 雖然這裏的海並不深，不過還是有大魚的，下面這條顏色夢幻，非常大的，估計好幾公斤。


 

這是我們釣的魚，午飯每人一條。


之後就要開始漁民生活，自己徒手宰魚。西內先示範，然後我們自己來。刮鱗。刮內臟，除鰓，清洗血水。由於太新鮮，心臟拿出來的時候還在跳。

 

 以下是血腥視頻，怕的不要看。
