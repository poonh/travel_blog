金秋中山道二戰松代大本營之旅 - 遊記

前言：這次行程主要是去松代大本營，我快離開東京了，松代大本營一直很想去，但又有點遠，不太想一個人去，趁着rgg來就一起了。畢竟那麼遠，只去一個地方不值回車費時間成本，之後我就在長野旅遊官網選地方了。一開始很想去阿爾卑斯山，不過十一月中下旬不是季節，長野很多登山的地方那個時候去太冷。選來選去，馬籠宿-妻籠宿-南木增（中山道，合共11公里）這條徒步的線路很合適，車費成本時間上也可行，所以就選定這裏了。

行程：

D1:下午4：50從東京新宿坐大巴去中央到馬籠宿(Chuodo Magome)，計劃9：20到，實際9：30到。
（車費（提前約一個月購買特價）：3300，住宿（含早餐）：5300）
(車票購票網址：https://www.kousokubus.net/BusRsv/ja/ )
(住宿：馬籠茶屋，http://en.magomechaya.com/，官網booking都可以預定)
D2:馬籠宿-妻籠宿-南木增徒步，全程約11公里，當中馬籠宿妻籠宿8公里，早8：30出發，最後坐14：44的火車從南木曾到松本，再接17：34的車從松本到長野）
（住宿：長野車站附近Moritomizu Backpackers,https://www.morimizu.net/english/,booking.com也能預定）
（車票（南木曽-長野）：2590，車站現場購票；住宿：3700，午餐：780，咖啡：250，晚餐：2300，包子約~300）
D3:松代大本營+松代城
（車票（長野-松代公交)：650,松代-東京大巴：3000，午餐：1000，晚餐：1900，博物館門票：200）
（松代-東京大巴: http://willerexpress.com/st/3/en/pc/,只有這家公司有車從松代返東京）
總成本：25270
（成本分析：住宿：35.6%，車資：37.8%,喫飯：25.8%，門票0.8%）

2017年11月19日週日
行程安排：松代大本營-松代城跡-返程東京
早上在7-11買早飯，順便喝杯熱咖啡，之後就準備坐8：15的公交車去松代大本營了，車程約半小時，在八十二代銀行下，車資650元。坐車地點在火車站邊上的公交站，3號站臺，發車方向川中島古戰場。

車先經過中島古戰場的，現在是一個公園，這裏紅葉非常漂亮呢。
車上拍的川中古戰場。

松代是一個安靜的小鎮，到處都是日式別墅，道路寬闊，在這個天晴的秋日很漂亮。
下車的地方，是個酒廠，後來我們進去嘗酒。


路上看的日式別墅。

又有紅葉看。

下車後走不到一公里，就可以到松代大本營了。路上看到指示的（象山地下壕）。

去大本營的路上，這個鳥居有點像京都的千鳥居。

松代大本營的詳細介紹，我另文詳談。很多人知道松代古代的歷史，但松代有另一段不爲人知的血淚史。這裏可看的兩部分:
1.隧道（地下壕，英語bunker，德國當時也有很多），早9店開門，免費
2.博物館，早10店開門，200日元
由於和朝鮮人有關，所以東京的朝鮮博物館也有介紹這個地方。如果時間允許，我覺得先看博物館，再看隧道感受會更深，不過當時我們時間不夠，9點就去看隧道了。門外有紀念朝鮮人的紀念碑，也有說對於戰爭，殖民地的反思，一看原來是個紀念朝鮮人的委員會立的碑。這個博物館我感覺是當地一羣想保存這段歷史的有心人成立的。
我們到的時候約9點10分吧，隧道負責人好像看到我們纔開門的，進去要帶頭盔，現在開放500米，我們慢慢看也看了50分鐘。這個隧道要慢慢感受的，裏面有一些介紹。

​削巖機介紹，在博物館還可以試用

隧道內沒信號的，但有幾個求救電話

盡頭有祈願和平的紙鶴。

這裏溫度低一點，但水沒結冰的。

我讓rgg幫我拍照時，我沒笑出來，我知道這個地方我不該笑。官網也有寫，建隧道的時候死了很多人，要尊重這個地方的。
隧道內很多碎石頭，我檢了一塊回來紀念。每當看見這塊石頭，我就想到那條血淚的隧道。

看完隧道不到10點，博物館已經開門。這個博物館很小，一眼能看完，只有日語，有簡單英語單張介紹。館內本身資料很豐富，他們還有自己的資料室。博物館只有一個負責人，那個阿姨也做導遊，會給你介紹，雖然我們只會一點點日語，阿姨也非常熱情，拿着小黑板給我們講解。日本博物館很着重“體驗”，這個也不例外。這麼小的博物館放了削巖機，當時要三個人同時用的，兩個人在兩邊，中間一個人，一個人後面打燈，這個機器非常重。他們先用炸彈炸隧道，之後就用削巖機削平。當時石頭人手搬運，兩個人大概搬70-80公斤石頭，這裏也有體驗，不過給的石頭只有60公斤。當我體驗過後，對當時建隧道的苦難感受更深。
下圖取自官網，這個削巖機是不對的，應該三個人用，博物館內有專門漫畫描述當時情況，那個就是三個人操作削巖機的。

這裏有過慰安所，90年代拆了，拆了部分木製懸樑現在放在博物館裏。博物館其他部分包括皇室居所介紹，當地老人證言等。那個時候連當地小學生也要幫忙，還有建隧道要逼遷，這些都有介紹。
後來有本地一家三口來了博物館，跟我們說了幾句，她覺得日本這段歷史日本人該正視，日本人都不知道這個地方。
博物館有留言本子，今年換了新的，舊的那個1998年開始，20年就就一個小本子，來這裏的人確實不多，一般是日本人，也有韓國人。
看完大本營我們去看了附近一個神社（好像叫象山神社），這個神社挺漂亮的，也大，我覺得也不比京都的八坂神社差，好處是人很少，可以好好看。
神社看到的小女孩。看她凍得鼻子都紅了。




這個神社（好像叫象山神社？）又是一個漂亮的紅葉公園。

也有竹林

出來我們就去喫飯了，松代這個小鎮喫的不多，我們喫了拉麪，看介紹這家店好像獲過獎，裏面還有漫畫看。


我的拉麪配了柚子汁。

埋單每人喫了大概1000。
喫完飯後我們去了對面酒廠看看，試嚐了一下本地的酒。松代是有自己特色的酒，不過我不太懂欣賞，我記得喝了一個米酒家梳打水的挺好喝。
下圖：酒廠

這裏有石榴樹，很多在樹上熟透裂開了。我找到一個裂的不厲害的，摘下來喫了。昨天偷柿子，今天偷石榴~這個石榴很甜的說。

我們之後的計劃是去松代城跡，從大本營走過去約1公里吧，經過真田邸。真田邸要門票，連附近景點有通票，不過我們時間不夠，也不懂欣賞就沒去。​松代城跡是不用門票的。
在路上遇到一個別墅，屬於古代不知道什麼人的，現在這裏有大本營的臨時展，我們就去看了。

這個別墅也有紅葉看，很漂亮。


別墅其他地方有書法展，我不懂欣賞，不過他們提供紙筆墨硯給你練習書法。


別墅一角

這個日式門，我不會打開，裏面是廁所。

再看一次展覽，玩完書法後就繼續前進了，先經過真田邸，然後到松代城跡。松代城跡不像松本城大阪城熊本城原址還保存的很好，這裏其實只有一點痕跡。不過在這個秋冬天氣，在這麼空曠的地方（這裏四面環山，中間一個大平原，戰略上應該有價值的），來到松代城確實有點感到以前打仗的感覺。

護城河

我們行程結束，先去酒店附近大超市買東西，最後不慌不忙的上車，時間剛剛好。這個松代信州ロイヤルホテル酒店很氣派，不知道爲什麼在這個小鎮會有那麼高級的酒店，只有Willer Express在這裏有站，出酒店門口往左幾十米就看到站牌。


走之前在酒店商店逛了一會，看到這款面膜。

今天天晴，坐車路上很多美景呢。
