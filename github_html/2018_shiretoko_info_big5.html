遊記

網走馬拉松及颱風下的知牀之旅 - 相關行程信息

日期：2018年9月

行程：
札幌 - 網走 - 知牀

我的行程地圖：
https://drive.google.com/open?id=1rOcU9GEylCz8job70Abvak6ymN0INob8&usp=sharing
網走地圖：

知牀地圖：


公交：
新千歲機場到札幌市區JR: (快車40分鐘到札幌，每15分鐘一趟)
http://www.new-chitose-airport.jp/cn/access/jr/timejr/
札幌公交：http://ekibus.city.sapporo.jp/
札幌-網走大巴（北海道中央バス，一天約8趟） ： http://www.chuo-bus.co.jp/highway/index.cgi?ope=det&n=3&o=1&t=69
札幌-知牀大巴（北海道中央バス，一天一趟） ： http://www.chuo-bus.co.jp/highway/index.cgi?ope=det&n=6&o=1&t=73
jtb bus可以買上面長途大巴： https://www.jtb.co.jp/bus/
知牀旅遊官網： https://www.shiretoko.asia/
網走-Utoro公交（網走バス株式會社):
http://www.abashiribus.com/jikoku/ShiretokoAirPortLiner2019.01.19.pdf
知牀公交pass（斜裏バス）: http://www.sharibus.co.jp/s_kip.html
知牀-羅臼公交( 阿寒バス株式會社 )： http://www.akanbus.co.jp/localbu/utoro.htm
羅臼-釧路大巴( 阿寒バス株式會社 )： http://www.akanbus.co.jp/localbu/senra.htm
知牀本地旅行團（旅行團網站都有英語，我們參加的Shinra郵件負責人不會英語，但郵件會用英語回覆）
知牀自行車團旅行社(可以純租車)： http://www.shiretokocycling.com/en/
斜裏バス觀光線（比旅行團便宜，其中一個晚間看動物挺有特色）：
http://www.sharibus.co.jp/kankoubusgoannai.html

遊玩
ツアー 株式會社,我們參加這個): https://www.shinra.or.jp/
星之時間：https://siretoko.jimdo.com/
釣魚團：http://www.troutandking.com/eng/index.html
知牀遊船： https://www.shiretoko.asia/list_cruise.html
上面網站是匯聚了不同的遊船公司，基本都是utoro北上到知牀岬，如果運氣好，夏天在這條路線上可以看到鯨魚的。
オホーツク網走の歴史文化探訪コース: https://www.abakanko.jp/rental-cycle/course-no2.html
如果對從utoro騎車到知牀五湖有興趣，參考這篇遊記，路上一直上坡，沒專門自行車路，不容易的：
https://sportsincycling.com/blog/cycling-trip-to-shiretoko-pakit-bike/
羅臼：
羅臼看鯨魚：https://kamuiwakka.jp/whale/index.php
如果是9月到11月到知牀，可以看三文魚產卵，河流是onnebetsu river（https://cn.tripadvisor.com/Attraction_Review-g1120888-d1912423-Reviews-Onnebetsu_River-Shari_cho_Shari_gun_Hokkaido.html)。有車可以自己開車，不然本地旅行社有旅行團。
