3.廣島百名山嶽山 - 遊記

日期：5月1日（週五）
地點：廣島縣府中市嶽山（廣島縣百名山）

前言：這次登山行程其實換來換去，一開始想去廣島縣第一峯恐羅漢山，但去那要走高速，不走要接近三小時車程，後來看到三倉嶽很吸引，但有一定危險性。我們之前一週爬了白木山，花了不少體力，有人還沒恢復，所以今周玩個簡單的吧。我看到世羅高原那邊有冰淇淋店，心動想去喫，碰巧隔壁有男女鹿山，就是太簡單了點，大概和西條第一名山鏡山差不多，後來不知道哪看到附近的嶽山，而且是廣島縣百名山，看官網一小時就能登頂，攀升才300多米，我就決定去這裏了（但事實證明不是那麼簡單的）。

行程開始：
早上9點多出發，到登山口前去世羅町的超市上廁所，具體地點我在地圖標了，有很多的。我們到斗升登山口大概11點半（其實從西條開車1個半小時就到）。
就在路邊的空地停車就行

另一角度看，左邊是停車場，照片差不多正中心位置是登山口前的牌子（下面有）

我們正式起步前我要先記洗碗王子及公主一功。洗碗公主今次全程爲我們揹着移動 ティッシュ(tissue)，方便我們隨時用。這個新穎的袋子設計，叫它做“先端紙巾袋”。
繼承上次的付出，洗碗王子今次繼續爲我們全程背垃圾袋
我們登山了。起點有填手帳的地方，寫的一手好字的洗碗王子負責寫。
起點拍照留念。

路一開始就是分岔口，我們走的短的那條上去。正式踏在路上，我們發現原來這是野山，地上全是樹葉，當時真慶幸買了登山杖，不然太滑了。路上一般就是下面那樣，不太難的。
偶爾也遇到過繩子，有一段很抖。不過原來以爲我們走另一條路下來，那條路因爲長點，會平坦點，但其實不是......後面再說
路上的標記
 在沒啥驚險之下，1小時15分之後我們登頂了！這裏不像白木山360度景色，山頂沒啥看的，我們就喫喫東西聊聊天，洗碗公主和onion學長還從手機翻出自己的老照片給我們看他們當年瘦的樣子.....
這是山頂，我們在這裏喫飯
 登頂留念
 
好了，想象不到的挑戰在前頭等着我們。上山的路上有點岩石，不過沒有下山那條路好看。下山時有時候路不很清晰，不過看樹上的絲帶標記就行。
下山遇到這個岩石，我們跨過它的後面下去
路有點窄的，不過我們都能穿過
之後就很多扶繩子的路，都已經忘記了幾次了
這段該是最陡的了。曾經有次另一側就是會翻下去那種.....洗碗公主一直要洗碗王子扶着。小z也一直扶我，或者在後面拉我
路上其實還有很多大岩石的，有些是特別景點，更多像這種
後來我們走到一個岩石平臺，那裏看風景很棒！
 
經歷過一段艱辛後，我們終於完成了繩索路，不過最後下山那段感覺也走了很久，有古木蒼天的森林，也有野草路，也有要跨越小河流，很漂亮呢
好了，終於下來了，拍照留念先~

這個山大家都覺得很驚險，但又很好玩，就來一次吧，下次不去三倉嶽了。花了兩個半小時下來，由於有上週的訓練，這次大家都覺得沒什麼。下山後事件還沒完，時間還夠去世羅冰淇淋店的（開車時快四點半了，店五點關門）。
快到冰淇淋店那條路，啥都沒有，真的很難想象在這荒涼郊外會有冰淇淋店，不過地圖是沒錯的，確實有！ 這個小店好漂亮呢，單球好像370，雙球400多，我還買了冰咖啡。這裏的酸奶也特別好喝，無添加的，190一杯。 
 
這裏的廁所特別可愛，放了牛牛
這裏還有馬場，放羊及斑點狗
圖偏左邊有隻羊在懸崖邊走路

喫完剛好五點，人家也要關門了，我們就回去繼續喫火鍋晚餐了。第二天起來，大家都完全不痛，證明上次白木山後大家水平都提升，訓練出來了。我們兩週後就挑戰廣島縣第一峯 - 恐羅漢山。
