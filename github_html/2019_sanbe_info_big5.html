2.深秋三瓶石見紅葉之旅 - 相關行程信息

(所有價格2024年底更新)

我們的行程地圖(線路取自navitime)：
(紅:住宿   藍:景點    綠:喫飯    橙:超市  啡:加油站    黃:廁所  紫:停車場)


大田旅遊官網
我們今次去的地方屬於島根大田。石見銀山和三瓶山是這裏的大景點。市政府下屬有石見銀山課，屬教育部。

我在大田政府網站找到大田市新観光振興計畫（2023-2026）(案)，我簡介一下。主要想看相關數據參考大田市新観光振興計畫（2023-2026）(資料編) 。

到2040年，大田人口大概比現在減少三分一（p2）。人口減少就產生一些地域課題，例如經濟，自然環境沒人保護，文化社區活動減少等，所以要發展旅遊業吸引人來。 大田的旅遊人數在2008年最高，190萬。疫情前2019年100萬，整個島根而言算是遊客少的地區（p10,最高爲出雲和松江，2018年各有1000多萬）。2021年大田觀光消費額25億，僱傭創出效果381人。爲了振興旅遊業，他們定下7個課題(p15)，包括人才培養，觀光資源優化，增加過夜旅客，環境整備，宣傳方式，廣泛合作等。每個課題報告內都有具體政策，其中一些政策制定根據之前做的統計調查。

關於廣泛合作方面(p18)，其中一個政策是和其他旅遊機構聯繫幫忙宣傳。我對這個深有感受，在日本相同類型的博物館會互相幫忙宣傳，你去某家博物館就可以看到其他家的海報。西條辦音樂會的場所くらら也找到附近地方音樂會的信息。旅遊宣傳方面，小地方住宿網站也能找到同一個地方其他住宿（及遊玩）信息，不怕競爭（例子吳市上蒲刈民宿かつら亭)，奧出雲觀光協會的airbnb收錄的民間住宿，我覺得也體現了一種官民合作。日本這些地方真的很值得學習。大小攜手打造地方發揮更大力量。

政策制訂他們也考慮到旅遊使用者意見，具體不同地區實施計劃在p19。日本很注重意見，參加任何活動（音樂會，イベント等），博物館參觀都有問卷(アンケート)，我想日本旅遊那麼吸引，不停聽意見不停改善是原因。

下面我簡單介紹一下三瓶山那邊我研究過的地方：
石見銀山
裏面好些地方，我們停車在遺產センター(Center)，騎車到龍源寺間歩，兩者距離2.3公里（不能開車)。我們去路線上一些不用門票的景點。以前有錢人的官邸也是景點，我們沒去。

資料室400（外國人有優惠，帶上護照），龍源寺間歩500（也是外國人有優惠）。另外有導遊團大久保間步(費用幾千)，這地方不能自己去的。除此之外，有一些銀器製作體驗。

以前礦工條件非常艱苦，龍源寺間歩是個充滿血淚的地方，開採還死過人，當時一個隊友說一個人不敢來。

租用自行車在おおもり會館，電動2小時700，非電動500。遺產センター(Center)到龍源寺路上有點坡，租電動好。

景區內有飯喫的，我們去了距離停車場1公里的地方羣言堂喫。這裏喫飯地方不多。

石見銀山有導遊協會，有不同路線導遊團，還有英語服務，每團8000。

三瓶山景區信息
官網:https://www.sanbesan.jp/mountain

除了三瓶山登山線路，還有附近的琴引山・大萬木山及石見銀山的。還有住宿遊玩公交車等信息。

三瓶山東南西北都有登山口的，登頂要注意不要隨便選一條路下來。我們從官網北部1.姫逃池（ひめのがいけ）ルート上去，然後從2. 名號（みょうごう）ルート下來。

比較特別的景區是埋沒林，80年代偶然發現的地下森林，成因和火山有關。

住宿
三瓶山北の原キャンプ場

我們住的地方，政府營地，能網上預約，10人棟ケビン(cabin)假日料金23000（我們2019去24000，我驚訝怎麼會降價），可以紮營（帳篷寢具燒烤用品都能租用）,開車去姬逃池登山口幾百米。

附近有天體觀測館，週六對散客開放。

三瓶溫泉
溫泉都位於三瓶山南邊，我們去了最貴的さひめ野溫泉，這是溫泉酒店，日帰入浴500，租浴巾300。

大田溫泉津也很有名，不過離我們遠就沒去。

超市
我們去的美鄉町A coop（6點關門）,營地以南約10公里，靠近我們泡的溫泉。

大田市有Aeon,good day,我都標在地圖橙色。

中國自然步道
如果你想在島根其他地方登山或徒步，參考官網信息。他們還給出停車站廁所地點。
