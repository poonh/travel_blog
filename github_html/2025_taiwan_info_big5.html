2.攜老帶少游高雄 - 相關行程資料

行程地圖：
(紅:住宿   藍:景點    綠:吃飯    橙:超市  啡:咖啡店    黃:伴手禮  紫:下車點)


引言：
前幾個月因為一些事情已經研究過高雄景點。之前都在日本旅遊的我，很欣賞日本官方旅遊網站信息齊全。除了官方旅遊網站， 日本那怕一些小餐廳都有自己網站。而台灣小店以社交媒體為主，對我而言，沒有網站方便。網站我可以容易查找到需要信息，但社交媒體是有問題方便私訊聯繫。網站需要成本，日本那麼多網站不知道純粹靠店家自己還是有政策支持。另外，旅遊信息方面，高雄官網實在不夠詳細，例如田寮月世界，高雄旅遊網對其地形介紹得很詳細，但沒有交通信息。交通部觀光署有交通信息，但沒介紹得那麼詳細。詳細程度和吸引力有關係。田寮區公所倒是有最詳細的交通信息，但一張景區照片也沒。高雄旅遊網是政府官網，下屬高雄市觀光局。像日本那樣，官網收錄齊全的信息最方便，不然像田寮月世界，官網一些信息然後地方政府一些信息，查找資料時實在麻煩。還有就是，類似景區，日本的網站可能有徒步線路地圖，長度，需時等。這樣我好估計我要花多少時間。一個例子，島根的鬼の舌震。縣級官方旅遊網站及奧出雲（鬼の舌震所在地區）旅遊官網都有介紹，但縣級官方旅遊網站收錄了奧出雲旅遊官網的鏈接，這就不亂，而且我需要的信息都齊備。高雄這方面需要完善。

高雄旅遊網有介紹自行車旅行，用的共享單車youbike,但又沒寫那裡可以租。線路寫的也好簡單。去一個陌生地方旅遊，便利性很重要。高雄旅遊網每條路線只是寫了長度，簡單介紹了路上景點，有些照片，對於不是騎行高手的我實在無法判斷是否適合自己，這麼不便利即使我感興趣也不會考慮。這點又要學習日本，像島根縣有專門自行車旅遊官網（日本好些縣都有專門自行車旅遊官網），裡面每條線路都有單獨一頁介紹，有可下載多格式電子地圖，還有海拔圖，適合什麼水平騎手等信息，租車信息也有。這高度的便利性就吸引人參與。

另外，我既然都去了高雄市政府網站，就順便逛逛吧（我好像是去網站旅遊而不是實地旅遊），發現幾個有意思的地方：性騷擾防治專區，高市校園霸凌申訴，高雄市性別平等辦公室，高雄人權專區，LGBT。這些專區反應了政府政策，而政府有政策表示政府對這方面關心。

高雄政府網站我找到施政報告，但不像日本政府網站找到各部門報告，會議ppt，會議記錄，招標文件等。信息公開是透明度，而透明度有助監管。

說了那麼久我都沒進入正題，好了，行程相關資料上了。之前調研高雄旅遊知道很多景點，我這裡就只說我們行程的。

住宿：
我們住捷運巨蛋站附近，選這裡的原因是，一來靠近瑞風夜市，二來有購物商場（漢神巨蛋）。我們D2去的左營北上兩站捷運，D3去高雄車站南下3站捷運，去機場20多分鐘，地點很適中。

巨蛋這邊好些酒店/民宿。在裕誠路（我們酒店所在位置），及裕誠路往東走的富國路，自由路都很多吃的。我們住的世奇商旅是個小酒店，每層大概五六個客房，不太受其他旅客經過打擾。六月底agoda預訂一個雙床間每天2400台幣，含精美豐富早餐。注意，酒店時文創風，每個客房裝修不一樣，如果你住雙床間，定502，這間明亮。酒店隔壁是董師傅麻薯店，除了新鮮麻薯還有盒裝伴手禮（也有鳳梨酥）。

悠游卡：
公交卡，買卡100元，不退。捷運，台鐵，公交車，便利店等也可以用。在機場5號門附近有得買並且加值。高雄機場很小，很容易找到。

捷運：
高雄捷運我很多欣賞的地方，除了內文說的無障礙車廂，適合不同高度人士的扶手，每個車站有編號外。看每個站的網頁（例如巨蛋站)，都有出入口咨詢,站內平面圖,車站位置圖 ,首末班車時間及零售販賣。
巨蛋捷運出入口咨詢截屏，我需要電梯，一看就知道去哪個出口。這很方便輪椅人士，我們拖著行李箱也需要此信息,這讓我的行程更順利,不需要臨時找電梯。雖然旅程還是有忘記了看網站相關信息的情況，但我在此記下來下次一定記得。另外，官網有專門介紹捷運的無障礙設施。我查找關於台灣無障礙時，找到這本書《無障礙旅遊：跟著輪椅導遊玩台灣》。本遊記其他地方我也記錄了我看到的無障礙措施。
巨蛋捷運站內平面圖截屏。一看就知道月台位置那裡下車有電梯（方便趕時間人士），出口間距離多遠。
巨蛋捷運車站位置圖截屏。看到車站在谷歌地圖位置。
巨蛋捷運首末班車時間截屏。看到首末班車通過本站時間，還可以看到末班車是否能趕得上轉乘的車。
巨蛋捷運零售販賣截屏。有什麼急需一看就知道車站有沒有。

總的來說，高雄捷運網站信息很全，很便民。你要看各車站信息在首頁左上的「乘車指南」，然後「各車站咨詢」就能找到每個車站自己的網頁。 不知道當初是誰設計網站內容，參考了什麼地方？寫此文我順便參考了出了名貼心的日本，東京地鐵有站內平面圖及周邊地圖，你可以看到那節車廂出來有扶梯/電梯,也有整個站的立體圖（含不同樓層信息），可以看到廁所，育嬰室，電梯等位置。香港的地鐵有列表能查到那些車站有育嬰間，飲用水，wifi等設施,但我覺得像高雄捷運每個車站有自己一個網頁，列出該站的設施比較方便。港鐵也有每個站的平面圖及出口地圖，但從主頁不好找出來，要搜出來。我也看了深圳地鐵，在「站點查詢」有首末班車時間，車次間隔，站點出口信息，公交換乘信息，有一張顯示車站位置的地圖，但沒有把所有出入口標全(例子大劇院)。深圳地鐵也沒站內平面圖。深圳一些大地鐵站出口很多，而且有些出口間不通。另外深圳地鐵長者免費，長者是有專門出入口的。有站內平面圖會為旅客規劃帶來方便。

捷運官網還有高雄捷運車站置物櫃設置一覽表，裡面有每個站置物櫃位置，不同大小置物櫃數量及尺寸。你擔心自己的箱子能不能放進去置物櫃，看一下尺寸的就可以,最大的能放28寸箱子。這些信息如果補充在每個車站自己的網站可能更方便。

我一向認為，這些細節就體現了領導的水平。硬件要成本，軟件靠智慧。硬件改善生活，軟件提升幸福感。有時候生活就是卡在一些小細節會煩上加煩。軟件做好容易得到人心，日本就是最佳代表。日本官方旅遊網站的細緻，我在本網站的日本遊記已經介紹過。

小琉球：
離高雄本土十幾公里小島，繞一圈約10公里。以前捕魚業為主，現在旅遊業為主。小島陸上有岩石景觀，海裡有海龜珊瑚等，現在很多水上活動。

小琉球人口1.2萬。現在是火爆的旅遊點，2023，2024全年遊客超過百萬,假日單日登島人數上二萬。小琉球是台灣唯一的珊瑚島，現在當然有環境破壞問題，有興趣自己查...

作為一個超熱門旅遊區，我覺得小琉球有些不足。在內文我提到，碼頭附近沒啥吃，要往西走一段到小琉球魅力商圈那邊吃的集中，而去那邊有段斜坡路。我這帶著不能運動的老少在盛夏覺得很不方便。餐廳很多也沒空調。當然台灣很多也是這樣。小琉球這幾年快速興起，商店可能也來不及轉型。午飯感覺難以好好享受（不是沒有舒服的餐廳，只是不太多）。

另外，小琉球另一個問題是海灘基礎設施配套不足。我去過的中澳沙灘在碼頭附近，很多人去的，沒有淋浴廁所小賣部。官網寫的設施也只有停車場。淋浴要去水上活動店家那裡，或者民宿。夏天很多人在海邊玩一天，沒有基礎設施就一個海灘真的非常不方便。政府既然要發展旅遊業，配套就要做的好點。旅遊海灘連這點基礎設施都沒，這裡旅遊發展到底如何規劃？

評論完後，我要到正題。到底如何去小琉球？

1）屏東客運9127D（左營站出發去東琉線碼頭）
在klook買票(上車時掃碼)或刷悠游卡半票61元。現場現金買票122。我klook買的週五，去程8:30最後4張票。返程18:00更是沒票。但我們去程的車坐不滿，而我們的車很多人刷悠游卡（不能提前訂座的），我不知道是不是klook預訂部分的票，現場票另外佔一部分。如果你klook訂不到，可以現場試試。現場即使大巴沒有，很容易找到拼車的，參考價200/人，如果想預約拼車或包車參考小琉球聯盟。我們返程從碼頭包車到酒店1000元。計程車的話，我用試算器約1200元。9127D那裡就有計程車站。大巴站的位置具體看內文。

東琉線碼頭是9127D左營發車後的首站，我們當時坐了55分鐘，路上不太堵車。順便說說，尾站的大鵬灣也是景區，有專門的自行車道的。

2）東琉線碼頭去小琉球
碼頭有三家船公司，分別為泰富航運,藍白航運及東琉聯營。我一開始想定東琉聯營，因為看它班次多，但後來看google評價發現藍白最快，大概20分鐘能到（其餘兩家30分鐘），而且船很新，我就選擇藍白了。船票klook預訂,現場換票。有點非常不好，就是不能預訂確定班次，只能預訂某天的船。職員告訴你，想坐哪趟船，提前30分鐘去排隊。我看評價，假日人多的時候要提前1小時排隊。這實在是非常不利於旅客規劃的。雖然船公司說人多會機動增加班次，但如果上一趟滿了，只剩下幾個人沒上，你會為這幾個旅客臨時增加班次嗎？還有就是有些旅客下船後可能要趕下一個行程，上不了預定的船就耽誤了。小琉球旅遊已經發展了好幾年，已經累積了一定旅客數據。船公司本身每天售多少票，什麼時候增加了什麼時刻的班次也有數據。難道不能用大數據做模型看什麼時候該增加班次嗎？難得去一趟旅行，前後還要花一小時排隊等船，這就減少了遊玩的時間，這實在很不好。船公司不會用大數據，有關旅遊的政府部門會不會協助？幾家船公司都是民營，發展旅遊要政府大方向。要做的好，雙方需要協調的。

特別提醒：取船票記得帶護照，klook沒提示的。

3）小琉球上
我們只是碼頭附近活動，走路就行。很多人租機車（不用預定也行，現場很多租的），也可以租汽車，也有包車。聽說電動自行車也有。小琉球本身有環島公交601。如果你坐601，最好下載ibus app，方便知道車什麼時候到。

3）小琉球水上活動
klook很多，獨木舟，SUP，水上自行車，浮潛，遊艇出海都有，不詳細介紹。說下半潛艇(40分鐘航程，主要看海龜，適合我們不同年齡的)，有藍瓊號（260台幣）（他們官網連預訂方法都沒，想定可以去kkday)及探索拉美（380台幣）。我看網上照片感覺探索拉美的船比較新，也乾淨，就選擇探索拉美了。我在klook訂完要facebook私訊他們預約具體班次，然後當天換票。

華僑市場：
位於東琉線碼頭隔壁。可以買海鮮給加工店（參考Google評價，有些店說的會坑人），也有生魚片一類。除了海鮮，有飲料（蓮藕茶是特產），乾貨等（雙潤膏，櫻花蝦，魚松等）。中午左右11點開門，六七點關門（沒晚飯的）。每月第一個週二公休。

注意，黑鮪魚只有4-6月才有。金山角（最肥美的部位）真的好吃哦。



計程車：
我用uber，不用手機號也能叫，或者小黃車55688 APP也可以。我每次用Uber叫車最後都是叫到小黃車。我事前都用大都會車隊的試算器估價，很靠譜。大都會車隊也有app，但要本地手機號註冊，註冊後送優惠券。
