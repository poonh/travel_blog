游记

福冈历史天草极致享受登山温泉看海豚之旅 - 相关行程资料

(所有价格为当年价格)

日记：2017年11月

行程：
福冈 - 太宰府九州博物馆亚洲体验馆 - 定远号（甲午战争)
福冈 - 熊本 - 天草

天草简介：
天草位于熊本西面的岛，又最接近熊本的上天草，上岛及下岛组成。岛上公交不发达，最好自己有车，我们是没有的，加上时间有限，所以就选择公交能到，最近熊本的前岛活动（位于上天草以南，上岛以北，方圆不到2公里的小小岛）

天草地图：



交通：除了从熊本坐大巴，可以坐普通火车或A车（特色火车，不是天天开的，票价约1800，必须预约）到三角，再倒sea cruise的船到前岛，有些人就直接去前岛看海豚之后就走。
A车：https://www.jrkyushu.co.jp/english/train/atrain.html
Sea Cruise: http://www.seacruise.jp/english/
周游bus(下岛）：http://www.t-island.jp/bus
这个周游bus是位于下岛的，需要打电话预定，如果需要的话就留言告诉下面facebook的Yuichi MM就行，她可以帮你打电话预定的。

旅游中心：
（上天草）http://kami-amakusa.jp/spot/index.php?category=4
(下岛)https://www.facebook.com/amakusatakarajima/
负责这个Facebook的Yuichi MM非常热心，我问她有没有用英语玩皮艇的店家，她告诉了我几家，并且帮我打电话去问有没有英语接待。
天草旅游官网：http://www.t-island.jp/en/
美景：
上天草介绍: http://www.city.kamiamakusa.kumamoto.jp/q/list/130.html （这里有天草几个桥介绍)
下岛有教堂的，周游bus好像能到，我对这个没兴趣就没多了解。
天草地质旅游介绍：http://amakusa-geo.amakusa-web.jp/MyHp/Pub/

美食：
海鲜：特色海鲜是海星，不过不是到处都有吃的，而且听说熊本地震后变少了。
上天草海产专门店：https://www.mizunoshokuhin.com/
另外，这里的鸡叫“天草大王”，也是很出名的。
我们在前岛那天的午餐在这里吃：
潮まねきhttp://www.shiomaneki.com/
Liso Terrace(前岛):综合饮食购物中心，环境很好http://www.lisolaterrace.com/
天草海鲜藏（下岛，要自驾，这里也提供不同水上活动及海豚团报名）：http://kaisenkura.com

游玩：
水上活动：SUP(stand-up paddle)为主，但我联系过很多家都不提供英语服务，出于安全理由必须得懂日语，我联系过一家在长崎的潜水教练会英语，不过这个要好几万，长崎也不适合我们。
长崎会英语的潜水教练：https://www.facebook.com/aquafaith1/
其他水上活动预定：https://en.activityjapan.com/
这是全日本各样活动预定的网站，如果要看供应商的官网，就得看日语网站。如果你不会日语的话，最好先联系店家然后再预定。
看海豚：上面的sea cruise，旺季一天三团，10:00，13:00及15:00，航程两小时，现场购票4500，网上预订4000，淡季不是天天有团的，自行看官网资料。天草海鲜藏也有海豚团的，好像2000左右，航程1小时。
徒步：这里山很多，都不高，但会有难度，很适合徒步。用日语搜“西目コース”就可以找到我们当日走过的路的部分。
具体我们走这条线路，从酒店出发
