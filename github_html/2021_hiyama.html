3.隆冬火山一天游 - 游记

日期：2020年12月29日（二）

对了，这个山名叫“火山”，日语ひやま(hiyama)、不是一般的かざん（kazan），所以这个不是喷火的那种火山。我们走的线路登山口就在桂浜温泉馆附近（有厕所），车就停在那里，但登山口不止一个的。

火山海拔400米，走得快的话两三小时可以往返。我们慢，最后花了4小时（包括午饭时间）。如果有余力，可以继续爬后面的一个山，再攀升100米，具体线路看yamap。整体来说，火山不算难，卖点是最高点可以坐着拍照的大石头（见下文）。 整体而言，路线算容易，没有难走的地方，消耗体力也不多。

今天是近期难得的好天气，晴天带云，直到晚上才下雨，第二天就下雪降温了。今天白天最多十几度，我多穿了，搞到路上出了好多汗。

早上我们九点出发，车程1个半小时到，我们大概11点正式登山，开车路上经过海边的7-11，之后很顺利就到桂浜温泉馆的停车场了。

温泉馆门口
我们先来欣赏一下pp的新装
我们就从这里走去登山口
当你看到这个屋子，往右走就是了
看到这个小牌子，跟着走就行
这就是刚开始那段路了
路有时候有楼梯，有时候不是
走了半小时，就已经走了三分一的路了，在第一个休息点拍照先

用了美颜的效果就是好啊！
虽然是冬天了，不过树木的颜色还是很好看。我们路上还看到大松塔和小松塔，路上还有很多其他美景
后面不知道什么名字的植物成了个很有特色的背景
“女子汉”
搞些松塔做圣诞圈
松塔戒指
这个也能做圣诞圈装饰

看些风景照~

2018西日本豪雨，吴是重灾区，这个估计就是当日的痕迹
这里的岩石很好看，都很大
火山一个特点就是特别多宗教雕像
慢慢走上去，开始见到海景了
又是芦苇（上次在平尾台见过很多的）

1点左右，我们快到山顶了，走最后一段路前有个亭子，我们就在那里吃饭（这里忘了拍照），当时ln还带了橘子给大家吃呢。亭子那里有两条路可走，当你面朝大海，左边一条路是往另外一个山的，右边就是去山顶的。我们选择了去山顶。走到这里，我一直出汗....

登顶了！
地图介绍，左边是爱媛(西侧)，右边是山口（东侧），海是朝南的。眼前很多小岛
这里好美，先凝望一下~
模特和摄影师
这里的重点是这个石头，这才是火山的最高峰。最上面的石头虽然很大，但不是很平的...
我们先琢磨一下如何上去
要拉绳子上的，出力！
继续努力~
最后一步了，要爬梯子上去

  好不容易终于上去了，耶！

耶！拍这张的时候腿其实在斗
来个自拍

raku老师没守规矩，选择危险的路上来，吓死我们了！

 
救命啊~
终于上来了~
我们也邀请你一同上来~

虽然大家一起坐着，但还是有点怕的。不过怕也要拍照。

 这次之后我爱上了美颜~
全家福
坐在石头上面，面对濑户内海，眼前的景象真的很适合沉思，尤其当时没人，我们可以尽情享受。当时天气有点朦胧，我们看见远方的山和云分不清，非常有意境。

 下面是爱媛，最远处就是山云分不清的地方，看视频就比较清楚。

山顶的另一侧是吴，这里是两边都是海，可以看到三个县
我还看到一个像运动场的地方，我特备敏感...

这个地方很美好，尤其是只属于我们（我们在路上遇到一些登山客，不过我们在山顶的时候完全没其他人）。第一次坐在山顶的石头看风景，很特别，以后都会记得这个地方。这里也果然是山顶，我终于觉得冷了。

我们要下去了，由于路不难走，我们下去也很快，大概一个多小时到了。最后一段路的树叶特别多，pp当了小尼姑帮大家打扫（扫把就是放在那里给人扫树叶用的）。

下来当然还要晚饭，今晚我家里吃火锅。锅底是韩式的海鲜清汤，配自制的韩国辣椒蘸酱，raku老师还把挺辣的辣椒切的非常碎把蘸料加工，很给力。

除了这些我们还买了很多其他的菜，今晚海鲜很多
韩式海鲜汤锅底

学长送的新锅首次出境，大家以后在我的游记会常见到它的。没有洗碗王子的日子，今次感谢raku老师洗碗。

今天隆冬登山非常开心~这个假期度过了满足的一整天。
