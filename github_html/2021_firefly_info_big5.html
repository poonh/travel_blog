2.比看電影更幸福的螢火蟲之旅 - 相關行程信息

行程地圖(無法顯示點擊此)
藍色是看螢火蟲的地方，紫色是停車場（含路邊停車）


整個廣島縣很多看螢火蟲地點，具體看官方旅遊網站，搜“ホタル”。 離西條最近，位於八本松靠北的志和是螢火蟲勝地，聽說不太好停車我就沒去。那裏有ホタルの宿自然學校，參見facebook。

我們去的竹園，每年都有螢火蟲活動：おなしホタルまつり(竹原市小梨小學校),內容包括:

・わかたけグルメコーナー(竹筍美食)
・ホタル教室（螢火蟲教室
・ビンゴゲーム（遊戲
・ホタル観賞（螢火蟲觀賞）
・コーラス大會（合唱)

竹園旅遊官網有活動信息(イベント)。

以後，如果想看政府的旅遊報告，參考2024年竹原市観光振興計畫。
