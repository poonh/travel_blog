攜老帶嫩遊富士三湖 - 西湖河口湖山中湖 - 遊記

前言：這次遊玩風格和我以前完全不一樣，由於攜老帶嫩，是超低運動量的。不過富士山可以運動的地方，不玩我還是查了一下的,見上一章。

行程地圖，紅色是我們沒去的景點。

行程：
週五：西湖裏根場閒逛及穿和服
週六：河口湖大石公園+中午山中湖喫餺飥+下午河口湖天鵝船
週日：山中湖半程馬拉松
開支（日圓）：
D1:兩天公交票-2550，住宿-每人約7800，和服-1000，紅薯下午茶-550，里根場門票-300，晚餐-約1000
D2:早餐 - 約250，冰淇淋 - 500，午餐 - 約2100，晚餐 - 約200，天鵝腳踏船 - 1000
D3:早餐 - 約250，往返山中湖公交 - 1560
全程合共：19060（公交：22%,玩：12%, 喫：25%, 住：41%）
哦，對了，其實這次是一次半馬之旅......
2017年5月26日週五：
行程：西湖裏根場
原定早上9:50的車，由於帶着小朋友加下雨，出門需時估計錯誤，快到點時我一個人冒着小雨飛奔過去車站，車還在，但馬上要開了，我跟職員說後面還有人，但不能等了，之後職員帶我去改票，不用錢，不然的話應該是要錢的。

後來換成11點多的車，喫了早飯纔出發。由於車晚點，我們又喫的飽，到了也沒有午飯。
到了先買兩天公交pass,2550日圓，6歲以下小朋友公交免票。我們放下行李就從河口湖車站坐車去西湖裏根場了，公交班次半小時一趟，約半小時到。

里根場不大，是仿古的村落，裏面大概20個店。那些店你不都不進去或者很快就看完的話，也許不到半小時逛完。

我打印了網上優惠，成人門票300,6歲以下免票。

進去有些田，看到農村景。
（下幾圖：公交站至里根場入口一段）



里根場裏面有一點點坡，門口還有柺杖提供。

日本的手信就是很漂亮很精緻，這裏的禮品還什麼都是富士山的。

走着走着就到和服店了，1000日圓比京都便宜很多，一看到漂亮的衣服孩子就想穿了，我和她媽媽也玩了，長頭髮還另外給頭飾，小朋友還可以配個包。和服也有男裝的，是武士裝。





由於這裏復古，加上當時人不多，拍照效果是不錯的。天氣好時西湖能看到富士山，但今天下小雨，煙雨濛濛的又是另一番景象。


我們坐約5點的車回去，這個時候餓了，買了個紅薯，挺貴的，半個要1100日圓（按重量），但分量也不少，我們兩個人分着喫。老闆看見有小朋友還送糖。
晚飯在住的地方附近喫，一個自家經營很鄉土的小店。炸的豬排很新鮮呢。




回去洗洗就睡了，今天第一次穿和服呢。
2017年5月27日週六：
行程：河口湖大石公園+山中湖餺飥+河口湖天鵝船
今天晴天，我們住的地方可以看到富士山，起來就看到。

原來計劃先去山中湖的忍野八海，之後在那午飯，下午再去大石公園及玩天鵝船，因爲起來晚了，就把忍野八海取消了，上午就去大石公園。

也是在河口湖車站出發，坐河口湖周遊巴士，約半小時到自然生活館下。其實從4月開始到秋天大石公園都是有特色的花的，唯有五月下旬兩週沒有，當然這個時候也有其他的花。

這裏看富士山真是很漂亮呢，好像一幅畫，自然生活館裏面好幾家賣冰淇淋的，在湖邊及室內都有地方坐，讓你欣賞美景。咖啡記得最低約400一杯，冰淇淋有300多的，也有500的。

看完富士山，我們就返程回到河口湖車站，去6號站臺坐車去山中湖了。坐上車發現中間的車站和網上查到的不太對，實際的站是多很多的。山中湖也有遊船和腳踏船，都是天鵝的。

我們去喫餺飥，在山中湖役場前下（選這裏喫飯也是因爲我要在這裏領半馬物品），下車後往回走兩三百米吧。門口是背向湖的。

我們到的時候一點多，不用等。裏面大部分是日式傳統那種跪着的座位，也有部分不是。老媽是不能跪着那種的，幸好給了我們坐着的位置。

下圖：門口

下圖：從餐廳窗口可以看到日式園林

下面是餐牌。我們點了4款不同的，後來看到天婦羅也想喫，點了一份蝦的（1500日圓）及魚的（1300日圓）。上來發現原來是定食，下面有米飯的。我們當然喫不下，但聰明的老媽竟然帶好了保鮮袋，我們就帶回去晚飯喫了。午飯4大1小喫了9600日圓，有外賣。
下圖：餐牌


這裏的甜品也很有特色呢，可惜當時喫不下了。

下圖：我們豐富的午餐


餐廳還有餺飥賣的呢，也有方便麪版，在河口湖超市賣380，但在餐廳外面的店賣480。
下兩圖：餐廳


山中湖的風格和河口湖不太一樣，一圈大概13公里，有點森林的感覺。

下圖：山中湖一景

喫完飯我去領物品，之後就坐車下去河口湖玩天鵝腳踏船了，實在不夠時間去忍野八海，不過他們也沒多大興趣。網上看這個月份如果是坐遊覽船末班是4:30的，天鵝腳踏船沒寫幾點完，不過5:30還可以玩。

從河口湖車站去遊覽船的地方，走路約一公里，腳踏船2000日圓半小時，可以坐三個人，如果是兩個大人兩個小朋友也可以的。在湖上走出來碼頭一點，就可以看到富士山了，還有那裏高級的酒店，很漂亮呢。


下圖：溫泉街


玩完約5:30，我們就步行回去了。由於午飯喫的很飽，還有外賣，我們就在路上再買了點黃瓜生魚片當晚飯喫了。

2017年5月28日週日
行程：山中湖半程馬拉松
有興趣看我的公衆號“PCL的馬拉松賽評”，這裏不多說。只想提醒一下，5月底週日想去山中湖，先看看是不是跑半程馬拉松，不然上午會封路，沒公交車的，即使12點後解封了，也會堵車堵很久的。我當時從山中湖役場前等12:19的公交車，最後13:30纔到......
下圖：我的賽評公衆號

​
我買了14:40的大巴從河口湖回去新宿，由於公交車晚點加上堵車，14:50纔到，還好大巴也晚點了，我趕得上，最後15:00開車，感謝主，不然回去挺麻煩的，要去富士市坐火車中間還要倒車。之後我們又開車瘋狂堵車之旅....最後6:30纔回到新宿。我當時以爲來不及趕上大巴，讓家人幫我去車站改票，說的全天的大巴都滿了，但我們那趟車最少有1/4空的座位.....這次也遇到不好的服務態度，家人說改票的職員態度很兇，又不會英語。

總結：這次行程順利結束，三天行程在天氣最好那天看到富士山美景
