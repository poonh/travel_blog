1.初春島根休閒遊 - 廣告

D1:
早餐後出發，到奧出雲最出名的景點 鬼之舌震 先喫本地正宗蕎麥麪。飯後進行徒步。該景點以四季不同景色，奇形怪狀石頭，甌穴，水流出名，是中國自然步道。

含艾草的蕎麥麪
徒步完到木次泡當地人氣溫泉おろち湯ったり館，隨後喝珍珠奶茶及買菜，入住健康之森準備晚飯。

健康之森住的小木屋陽臺可以賞櫻

D2:
方案1(這是我們原計劃):
早餐後到奧出雲町サイクリングターミナル(cycling terminal)租自行車騎行尾原大壩。尾原大壩環境優美，騎行好地方。騎行線路全程約20公里，繞大壩一週，午飯前結束。
方案2(這是實際計劃):
早餐後到奧出雲葡萄園。這裏餐廳俯視葡萄園，是情侶約會的好地方。這裏最受歡迎的酒用日本不同野生葡萄品種交配而成，酸味突出。

遊覽葡萄園後，到尾原大壩觀景臺欣賞大壩風景。

無論方案1或方案2，我們隨後都去棉打公園。這一帶古代有鍊鐵歷史(たたら製鉄，1400年前日本鍊鐵方法，奧出雲曾經出產全國8成鐵)。當時鍊鐵需要大量原料 - 砂鉄，奧出雲大規模把山體砍伐開採原料 。開採過後把遺址改成梯田，還衍生一系列生態包括香菇，蕎麥和牛等，成爲循環型農業。我們在棉打公園欣賞梯田及於以前練鐵的古民家鉄師卜藏家 鈩の舎午飯。那裏用的是梯田特產仁多米，咖啡料理用水都是500年前鍊鐵用的泉水。店內充滿歷史痕跡，店外還保留着原來練鐵的儲水池。
以前鍊鐵現在成咖啡店的古民家
春天時的梯田
返程時到みなり遊園地玩仿鬼之舌震的超巨星滑梯。
