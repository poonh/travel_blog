1.疫情下爽極免費滑雪 - 相關行程信息

(所有價格2024年底更新)

我們的行程地圖(線路取自navitime，全程高速)：
這個地圖只有起點及終點，沒有mapsme版本，需要離線自行谷歌下載



邑南旅遊官網(滑雪場所在地,有一切你需要喫住玩信息)。
政府旅遊報告：
2020年邑南町観光戦略及意見募集。
2024年協力隊還招人搞旅遊(地域おこし協力隊観光支援マネージャー）呢。

滑雪地點：瑞穂ハイランド
這是疫情剛開始時，滑雪場爲了宣傳專門免費招待外國人。除了免門票還免費租用所有滑雪用具。不過我們去完兩個月滑雪場破產倒閉，但2021年底又重開了,由廣島縣的土木工事業アオイテクノサービス繼承。

好像廣島的恐羅漢一樣，冬夏都有事業。冬天滑雪，夏天觀星露營。五一黃金週還有特別活動。滑雪價格也便宜，平時1日券5700（含1000午飯券），但索道及租用滑雪用具沒信息，他們的網站我覺得設計的不太好。不過附近有店出租滑雪工具，可能比滑雪場便宜，參看K2 rental。

滑雪道方面，海拔從1000-1200米。我們幾個都是第一次，高級的玩不了，只能滑下去一段又自己走上去，完全不需要用索道，所以我說不上這裏好不好玩，不過我們玩的很開心呢。

這裏開車大概1.5小時，全程高速，高速費2000多。我們當時往返一輛車總費用約8000。
