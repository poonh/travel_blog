2.山口北九州喀斯特傻瓜钓鱼海景之旅 - 相关行程信息

(所有价格2024年底更新)

我先介绍一下这条线路我研究过的地方，包括我们有去及没有去的。

1.秋吉台国定公园
全国三大喀斯特当中最大的（另外的是我们7月份去的四国喀斯特及这次行程最后一个景点平尾台），开车也最方便（海拔最低，不用翻山）。里面有些溶洞（收费）可以参观，也有徒步线路（参见yamap)。有吃饭的地方。

开车的路叫カルストロード(Karst Road)，两边就是石头。

 2.青海岛 
有“海上阿尔卑斯山”之称。游步道(青海岛自然研究路)分开A及B路线，A是看海，B是看山。我们只走了A,大概两公里，很好走，因为拍照走了大概1小时。这里风景很美，主要看岩石。另外有观光汽船，看的地方是走路看不到的。这里停车场收费。

这个地方虽然不太出名，人很少，我们当时去整条路都只有我们。

2A.元乃隅神社
差不多是这带最出名的景点，面朝日本海的神社。走得快的话大概半小时甚至更短可以看完。停车场收费。 

这里附近有梯田(東後畑棚田)。

3.角岛 
自从20年前（2000）通车后角岛大桥就成为著名的景点了。岛上本身很小，主干道只有4公里，有些零散景点，所以我们选择骑自行车，这样可以更好看海景。骑车线路网站有。电动自行车，2小时410元。由于坡多，最好骑电动。纯骑行两条线路加起来两小时足够的。我们因为去了博物馆，节奏很慢，两小时只骑了一条线路。租车直接去店里就行，不用预约。我们在路上还买了100元1公斤的红薯。

想跑角度大桥，每年10月份有10公里(つのしま夕やけマラソン)比赛哦。

4. Mermaid Lodge
靠近角度的海景小酒店，我们当时住的地方。现在已经倒闭了。

5. 土井ヶ浜遺跡・人類学ミュージアム(考古博物馆) 
这是绳文时代（日本远古时代）的人类学遗迹。要门票。看网站挺有趣，要是对考古有兴趣可以一去。

从这里南下到下关开车路上好些考古遗迹，在google地图顺着车路看就能找到。

6. 一の俣桜公園

 https://shimonoseki.travel/spot/detail.php?uid=367&c=80&sc=13&area=

是Mermaid Lodge的老板娘 推荐的地方，是个秘境（就在路的一旁，附近什么都没有的），水非常清澈， 有很多大锦鲤，春天可以看樱花。公园本身很小，很快看完。

7. 川棚温泉 

 官网：https://www.kawatana.com/

川棚グランドホテルお多福（我们泡温泉及吃饭的地方）

元祖　瓦そば　

川棚是个温泉小镇， 里面有很多个汤，都是老式的。我之所以选择多福是因为他家提供午饭温泉套餐，温泉入场原价700，在他家吃午饭多给360就可以泡温泉，而且这个温泉是这边最高级的，老式温泉不带露天风吕的。除了一个室内一个室外，温泉还有桑拿。

小镇本身也可逛，有博物馆。

另外，下关名物 瓦そば发源地就是这地，始创人是元祖，不过川棚其他地方也有的吃。

8.本州最西端

 https://kyushu.tv/honshu_westernmost/

这里也是可以看日落的地方，要走段路上去，我们时间不够，没去这个地方。

9. 脇田海釣り桟橋及 響灘緑地（グリーンパーク）

i. 脇田海釣り桟橋

其实原来我们没打算去钓鱼的，打算去英彦山看红叶，但是为了方便raku老师网上开会就来这里了。其实登过那么多次山，玩玩钓鱼也不错。这里入场费1000，租用鱼竿及鱼饵800元，只要你选对地方， 鱼饵够你钓几十条（都是竹荚鱼）。租用鱼竿是sabiki(内文再科普)，如果你想钓大点的，要自己带鱼竿。

我们当时是自己带了泡沫箱去装鱼的，我不知道那里有没有得买。我们去的时候是假期，有热食买（咖喱，おでん等）。平时可能没有，但有碗面。入场之后就不能出来的，所以你不想在那里吃饭的话就要自己带好食物。 另外，这里有厕所及足够的休息地方，配套做的很好，风景也很漂亮。

另外，这里停车场收费一次性300元。

ii. 響灘緑地

位于 脇田海釣り桟橋南部两三公里的地方，入场费150，里面另外有再收费地方。这是一个很大的公园，也是学习基地，停车场要收费。

如果你想徒步，绿地南下的 頓田貯水池是个好地方。一圈大概5公里，看的是生态环境。 貯水池边上的停车场是免费的。

9A.旦过市场

https://tangaichiba.jp/

这是一个老式市场，里面熟食，肉类，海鲜，蔬菜，酒类什么都有。蔬菜是在日本没见过那么便宜。地点位于地铁旦过站，小仓南下走十几分钟就到。

10.平尾台

http://www.hiraodai.jp/

这是全国三大喀斯特里面最小的，但我们都很喜欢。和四国的及秋吉台不一样，这两个都有看喀斯特的车路，你开车走就可以，但这个你要看石头就要徒步。这里你可以亲近石头，尝尝在石头缝中间穿插的乐趣。这里也有很多芦苇，每年春天进行野烧，目的为‘景観維持や害虫駆除、林野火災の防止など’。徒步线路具体看yamap。

如果你不想徒步的话，这里有平尾台有自然観察センター可以学习，还有营地可以住，还有些收费户外体验活动可以参加。

停车场方面，这里有好几个，有免费也有收费。

11. 英彦山

http://www.soeda-kankou.com/area01/

这是个佛教的山，有寺庙等可以看，也是北九州看红叶出名的地方，海拔1200米。
