D2:小琉球 - 東港華僑市場

今天行程：
早上吃完早餐到左營站坐8:30的屏東客運9127D(票價61元，klook預訂)到東琉線碼頭，計劃坐10:10藍白航運到小琉球，船程約20分鐘（最後坐了9:45的，往返船票450元，klook預訂）。11:00-12:30玩水行俠的水上自行車(1200元，klook預訂)，隨後午餐吃簡餐。14:30-15:10坐探索拉美半潛艇看海龜(380元，klook預訂 )。最後坐16:00的船返程（實際坐了15:30）。回到東港(即東琉線碼頭)於碼頭隔壁的華僑市場（當地海鮮市場）吃當造肥美的黑鮪魚（藍瓊吞拿，4-6月是他們洄游產卵季）及買手信。最後包車回酒店(1000元)。詳細信息看行程相關資料。

今天出海是行程重點～

行程開始:
 酒店包早餐，不是自助的。有6款選擇，四款是漢堡，另外兩款分別是粥及肉臊飯。每款漢堡用的麵包都不一樣，例如竹炭，巧克力，蕎麥等。每款早餐賣相都很漂亮，而且很豐富，根本吃不完。漢堡餐有些炸物，不是新鮮炸的，不過除此之外都很好，沙拉醬汁尤其好吃。有一點特別不好是由於分量太多，我要求職員減量，她以怕亂為理由拒絕。這就逼我浪費食物（實際上我看其他客人也吃不完）。

這裡是文創酒店，很多漂亮的餐具

餐廳一角

用餐環境

早餐 - 白粥配鯖魚

早餐 - 煎蛋肉臊飯，雞蛋很脆，魚丸也筋道

早餐 - 漢堡餐，還有馬卡龍被擋住。忌廉湯很濃稠。還可以配咖啡/紅茶/綠茶。

漢堡有四層，現在解構一下，以後自己做漢堡可以參考。
第四層的是土豆碎蛋+美乃滋，第三層是番茄生洋蔥生菜

第一層是主菜（牛肉或豬肉或鮮蝦等），第二層是奶酪


吃完早飯，由於我早一天探好路，算好我們7:50分出發（約8:10到左營大巴站等8:30的大巴）。最後我們7:47離開酒店，8:10到達目的地。在左營捷運先上廁所，然後在捷運1出口（高鐵方向）坐扶手電梯上去，上去後是捷運和高鐵的半露天連接處，在連接處往左方看就看到大巴站，到最盡頭的1號站台就是我們上車處。從下捷運到大巴站台大概2分鐘。

話說，在我們坐扶梯上去大巴站台時，"少"突然說肚子痛，要上廁所。最後我們時間剛剛好，8:26上車。出遠門真的要提早點。如果你萬一錯過了車，這裡也有拼車兜客的，參考價200/人。

我們第一站東琉線碼頭到站，路上沒怎麼堵車。我在ibus app觀察了幾天(網頁版在此,我主要看首班車8:00的），平日早上這段時間基本55分鐘到。
左營捷運1號門出來

出來是通往高鐵半露天連接處。左面扶梯上高鐵，右面是捷運，往紅色箭頭方向看就是屏東客運站，去1號站台。

這裡是大巴乘車出處，最前面1號去東港（碼頭叫東琉線碼頭，港口的名字是東港），2號去墾丁。

去墾丁的車。

大巴候車處能看到新光三越的彩虹市集

1號站台，我們的大巴9127D。有電子顯示屏顯示下一趟車時刻，如果有整天時刻更好。

我們的大巴 - 9127D

車上有wifi，有充電

上圖說的是這條橋 - 進德大橋，橫跨東港溪

下車後(9:25)走幾十米到藍白售票處。當天人不多，我們趕得上9:45的船。換票除了二維碼還要護照。klook那裡沒說要護照不過還好我們有帶。

下車處對面是特產店，有雙潤膏，櫻花蝦等。不知道價格比起華僑市場如何。

除了特產店還有便利店。

這裡有顯示船上了多少人。如果人多會臨時加開。

售票窗口不用排隊。還有無障礙專用窗口。

碼頭上船處，藍白和泰富靠在一起

另一角度看碼頭，左側中間有蓋的地方是華僑市場，我們下午吃海鮮的地方

排隊上船，我們比較早，上了2層。2層大概二三十個座位。選擇2層是因為一般2層不那麼晃。

我們要登船了！

小琉球碼頭一景

碼頭的免稅店，也是碼頭一帶少數有冷氣的地方

一到步有個寺廟，上去有很多賣帽子墨鏡的地方。碼頭附近也有店鋪賣不過要貴。

我們在這位伯伯那裡買墨鏡。他很會做生意，我們買墨鏡，他就拿出一張只有用偏光鏡才能看得清的圖出來，證明他的墨鏡有效。這是我學習的一點。走的時候伯伯還提醒我們那裡有環島公交車呢。

狗狗坐機車。它應該習慣了，坐的很穩。話說這次行程經常看到狗狗，還可以帶進便利店。



我們買完東西，就慢慢走去水行俠那邊玩水上自行車了。這邊的街道明顯和碼頭那邊不一樣。
水行俠s那邊的街道


到了職員說海況不太好，問我們能不能下午1點玩。但我們14:30要玩半潛艇，時間無法改。後來我發現他們10點半左右發過郵件問我能不能改時間，不過我沒看到。其實我之前Line聯繫過他們的，不過他們不知道那個line是我。臨時聯繫用郵件很不方便，他們大可以預約時讓客人留下line。雖然我看當時風平浪靜但也可以海況不好臨時改時間，預約的時候最好就說明。溝通上我覺得他們有改善空間。當時我們準時到，除了海況原因教練也沒回來，等到11:20左右我們出發。期間「少」看到店裡的狗又發狂，今次闖禍了，被狗咬。她摸狗的臉，人家不喜歡。我突然聽到狗叫了一聲，人家大概是反抗。狗狗好像叫阿東，其實很乖的，我剛進門時不小心踩到它也沒事，職員還說它不咬人，但咬了少後，之後有小朋友想摸它，職員就說它咬人了...

為此，後來搞了很多事，主要是老特別擔心。老說"少"出海前還很開心，回來不一樣。我沒有她敏感沒看得出。我自己本身對狂犬病沒瞭解過，現在也瞭解到一點。被狗咬一定要馬上用水及洗手液沖洗10-15分鐘，然後消毒。萬一中了病毒，死亡率百分百。不過越落後的地方狂犬病越多，台灣2013年從菲律賓引入病毒，在此之前已經50年沒有人染病。台灣的傳染病統計資料查詢系統有台灣狂犬病數據，參考此。基本上，在台灣感染到狂犬病的概率很低。

回到我們的水上自行車，雖然今天很熱，但玩的時候有海風，一點都不覺得熱。穿的救生衣有兜，可以帶一小瓶水（200ml的能放下，500ml不可以）。不過當時情況全程不喝也行，我喝了幾口。

這時大概一點。我們準備午飯。水行俠位於碼頭東側，吃的碼頭西側多。碼頭附近很多是租機車或賣商品，沒見到有啥吃。這點實在不太方便。之前看地圖知道小琉球魅力商圈那帶有吃（靠近花瓶岩，從碼頭走約十分鐘，到達前有幾十米斜坡），我們往那邊走。這一帶看上去好點的有海鮮餐廳，也有一些小店，好些沒冷氣的...

我們在小琉球魅力商圈的7-11歲便買了點吃的，還有麻花（兩袋120元，麻花店內也賣魷魚絲，麻花是烘焙的，非油炸，比起油炸的硬點），然後去碼頭免稅店那邊坐。當日早餐吃的很飽午餐不覺得餓。

碼頭附近很多租機車的地方，我見過有家說沒機車牌照也能租。沒有預定水上活動的話也可以現場預定。

小琉球魅力商圈。

下船後我們就直奔華僑市場吃黑鮪魚了。由於怕熱，我們不考慮熱炒。這裡生魚片店很多，由於貨源都一樣，我相信每家都差不多。我們去的守席。吃了1）5片金三角（黑鮪魚最貴的部位）400元，2）兩碗海鮮當（400一碗），3）星鰻當，忘記多少錢，好像400左右，4）三個不同部位黑鮪魚生魚片，好像500， 5）熟蝦壽司及 6）生墨魚片。4個人吃，吃的很飽，總共2400多。另外，吃當飯送海鮮味增湯（非常鮮美），可以續。這裡用的米很好吃，環境主要問題是蒼蠅太多。溫度倒是沒有想象中熱，能接受。比起昨天的龍華市場好。

另外，這邊早上五六點就是拍賣魚的時候，如果你想看可以這個時候來看。我在youtube看過視頻，還可以直擊宰魚過程，但畢竟是生魚片，切了就吃，普通人能近距離看覺得有點不衛生...

各種魚的魚松，包括三文魚，黑鮪魚,旗魚，大概100一代。


我們還買了櫻花蝦及本地大蝦乾，忘了拍照。散裝的，買多少稱多少。賣的女士同一隻手撿蝦乾同一隻手收錢，衛生需要改善一下...我前面提過的食物法規，不知道在此是否適用。

在市場期間，突然下了很大的雨，可能超過1小時。當時特別涼快，在市場內好像開冷氣一樣。等到雨差不多停了，我們也逛完，我們就去泰富等候室坐，等18:15的包車，當時大概17:45。

話說，我之前在youtube看到有人在華僑市場吃海膽，連殼一起吃的，不過沒見到有賣，不知道現在不是時候還是去晚了。

泰富門口就是停車的地方，約包車可以在這裡等。打車也在這。

泰富的等候室。

當時雨太大還緊急停航，今天我們行程沒半點耽誤，真走運。

看來台灣發票有一定價值

等包車的時候，看到一些小孩下船，一開始還以為是學校旅行。後來不知道是不是泰富的老闆出來介紹他們，是小琉球溜冰隊，拿過三面金牌，泰富贊助了他們5萬元的裝備。小琉球雖然是個人口只有1萬多的鄉下地方，聽我們水上自行車教練說高中都要去本土，但竟然能有自己的溜冰隊。我感覺在台灣即使在鄉下也能有發揮自己的機會，但當然我相信小琉球在台灣是個比一般鄉下富裕的地方。

小琉球溜冰隊的孩子

泰富的facebook有介紹這次贊助。


大概18:20分我們上車了，劉師傅很謹慎，沒超速。這段時間下班，路上有點堵，最後我們19：30返回酒店。路上遇到雙彩虹及日落。

剛下了雨，坐車時看到雙彩虹，"少"第一次見真彩虹。

車上看日落


其實在華僑市場吃的很飽，不過在發育時期的"少"還要吃晚飯。我們返回酒店後就再出來。開車回來時經過一些酒店附近街道很多吃的，我們就回去找。酒店對面是刁民酸菜魚，外面排隊的人非常多。那個四人份量的盤超級大，國內的沒那麼大（我沒拍照）。這下引起了"少"的興趣，想第二天來吃。

我們沿著裕誠路往東走，首先看到一家炸雞店，但裡面沒人不敢去吃。但我後來查google評價，吃的人很多的。後來我們去了富民路那邊一家看上去挺上檔次的韓國料理店，裡面裝修也很好，不過低銷要每人350元，當時我們根本吃不了那麼多（台灣很多地方有低銷一說），然後就去別的地方了。最後去了一家主打雞湯的店 - 龍涎居雞膳食坊。當時其實也沒人吃，不過這店感覺不錯我們就去了。

店鋪介紹

餐牌。據店員介紹，山雞是雞胸肉，山雞腿q彈，土雞腿結實。一份湯挺多的，夠兩三個人喝。我們喝的人參及鳳梨苦瓜。鳳梨熬湯第一次喝，酸酸甜甜的味道很不錯。人參味道也很濃。

我們的晚餐，除了兩個湯還有蔥油拌面，雞腎及地瓜葉，總共650元。

我們吃完時有人進來了，當時都快九點了。台灣人難道喜歡晚吃飯嗎？還是大家都去了刁民酸菜魚呢？

吃完飯"少"去隔壁的天豆福有機豆花豆漿。台灣豆花店很多的。這家不是連鎖店。

這裡只有外賣

"少"點了南瓜豆花。店員說只有這家店才有。

我還買了一瓶(910ml)特濃無糖杏仁茶(150台幣)及極品白豆漿。杏仁茶果然很濃，白豆漿很香豆味

確實很濃的特濃無糖杏仁茶
