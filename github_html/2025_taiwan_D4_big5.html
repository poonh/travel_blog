D4:紅頂谷創(馬玉山工廠) - 華饌湯包麵食館午餐 - skm park

今天行程：
早10:00出發，打車到兩公裡外的紅頂谷創（打車120元，這是馬玉山工廠），參加10:30的鳳梨酥製作(klook預訂雙人門票+鳳梨酥製作套餐,也可以網站預約鳳梨酥製作,400台幣，現場報名不一定有名額。總共13組，每組2人，但我看有人一家三口參加也沒問題），然後參觀紅頂谷創。12點半左右返程到酒店附近的華饌湯包麵食館 吃台灣麵食。隨後到機場附近的SKM Park購物，5點左右去機場。

今天早上的行程其實考慮過不同地方，原來想過去看地質公園 - 漯底山自然公園或田寮月世界，只是這兩個地方坐車時間很長。我也考慮過其他博物館例如國立科學工藝博物館。這是很受歡迎的博物館。在高雄車站要打車有點小麻煩，最後發現酒店附近的紅頂谷創，而且DIY鳳梨酥也適合「少「，就決定去這了。

行程開始：
由於「少」難起床，我們又比原定計劃晚了出發，到紅頂谷創大概10:15。下車時有位老人問我們：」用餐嗎？「我後面再說他是誰。
下車看到觀光工廠


這裡是國際級觀光工廠

前台職員給了我們每人一張遊戲卡，還有手帶（購物減20元用），之後我們坐電梯上5樓。他們預先通知了5樓職員，知道我們要去2樓做鳳梨酥，到了就有人接待我們。這裡的職員每個人的態度都非常好。有次等電梯時職員還向我們介紹工廠。

10:25我們就進去做鳳梨酥了～

遊戲卡，集齊蓋印就有禮物哦

可愛的電梯

鳳梨酥製作開始了！首先在盒子上寫上自己的名字

我們的原料，鳳梨酥的面皮及鳳梨餡都是準備好的，在冰箱里拿出來。我刻意問過鳳梨餡是不是百分百的，職員小姐姐說是，用的2號土鳳梨，酸甜味的，因為老闆不喜歡甜。

我們做6塊鳳梨酥，所以把材料分成6份。

首先是搓面皮，好像餃子那樣，然後鳳梨餡要折幾折才放進去。

最後放在模具，不要用手指去壓，會不平滑，用手掌最厚的部位。

整個過程約半小時，11點完成，然後12點多去前台取。空檔的1個多小時就逛工廠。這個工廠有三部分，2樓主要介紹穀物，由不同藝術家設計。如果懂得藝術，有時間靜心欣賞，我相信可以很好感受，不過當時有點趕時間，只能匆匆一看。3樓是馬玉山製作工廠，週末不開，但鳳梨酥DIY又只有週末才有，意思是參觀工廠和鳳梨酥不可兼得。5樓是品牌故事。

若能再來，我選個平時，安安靜靜地感受藝術，看看生產製作過程。話說我小時候暑假很多工廠參觀活動，我記得去過可樂廠，不過現在香港已經沒工廠了。

現在我們看看2樓展廳，都是藝術創作。整個展館有兩個遊戲能玩，因為有禮物就有動力。

我們最後買了一盒鳳梨酥（360元），一些散裝的酥（80元一塊）（我在文末再點評我們吃過的鳳梨酥）及一些穀物飲。由於我們買了不同口味的酥而它們包裝都差不多的，結賬時店員確認了我們買了什麼酥，確保我們沒買錯，這點很貼心。

12點多的時候，我們去前台換禮品及取鳳梨酥。玩遊戲完成任務一份禮品，蓋章一份禮品，買東西時老闆發了兩份禮品，一個人總共拿到四份。

我們沒吃蒸餃，不過看別人的蒸餃的餡很多，下次可以試試。吃完我們回去酒店取行李，「少」去了隔壁的mini咖啡店喝咖啡。店裡人不多，不過要等15分鐘。他家的凍飲用的錫箔紙封口，更好保溫，我又學習了一點。我們最後兩點上地鐵，比預計晚了約半小時。

mini咖啡店

坐捷運到草衙（4A）下，2號門出去走一兩百米到SKM Park。到了第一件事 - 存包，網站有介紹E7門及E8門門有置物櫃，分開大中小，但沒寫多大，可以現金，悠游卡及line pay等。我不肯定能否放我們的大箱子，實際觀察是不可以的。如果要存放大行李箱，到捷運站1號或3號門（里我們出口2號門二十米左右），具體看捷運官網的高雄捷運車站置物櫃設置一覽表。
從車站2號門出來，走過這條路就到skm park

這個skm park的置物櫃，只能放小行李箱

置物櫃在E7及E8門，這裡有休息的地方。整個Skm Park都很多休息的空間。

存包後，正好看到街頭藝人兔兔的表演（他的youtube）。他表演魔術及擰氣球，現場送給小朋友。表演完他先說自己的故事，第一場表演沒有人鼓掌，之後他努力不斷改良。最後他問打賞時，不直接說，說自己丟了東西，希望大家幫忙找回來，之後拿了一張大的假鈔票出來，這時候大家都知道什麼意思了。沒有現金也可以line pay。他這個方式也是讓我學習了。和之前一天在巨蛋捷運看到嘿嘿視障陶笛樂團的打賞箱一樣，兔兔也是用差不多的箱子。

查他的資料時，我查到原來台灣街頭藝人是要拿牌照的。參看台中市街頭藝人咨詢平台，可以看到不同藝人介紹，有「藝人證號「，有個人及團體，有表演藝術，視覺藝術及創意工藝。他們還可以在網上申請表演場所。台灣的文化部官網，有專門一頁關於街頭藝人介紹，裡面接近兩萬名，有些還附上社交媒體賬號及電子信箱。我也查到裡面好些視障樂團。

這種制度下，在台灣，街頭藝人就變了一門正當職業，你可以合法找到自己的演出空間，可以網絡上宣傳自己，有自己的粉絲。看到台灣的街頭藝人我想到什麼？馬宏傑的《最後的耍猴人》。馬宏傑是國內記者，跟拍了10多年河南某地的耍猴人。他們也是街頭演出，也可以算是「街頭藝人「。但他們沒有合法場所，經常擔心被人趕。他們也沒有台灣那種「打賞箱」，演出後就問人要錢，有些人看他們就是「討飯」。「街頭藝人」在國內並不是正當職業，沒有生存空間，所以看書很多人對耍猴人有誤解。其實這些表演很多時候帶有地方色彩，好像耍猴人，就是因為他們的地方特別窮困，不能靠務農為生才衍生這種技藝。街頭藝人也是藝人，和那些明星沒兩樣。他們只是不需要你付出高價去欣賞。當你無聊的時候，想看看他們表演就可以在網上查到他們在什麼地方演出。如果耍猴人的生存環境像台灣那樣，不知道耍猴這門技藝會怎樣走下去？

很多小朋友排隊要兔兔的氣球，不是那麼多人我也想要。演出完後兩小時，我依然看到他在送氣球。

我也會擰氣球，所以刻意觀察他擰，我沒他那麼熟練的。

行程總結：
總結今次台灣行程，一老感受最深是吃的東西不像深圳那麼多味精。深圳很多都是連鎖店，味道是好，但吃完太渴。深圳的連鎖店服務裝修味道樣樣都好，但味精對健康並不理想，感覺就是華而不實。台灣的東西實而不華。我們在台灣吃過的都是非連鎖，即使是簡單食品也有自己的特色，好像豆花店。在香港，現在也是商場多，商場內也是連鎖，個人開的小店很難生存。感覺飲食業台灣更多生存空間。街頭藝人也是讓我見識到，社會環境開放更多生存空間。好些地方看到對殘疾人士方便，這點我也很欣賞。總的來說，今次台灣之旅看到對低下層的照顧，這都是靠政策上的。至於政策的由來，有機會我再學習。

後記：
記得我們買了好幾種鳳梨酥嗎？有紅頂谷創自己做的，那裡買的，董師傅也買了貴的及便宜的，總共4種。回家我們試嘗了。
左起：紅頂谷創DIY（100%鳳梨），紅頂谷創賣的（100%鳳梨），董師傅（20%鳳梨餡）。下面的是董師傅（100%鳳梨）


幾個鳳梨酥切開這樣，左起：董師傅（100%鳳梨，50元一個），紅頂谷創DIY（100%鳳梨），紅頂谷創賣的（100%鳳梨，45元一個），董師傅（20%鳳梨餡，150一盒8個）


從下面的圖看到董師傅（100%鳳梨）的很結實，切開沒啥碎，紅頂谷創兩款都很酥。實際上兩款口味很接近，不過成分有點不一樣。味道方面，紅頂谷創的酸甜，董師傅的甜味多。董師傅廉價的也不錯，有鳳梨味。幾款都好吃，喜歡結實點，甜的選董師傅（100%鳳梨），喜歡酥的酸甜的紅頂谷創。

當然，買食品不能只看味道，還得看成分，現在看一下每種的成分：

紅頂谷創賣的：土鳳梨餡（土鳳梨、砂糖、麥芽糖、椰子油、蘋果酸），麵粉，奶油，煉乳（蔗糖、奶粉、水、沙拉油、無水奶油、脂肪酸甘油酯（脂肪酸甘油酯、檸檬酸）、乳糖），糖，蛋，奶粉，芝士粉（乾酪、乳化鹽（磷酸氫二鈉）），鹽。
