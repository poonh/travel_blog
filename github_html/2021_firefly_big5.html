3.比看電影更幸福的螢火蟲之旅 - 遊記

日期：2021年6月13日（週日）

地點：竹原仁賀ダーム附近

我是在Press Net6月初那期看到竹原仁賀ダーム的螢火蟲介紹。 我原來知道志和有的看，但開車這裏近些，所以就去這。

昨天raku老師找我看螢火蟲，也是臨時決定（其實去年就說要看的，不過當時已經過了時間）。我昨天精神不好，晚上也做不了什麼，不如去看吧。五點半的時候接raku老師過來喫了我的新菜（印尼+中東做法的印度香米），之後就出發去仁賀ダーム。從大西條出發，上了2號線走10分鐘左右，看到330往右拐，看到大壩就停。路線簡單得不用導航。我們喫完飯出門的時候還下不小的雨，不過看雨雲圖竹原不下雨，我們就決定繼續上路了。

我的新菜要跑題出個鏡~
由於Press Net是在大壩拍的照片，我們就去大壩了。其實可以走到大壩很近的位置看，不過從停車場走過去有點遠，而且入黑了我們又不懂路不敢走。一開始就在大壩上面看，遠距離看到一閃一閃。他們其實是在水上飛，但我看着就好像在游泳那樣。 我們看了一會下雨了，就回到車上準備返程。
 
順便科普一下，這裏看的螢火蟲品種是ゲンジボタル，是比較亮的。螢火蟲在日本還有傳說...

開了幾分鐘後，還在330路上,看到很多路邊停車（具體位置已標在地圖），估計是有螢火蟲看。我們也在路邊停車了，一下車就看到螢火蟲，走進去發現滿山螢火蟲。有個女孩抓了個螢火蟲給我，我拿在手上覺得很神奇。 
 這是螢火蟲的真身，尾部就是發光的部分

這條河流靠山，非常非常多螢火蟲。走了一小段有條短橋。我們在橋上看，螢火蟲在我們頭上飛過，橋一面是山溝，另一面是往馬路，往山溝那邊看真是很夢幻，好像電影那樣，但比電影漂亮多了。照片拍不出那個美，只有身在其中，才能感受那種心情和奇景。這就好像去年秋天我們去龍王島，躺在草地看星星那樣，不像真實的，像做夢。
我們懷着美夢般的心情回去了。實在想再看一次呢。感覺實在是太美好太幸福了。
最後補充點資料，我們大概不到9點到後面那個地方。當時路邊停車大概不到10輛。我們看完的時候車也走了一半了。 整體看的人不算多的。
