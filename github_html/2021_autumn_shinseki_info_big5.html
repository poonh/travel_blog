2.神石高原愛護動物紅葉之旅- 相關行程資料

(所有價格2024年底更新)

我們的行程地圖(線路取自navitime)：
(紅:住宿   藍:景點    綠:喫飯    橙:超市  啡:加油站    黃:廁所  紫:停車場)



神石高原很多玩的，雖然不大。不過喫的有特色挺多。這裏最出名就是和牛，還有羊肉(根據2022年《第12回全國和牛能力共進會》神石高原和牛拿過優等賞，參考2022年11月広島和牛News)。看政府旅遊官網也好些特色咖啡店，我們去小學改裝成餐廳的"學校食堂"喫飯也很好。我們去神石高原的時候有現金券活動（好像go to eat)，付5000現金給6000現金券。

神石高原旅遊官網
裏面有一切你需要的喫喝玩樂等信息。根據廣島縣政府2022年《広島県観光客數の動向》(p28)，神石高原乃全縣最少觀光客地區之一。當地政府爲了吸引遊客也制定了一些政策，具體參考2022年《神石高原町観光振興企畫》，裏面有調查數據，具體政策，增長目標等。

神石高原ティアガルテン(Tiergarten)
我們住的地方，也是一個公園，樂天也能預約。住宿和公園門票分開（只住不用買公園門票的,門票500）。我們住的5人棟一棟使用料10890，另每人住宿費1100。這裏可以露營。可能這裏不是政府設施，小木屋沒有我們之前住政府的大，但也可以。

這個地方由愛護狗只NPO Peace Wanko(ピースワンコ)營運。他們全國都有領養中心，基地在神石高原。 Peace Wanko是日本救援名犬夢之承的老家。當年它快要被人道毀滅，但及時被養父大西健承收養了。後來來了神石高原受訓，成爲救護犬。它參與過很多國內外搜救活動的。Peace Wanko是一個很有意思的地方，他們愛護動物的系統是德國引入。 ティアガルテン原文就是德語Tiergarten(動物園)。這裏不會對狗進行人道毀滅的( 殺処分ゼロ，廣島縣原來是全國殺害最多貓狗的縣)。

公園很適合小朋友，大人也能玩的開心。這裏也有紅葉看，有尋寶及抽獎，滑梯，吊牀等，也可以去Peace Wanko見學（在前臺填個表就行），當然也有專門的狗設施，具體參看遊記。

Peace Wanko youtube

PS.政府還在這裏辦過相親活動

戀人之聖地 （米見山）
正式名字 とよまつ紙ヒコーキ・タワー(紙飛機爲主題)。這個地方令我佩服地方政府發展地方旅遊的心思。這座山說不上什麼特別，路上沒驚豔風景，也不高。特點是比附近的山高，而且海拔也不特別高(600多米)，開車登頂不麻煩。山頂可以360度看紅葉及雲海，爲了更好的體驗就建造一個塔，以紙飛機做專題活動，還有爲戀人而設的東西,這就包裝得很吸引。政府網站還介紹了這個地方出去的紙飛機最多飛行1公里。

秋冬可以看雲海。前兩年沒有疫情時，11月初塔還早上6：00-8：00開放，方便人看雲海。我們去的時候雖然不開放，但不登塔也很好看的。

神石高原另一個可以看雲海的地方叫雲海の裏

如果你願意多花點錢，還可以住能看到雲海的古民家。

學校食堂
 神石高原和牛很出名，這裏就有的喫。日本羊肉不多，這裏也有羊肉。這個70年高齡的學校，因爲老化已於2023年3月停止營運，準備拆卸拆卸。我們早幾個月去趕上末班車了。

光信寺の湯ゆっくら
 神石高原的溫泉，有藥湯，費用850日元。

道後山高原莊
道後山附近的溫泉，620日元。附近有紅葉看。

冰淇淋グリーンフィールド
沒有網站，我在上面地圖(綠色)標了，在後山高原莊附近。冰淇淋自家制，很多新奇的口味，特別推薦柿子，超級好喫。日本地方牧場用自家牛奶產冰淇淋店很多，味道都是唯一的。這些農村地方不知道那裏學來的冰淇淋技術。我們在其他鄉村地方喫過的冰淇淋在以下游記有：《黃金週登嶽山》，《疫情下上山下海遊愛媛》，《夏日消暑勇闖中國第一峯》。
