2.安芸高田美容歷史文化之旅 - 相關行程信息

(所有價格2024年底更新)

(紅:住宿   藍:景點    綠:喫飯    橙:超市  啡:加油站    黃:廁所  紫:停車場)


安芸高田官方旅遊網站
有你一切所需的旅遊諮詢。

我簡介一下我在政府網站查到神樂門前湯治村的報告。

神樂門前湯治村是安芸高田一個大旅遊景點，政府公共設施，1998年開村，佔地5000多平米。疫情2020年獲得政府補助2000萬。2018年債務高達1400萬。2022年政府《観光関連施設の一體管理に係る官民連攜手法検討調査業務》的模擬(p20)顯示，假設宿泊飲食一定的增長率下，升級一定客房數及增加一定客房數需要成本。由於成本太高（每個客房最多接近7萬），難以吸引民間投資，所以不進行升級而進行更新，但報告內一些設施會考慮和民間營運。

神樂門前湯治村
神樂是日本地方傳統歌舞，造型以鬼怪見稱。位於一個山坡上的仿古村落，賣點是神樂，還有住宿，溫泉，飲食(由於在山披上，附近啥都沒的)。主要在週末，連續看兩場1200，一場800。座位分開日式榻榻米座位及椅子。除了神樂欣賞，還有小朋友帶神樂面具的體驗活動(收費)。

看網站安芸高田有20多個神樂團。日本這些傳統保留的很好，除了做旅遊宣傳，政府イベント（活動）邀請他們演出外，學校也可能有的學。這次疫情臨時取消，不過後來我在一活動看到高校生演出並親身體驗 。

喫飯推薦ふくすけ（旅籠千両萬両內) 。很多特色的菜其他地方沒有的，價格也沒一般旅遊區貴。當時還有一家福助トーフ豆腐店，裏面的豆腐和超市貨完全不一樣，現在已經倒閉，但你可以看看遊記。

這裏有兩家旅館，提供一泊兩食，當然還有溫泉，詳見官網。

酵素風呂
酵素風呂是日本獨有的，把人埋在發酵米糠裏面，有美容減肥治療病痛等功能。我們主要目的爲玩，的確很獨特的體驗。要專人服務，玩完之後全身是米糠打掃也不容易的（所以要穿他們提供的衣服）。注意，玩完千萬不要用沐浴露洗澡，會把身上的酵素洗掉的。

我是無意中在地圖看到，當時是東廣島福富的，一隻想玩。全國都有得玩，在hotpepper預約都比較貴，一般要三四千。森藏酵素風呂算便宜，2420含浴巾和衣服。而且這家店在森林之中，還可以玩燒烤機露營。這裏不大，我們去的假日早上包場。店家是木匠還有很多木工賣。靠南幾公里有咖啡店，如果你們有人想玩有人想不玩，不玩的人可以去咖啡店坐,具體位置我地圖標了的。

毛利元就歷史遺蹟
有博物館及郡山城古蹟。

毛利元就是三國時代名將。1552年他在西條第一名山-鏡山和本地的尼子氏和大內氏激戰，史稱 鏡山城の戦い，最後把本地打敗。郡山城是他的城，有500年曆史，是日本100大名城。官網有詳細歷史介紹，導遊服務及登山地圖。

我們沒有導遊，自己逛也從山下到最高點本丸，慢慢走大概1.5小時吧，全都走完可能2個多小時，我們是走的比較慢的，中間也玩了。更詳細介紹參考此。

縄文あいす(冰淇淋)
本地出名冰淇淋店，用的古代米。當時我們時間不夠就沒去。

土師ダム(大壩)
一個郊遊的好地方，有高爾夫(グラウンドゴルフ場，連租用球具幾百日元），自行車，營地，春看櫻花秋看紅葉。
