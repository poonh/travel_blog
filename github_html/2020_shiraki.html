3.广岛百名山白木山 - 游记

日期：2020年4月25日（周六）

行程开始：我们8点多出发，大概10点正式登山，停好车后，经过农村，跨国铁路及小河，我们到登山口了。
停车在这个超市门口

经过清澈的河流

宣传一下先

这个穿过农村的铁路很漂亮

 每一个半小时才有一次的火车，被我们遇到了，这是芸备线，去三次的
车站附近的农村
这个下水道很有特色，上面是鱼鱼

我们一开始走错了，走了地图上左边的线路，但没找到具体的路所以原路返回去正式登山口，但走错的路上遇到这个寺庙，很漂亮，这个照片没真实的漂亮。

去登山口路上的农村景
我们终于到登山口了
我首先要记录一下，这次及之后的岳山，洗碗王子全程背着垃圾袋方便我们扔垃圾。
路上一般是这样的，危险性不大（除了下山可能部分有点滑），但800米攀升还是需要些体能的
开始没多久，看到这个“入山禁止”的标志，不用管的，大家都进去
无惊无险，到山顶了！这里是360度全景的，很棒！顶上有小草坪，可以带个垫子来坐。这天风有点大，山顶有点冷呢
 我们就坐在草地上吃午饭，爬了两小时大家都很饿。
 这里还有免费望远镜
山顶景色很棒！
 拿旗拍照留念
下山时洗碗王子和公主又在后面聊.....
下山时看到原来这里明治时代就有的

下来去Shoji吃下午茶及买菜，今天行程完满结束！天气很好，拍照很美，运动量很大，腿很疼！特别鸣谢小z路上一直贴心照顾，一看到我有需要就来扶我。
