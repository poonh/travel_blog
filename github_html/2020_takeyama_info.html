2.广岛百名山岳山 - 相关行程信息

我们的行程地图(行车线路取自navitime)(无法显示点击此)
（蓝:景点    绿:吃饭   黄:厕所  紫:停车场)）
徒步线路走的是红色线路（东侧要放大看，线路取自yamap），去程1.4公里，1小时15分到顶。返程2.3公里，花了2个半小时（纯属技术问题）。



提示一下，这山没有正规停车场，具体地点看地图紫色，在登山口那条路再往前走几十米一块可以随便停车的空地，那地还是斜坡，当时难停死我了。那块地不大，我觉得应该停不了十辆车的。这里附近有公交车站的，下车后走一公里可以到斗升登山口，即我们的登山口。

政府旅游报告：
2023年《第2期世羅町観光振興基本計画》(商工観光課)

岳山是广岛百名山，很好听吧，日本有“全国百名山”(还有全国两百名山，三百名山，四百名山，花百名山）。全国百名山广岛县没有在内的（中国地区只有大山）。我们去过其他的七国见山，火山，白木山都是广岛百名山。我没写过游记位于东广岛的龙王山也是。这山比西条第一名山镜山还要简单，登顶可以俯视西条景色。

岳山简介：
这个山的卖点是岩石，的确挺好看，我们中间还在岩石缝中穿过。去之前我不知道这是半野山，完全没楼梯的，走这个山有登山杖会好很多，不然容易滑。路上很多树叶，刚下完雨不要去，因为没楼梯下完雨太滑了。上山有些地方比较陡，要拉绳子，下山那段路绳子更多，爬完这个山后大家有点怕，觉得危险（我还可以接受），就决定下次不去三仓岳，去恐罗汉了。
世罗高原农场就在附近，有时间可以去玩，冰淇淋店有冰淇淋，黄油制作，骑马体验。不过当时因为疫情没开，但还好冰淇淋店在等我们呢。

世羅高原ファームランド ジェラート工房（世罗冰淇淋店）
