1.県民の浜复兴之旅 - 广告

日期：2020年9月（2天1夜）

疫情期间，为了复兴小岛，位于吴市上蒲刈島的県民の浜提供特惠住宿，每人只要1000日元，就能入住海景小木屋+烧烤晚餐。

行程计划：

D1：午饭后出发，到県民の浜附近咖啡店享受海边下午，之后打网球。晚饭在海边小木屋阳台吃烧烤，欣赏晚霞。饭后观星及海边观看闪闪发光水母。
当地晚上星空
D2:早上阳台享受海景早餐（自己准备),然后海滩漫步，再到古镇御手洗游览。御手洗乃当地最著名景点，江户时代留下的街道，日本的重要伝統的建造物群保存地区。
古镇附近的海边，以前是港口
成本估计:
住宿：1000
车费：～1000
网球：～ 550/小时（5人平摊）
另一早一午及晚饭可能加菜，合共～3000多
