1.夏日消暑勇闖中國第一峯 - 廣告

日期：2019年8月底（2天1夜）

D1:早餐後西條出發，到全西條超市最貴的牛奶產地 - 蒜山高原牧場喫新鮮牛扒及蒜山高原牛奶冰淇淋。餐廳設計高級，落地玻璃，邊午餐邊看牧場大草原。隨後到蒜山酒廠。這裏有一種獲獎葡萄汁，用只有日本纔有的本地高原品種，前所未有喝過那麼香濃。 最後入住塩釜camping village，晚上睡帳篷，在營地喫火鍋。天氣好可以觀星。

注意，蒜山高原海拔500多米，野營不用擔心天氣熱。另外，記得穿長袖衣及長褲防蚊子。
蒜山高原牧場餐廳
蒜山高原牧場
我們的營地
營地太陽下早餐
D2:早餐戶外喫，享受野營樂趣。稍後出發到中國第一峯 - 大山。大山又成爲“伯耆富士”。海拔1700多米，乃中國地區最高峯。登山起點海拔700多米，攀升1000。全程鋪好樓梯，危險性不高但很費體能，最好帶上登山杖。往返約6小時，下山後冷泉泡腳，消除疲勞，然後返程。
成本估計:
車費：1500
住宿 ～ 4000
另一早二午一晚(含牛扒午餐)，合共～10000
