3.廣島百名山白木山 - 遊記

日期：2020年4月25日（週六）

行程開始：我們8點多出發，大概10點正式登山，停好車後，經過農村，跨國鐵路及小河，我們到登山口了。
停車在這個超市門口

經過清澈的河流

宣傳一下先

這個穿過農村的鐵路很漂亮

 每一個半小時纔有一次的火車，被我們遇到了，這是芸備線，去三次的
車站附近的農村
這個下水道很有特色，上面是魚魚

我們一開始走錯了，走了地圖上左邊的線路，但沒找到具體的路所以原路返回去正式登山口，但走錯的路上遇到這個寺廟，很漂亮，這個照片沒真實的漂亮。

去登山口路上的農村景
我們終於到登山口了
我首先要記錄一下，這次及之後的嶽山，洗碗王子全程揹着垃圾袋方便我們扔垃圾。
路上一般是這樣的，危險性不大（除了下山可能部分有點滑），但800米攀升還是需要些體能的
開始沒多久，看到這個“入山禁止”的標誌，不用管的，大家都進去
無驚無險，到山頂了！這裏是360度全景的，很棒！頂上有小草坪，可以帶個墊子來坐。這天風有點大，山頂有點冷呢
 我們就坐在草地上喫午飯，爬了兩小時大家都很餓。
 這裏還有免費望遠鏡
山頂景色很棒！
 拿旗拍照留念
下山時洗碗王子和公主又在後面聊.....
下山時看到原來這裏明治時代就有的

下來去Shoji喫下午茶及買菜，今天行程完滿結束！天氣很好，拍照很美，運動量很大，腿很疼！特別鳴謝小z路上一直貼心照顧，一看到我有需要就來扶我。
