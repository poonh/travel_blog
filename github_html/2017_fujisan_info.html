携老带嫩游富士三湖 - 西湖河口湖山中湖 - 相关行程信息

所有价格为当年价格

日记：2017年5月

交通：从东京过去，新宿有大巴直达，到河口湖单程票1750日圆，车上带厕所。准时约1小时40分到，但估计必定会晚点。去程是周五上午，我们晚点一小时吧，返程是周日下午，晚点差不多两小时。疯狂晚点的原因是因为没有高速（个人认为）。
买票网址： http://bus.fujikyu.co.jp/highway/fujisan
网上订了票然后最晚开车前10分钟到便利店付款就行。

公交票：
富士五湖公交：h ttp://bus.fujikyu.co.jp/rosen/shuyu
如果只去河口湖及西湖，就买1300的公交pass，也有贵一点包游船和索道的。
我们因为要去河口湖山中湖及西湖，买的2550日圆，用两天的passport: http://bus.fujikyu.co.jp/otoku/fuji-passport
​河口湖车站售票处出来就是河口湖周游和西湖周游的车，去山中湖的在6号站台。
富士五湖区域点对点公交查询：http://transfer.navitime.biz/fujikyu/pc/map/Top
河口湖-山中湖时刻表： http://bus.fujikyu.co.jp/rosen/detail/id/3/#d_h01
河口湖周游及西游周游巴士时刻表： http://bus.fujikyu.co.jp/pdf/rosen/fujigoko/20160502/omni_timetable.pdf
去山中湖的车半小时一趟，发车准时，但中间必定晚点，到山中湖役场前起码晚十分钟。这趟车可以用pasmo，河口湖周游及西湖周游不可以。

富士五湖简介：最大及交通枢纽是河口湖，住的也多，交通网络完善。第二大是山中湖，位于河口湖东南面，从河口湖过去要坐半小时的车，海拔1000米。西湖位于山中湖西面，也有开发。精进湖及本栖湖在西湖西面，两个都很小，玩的不多。

河口湖旅游官网（景点坐河口湖周游巴士）：
http://www.fujisan.ne.jp/
那里景点很多，很多博物馆，我就简介自己去过的。

大石公园（公交站：河口湖自然生活馆）：
位于湖的北侧，面对着富士山，不同季节开不同的花，拍照绝佳的地方，有小餐厅可以喝咖啡吃冰淇淋
自然生活馆（公交站：河口湖自然生活馆）（可以做果酱及摘野果）：http://www.seikatsukan.jp/
游览船（公交站：游览船）： http://www.fujigokokisen.jp/contents/ensoleille/
930日圆一位，航程半小时。
天鹅脚踏船：和游览船同一个地方，2000日圆半小时，坐三个成人，5月底时5点半还有得玩，最后不知道几点结束。
另外沿湖还有好些博物馆及神社，自行参考官网。我坐车经过浅见神社，外面看上去古墓苍天的，觉得比日本一般神社有特色。此神社位于富士吉田市，从河口湖坐车去山中湖的巴士会经过。
温泉（公交站：船津滨温泉街）：河口湖东北侧离车站大概1公里的位置有很多温泉酒店的。
运动：
皮艇，钓鱼及骑车：http://www.c-ls.jp/%E3%83%AC%E3%83%B3%E3%82%BF%E3%83%AB
划船（河口湖北侧）：http://www.fujisan.ne.jp/spot/info.php?no=163
另外，河口湖车站附近有自行车店铺的，百度一下该能查到。
其实河口湖西湖山中湖我觉得都不是特别适合骑车，因为没专门自行车道，机动车又多。
美食：馎饦 （当地特产面食）
有两家出名的连锁，下面这个在河口湖车站对面就有。
http://www.houtou-fudou.jp/
我们去的这家，山中湖店:http://www.kosaku.co.jp/
面条看上去像刀削面，汤是素菜汤，挺鲜味，面条分量正常，因为还有南瓜土豆，吃的很饱，可以三个人吃两份。而且价格也不低，1500左右一碗。

西湖（下面景点都是坐西湖周游巴士）：
手作坊：
西湖的里根场：http://www.fujisan.ne.jp/iyashi/dantaitaikenichiran.html
官网有优惠券，原价门票350元减50元，6岁以下免票
这是一条仿古的村子，里面有造纸，蚕丝，陶艺等手作工艺，都要电话预约。我试过打电话去造纸，人家不会英语，预约不成。
和服：
也是里根场的，火之见屋，1000日圆。里根场不大，虽然和服可以穿一天，但大概一两小时就会逛完。

山中湖（下面景点都在河口湖6号站台坐车，假日到站必定会晚点）：
红山之汤温泉（公交站：花之都公园入口）: http://www.benifuji.co.jp/
这个温泉卖点是可以边泡温泉边看富士山，门票800，网上打印优惠券便宜100。有1000日圆左右的简餐，也有需要预定的名贵套餐（3500-5000日圆，价格包入场）
花之都公园（公交站：花之都公园）： http://www.hananomiyakokouen.jp/
看花的，要门票。
忍野八海（公交站：忍野八海，1小时1趟）： http://www.vill.oshino.yamanashi.jp/8lake.html
忍野八海是八个泉眼，世界文化遗产。我原来打算去，后来起来晚了时间不够。
游船（公交站：观光船）：http://www.yamanakako-yuransen.jp/
有天鹅形游船，也有天鹅脚踏船。

富士芝樱：http://www.shibazakura.jp/
每年四五月份赏富士芝樱，位于富士五湖最西面的本栖湖附近，场地能看富士山，要门票，展览期间新宿有车直达，一天大概两三趟。
（我原来打算去看这个，但新宿的车是8:45，带着小朋友太早了，当然到了河口湖也能另外坐车）

住宿：这次住的airbnb，在河口湖附近
