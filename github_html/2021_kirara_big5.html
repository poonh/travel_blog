3.盛夏暢遊日本 - 盛夏暢遊日本遊記

日期：2021年8月26-27日

地點：島根多伎町

前言：

這次旅程碰巧是我兩年前剛考了駕照一個月，有車後首次旅程同時間，那次是《夏日消暑勇闖中國第一峯》，去的鳥取大山，睡蒜山帳篷。去年初夏《疫情下上山下海遊愛媛》，我們四天三夜旅程既上山也下海，今年夏天我們則去海灘游泳了。這是我第一次在日本海灘游泳，之前就一直很想去的。除了小時候都沒去過海灘了。

今年3月份，無意中在一個日本網頁看到一些キャンプ場(camp)介紹，看到キララコテーンジ(kirara cottage)，當時就特別感興趣，特別想去住，並且馬上發給raku老師看。當時就想夏天去海灘游泳，住日本海海景別墅，還玩私人桑拿。夏天到了，我知道我們現在沒有旅伴，也去不成，我也沒具體把旅程納入計劃。6月底的時候，一次無意中又和raku老師提到這事，她就說去拉人 ，沒想到竟然拉到兩個，馬上訂房敲定行程，定在兩個月後。我覺得我和她好像在合作搞旅行社，各自做自己喜歡的事。我是隻喜歡研究，不喜歡做生意，她就很喜歡做生意，是個商業奇才。我就負責研究行程，做好了就給她賣，只是缺少了掙錢的一環（我今次是專門編了廣告給她去賣的）。

話說我們去之前一週這裏因爲暴雨爆了水管，住的地方停業了三天，幸好我們去的時候沒事。

行程：

這不是原計劃，而是最終執行計劃。

D1:

西條 - 川西鄉Family Mart買菜及欣賞本地手工藝 - Kirara海灘海景酒店午餐 - 立久恵峽 - 超市買菜及無花果園買無花果 - 別墅欣賞海景日落及準備晚餐 - 燒烤晚餐 - 別墅二次會 - 超豪華泡泡浴及私人芬蘭桑拿

D2:

別墅陽臺喫海景早餐 - 海灘游泳 - 無花果冰淇淋及午飯 - 無花果園買無花果 - 返程

開支(全部4人平攤)：

從西條出發油費：3225（一車）
住宿+燒烤套餐：20900+1600
無花果：3870（總共約40粒）
燒烤晚餐及早餐：5749
雜費：~1600
另外兩頓午飯及無花果冰淇淋每人合共約2000。
全程總費用爲每人約11000。

成本分析：

交通：7.2% ，喫： 42.8%，住宿：46.5%  雜費：3.6%

打キララグループ的信封，裏面有確認資料及注意事項。

 

另外還有很多廣告資料

 

 

這個服務是很讚的，可以預先把啤酒放在冰箱裏，讓你去到就能喝涼的。啤酒可是都是高檔貨，第一個是Asahi Super Dry，很出名的。


 
還有不同套餐可以選擇


還有屋子介紹

等了差不多兩個月，日子終於到了！在行程之前幾天，HY聽島根朋友說原定計劃去的八重滝因爲7月受到豪雨破壞封了，之後我就改變行程去立久惠峽。立久惠峽我一開始考慮過，不過嫌它走路太少，遊步道只有兩公里多，步行時間40分鐘。八重滝畢竟有好幾個瀑布，走路時間長點，所以我就決定去八重滝了。不過後來因爲原定計劃第一天和第二天行程反過來進行，去立久惠峽是更適合的選擇，因爲車程較短。而且40分鐘遊步道我們最後玩了兩小時（拍照原因，下面請繼續欣賞美照）。

8月26日週四：

因爲要去海灘，要帶冰飲料，我提前幾天準備好大塊小塊的冰，感謝教會提供保溫箱，每次出門總是拿教會的東西。今次喫燒烤不是在屋子陽臺喫，要去燒烤場，我提前很多就蒐集好我們要用來放菜的盒子，還把喫完7-11帶蓋的盒子留着喫沙拉用。

早8點出發，在離開廣島縣最後一個便利店(family mart川西鄉店)上廁所順便買菜。我們春天去島根時就在這裏買過菜，便宜又好。最後發現這是很明智的選擇，因爲kirara那邊的超市ラピタ選擇不多而且更貴。

由於這裏很多手工藝品，我們逛了好一會，最後一個普通的便利店變了行程第一個景點...

 

就是這個地方，離開這裏走10公里就上高速去島根了。大家經過也來逛逛。

喫完飯，我們就去酒店的泳池見學了。


游泳池是有室內及室外的，用的是過濾過的海水。

喫飯完大概1點，我們就去游泳了。我們先去道の駅逛。這裏是北歐風格，我們來看看外形。

 

裏面有喫的，也是海景。

這裏無花果是特產，按照日本的風格，就是無花果大轟炸，巧克力，餅乾，蛋糕，冰淇淋什麼都有。

我還買了穴道湖產的しじみの素（一種貝殼類海產，味道很鮮的），540元，準備做清湯鍋底用。


 買完我們準備去游泳了，我們發現竟然有“禁止”紅旗。入口也封了。

一開始不知道是因爲疫情還是什麼，raku老師去問，是因爲今天浪太大了，明天有可能可以。我們當時都準備了明天再不能就去酒店的游泳池。之後我們就立刻改變行程，去明天計劃去的地方 - 立久惠峽。當時大概1：30，我們2：05到。

我加插一下，第二天有點浪但能遊的時候，就變了“注意”的黃色旗子。

 

 

到了立久惠峽，久車停在わかあゆの(waka-ayuno)裏，那是一個再山谷裏的營地，環境很好。到了就看到遊步道的起點，有廁所。

我們現在要回去多伎買菜了，現在時間四點過。我們玩了兩小時~很開心。我們去這個超市ラピタ(rapita)，是這裏唯一的超市。

超市隔壁就是無花果園，上面寫了“産地直売”，我們就進去看了。

果園遠景

 我們看看聖經傳說中的無花果說，今次終於看到真樹了。

 

 

 

老闆看上去像一對父女。年輕媽媽的兒子在門口看門，有客人來就馬上叫媽媽過來。當時五點多，當日無花果已經收成了。我們如果買完好的，是400元一盒。稍微熟點的，1000元一大盒，有14粒左右。我們買了1000元的。

 

 

這裏的無花果真是很大很甜，B老師還是第一次喫，就喫到最好的。我們看到都忍不住馬上喫一粒。

快六點了，我們要回去做晚餐了~

營地接待處有很多姆明（moomin,北歐動漫）的產品賣，碰巧我兩週前剛和raku老師玩過姆明拼圖，2小時完成300塊~

 

這拼圖是跑題的~

 這邊的北歐風格和溫泉估計和出雲市芬蘭協會有關，所以這裏掛了旗。

對了，前臺會寫每天的日落時間，今天是18：45。

 


如果你忘記一次性碗筷那些東西，這裏也有賣的。

 

 當然還有浮輪出租，每個200。


我們來了~


取了鑰匙我們就去我們的豪華海景別墅了~整個房子，我估計七八十平米。下面視頻，我們先讓B老師介紹一下。

是不是很漂亮呢？我再給大家詳細介紹。

客廳。我很久很久沒看過電視，那個時候有殘奧，正好看了一小會。

大家都先來露個臉先~


 

B老師的刀工很好，洋蔥絲切的很細。

 

這就是我爲了這次旅程刻意喫完7-11留下來的沙拉盒，原來是喫涼麪的。

 

下圖辣椒（ししと） 是早上在Familymart買的，燒烤很好喫。蒜烤茄子去年raku老師在明浜也弄過。這裏有磨蒜泥器，我看她磨的很使勁呢。

18：35的時候，我們都放下菜刀，出去陽臺迎接日落。

 

太陽伯伯，我們來迎送你下山了~

我們拍了很多浪漫迷人的剪影照。

 

太陽要下山了

我們弄好了，要出發去燒烤場了。走路的話只要一兩百米，不過我們東西多，那邊也有停車場，就開車去了。 

這是我們部分東西

 

 

我們到燒烤場了，4個人一張桌子，很寬的。


 這裏還有鐵板及鍋鏟提供，所以這裏提供的燒烤套餐有焼きそば的（炒拉麪）。

 

這裏的燒烤套餐是手ぶら(tebura)的，這是我在旅遊網站學的日語，意思是“空手”，即你什麼都不用帶。這裏給的燒烤套餐的確很足，三公斤的碳，着火劑，打火槍，兩個網，四把扇子，兩個夾肉夾，一把手套，應有的都有。

牆上面的說明，我後來看了才知道我們爲什麼折騰那麼久才能生火。

今次raku老師負責生火，去年我和洗碗王子在明浜生火很快就弄好，今年折騰了很久，後來我看牆上指示，我們一開始碳就好像沒方對...,其實去年明浜學過，不過忘記了，下次我肯定記得了。

 

 

這裏給的兩個烤網質量非常好，烤完都不黑，絕不是百元店的貨色。

 

四個人猛力扇風（給了4把扇子的）

 

 終於差不多好了

折騰了好一會，終於有的喫了，耶！現在來看看我們的盛宴~

 

自從去年在明浜喫過煙燻烤パプリカ後，我就愛上了它的煙燻味，今次買了三個大的來烤。當然還準備了我最愛的烤蒜。

很久沒喫過烤雞翅了，皮脆脆的很好喫，我們還烤了一片面包。碳烤的味道和烤箱的真不一樣。

 

看看下圖用錫紙裹着的烤魚，我烤好了打開給你們看。

 

我的魚好了~

特別跟大家推薦圖裏的烤炸豆腐厚揚げ(あつあげ)，烤出來外脆內嫩特別好喫。

我們的調料有raku老師的香蔥，她的刀工是不是很好？對了，我們沙拉的胡蘿蔔絲也是她切的哦。

 美顏自拍~

 

 等到11點我們才玩桑拿及泡澡。一開始我和HY沒看清楚，把桑拿弄錯了，搞了很久才一點點熱。不過進去玩了一會還算是出了點汗，後來raku老師會搞，幾分鐘就熱了。應該左面有1-8刻度的調零，右面的調到最大就對了。

 玩完豪華海景別墅的私人桑拿，怎麼少得了豪華的泡澡？今次我帶了還沒上市的新產品 - 奢華超多泡泡沐浴球過來給大家玩，讓大家極致享受。現在看看我的模特玩的多開心。

我的沐浴球。優先給你們看還沒公開的廣告照（特別鳴謝攝影師raku老師）。


 

感謝我的出浴模特。至於是誰我就不說了。

 

 

我們玩到12點多1點才睡，還好第二天不用早起。

8月27日（週五）

早上我7：30起來先做早飯了。我們喫炒雞蛋，煎香腸和烤麪包（基本每次出門都這樣喫）。廚房的烤箱折騰了很久不會用，之後raku老師就說用鍋幹煎，效果也很好呢。


 我們要來享受無敵海景早餐了~


 

 

又來美顏自拍。

 

 

早餐的時候B老師還教我們做她以前在居酒屋學會的日本菜呢。她告訴我們一個很重要的調味料 - 九州的久原高湯（つゆ）。

我們早餐買了一大瓶咖啡和牛奶，沒喝完，真好混合了放在保溫箱拿去海灘喝。有冰咖啡真爽哦~

最後我們等到快10點退房才走。今天起來我往陽臺一看，發現浪小了很多，看風速也就兩三米，我有信心能遊的。

我們到了道の駅下車後，聰明的B老師就說今天那麼多車肯定能游泳，最後，果然是，耶！


 這裏很美，下海前我們當然是先拍照~大家都要和這個小精靈拍照。

 

 這個Kirara的標誌在2020春天才立的。

下面這幾張4個人換角度拍的效果很好。

 

 我們現在要下去準備暢遊日本海了~

 

 

 這個時候游泳的人不多，很容易佔到好位置，遊得也舒服。

三個沒給浮輪打過氣的無知少女，一開始沒加變壓的東西，因爲覺得加了很難打，打到一定程度發現怎麼氣打不進浮輪？原來那個變壓器是一定要加的。 

話說我們地盤用的紅色墊子是我之前爲了去龍王島紮營買的，今次第二次用，買的真好。

 

後來有經驗的B老師出馬就會打了。


 開始游泳前，當然又要先拍照。今天藍天碧海，有時候有太陽有時候沒有，很舒服。


 

raku老師教，拍這種照片攝影師要蹲下來，鏡頭朝上，這樣人就會顯得跳的很高。

 

我專門爲這次旅程研發的高防水高度防曬，很管用，防水性能經測試確實很好。

 

大概11點，我們終於要下海了~

raku老師不會游泳，要扶着她。

看看我們的浮輪照，我特別喜歡，都拍的我們笑很開心。對了，我一直說浮輪，日語浮き輪(ukiwa)，這是我很得意我教raku老師的日語。

 

我不帶浮輪遊了一小會~

與此同時，HY自己一個在海邊享受。

 

B老師在我們的地盤曬太陽。大家注意一下B老師黑色的小傘，打了折都3000多，怪不得那麼美。

 

 這個海灘真是很漂亮，近看是道の駅，遠看是我們的小別墅，中間兩個風車，完全是北歐風，差點以爲自己去了歐洲呢。


遠處是我們住的別墅，現場看就像高級房子。l

兩點半左右，我和raku老師去洗澡了。洗澡的地方有點窄，不過還夠用。大家記得備好三個100元硬幣，洗澡三分鐘，中間不能停水。

洗完澡把腳擦乾時... 我百元店的拖鞋竟然顏色！！！我回去馬山扔瞭然後在百元店買了300元的。

 

 

raku老師纖細的腿曬黑了。

洗完澡我們就去道の駅喫無花果冰淇淋了，300元。我們4個的冰淇淋，有無花果的清香又不太甜。



 

 

 

這裏還有無花果汁。

 這是小夥伴在道の駅喫的午餐。

喫完快4點半，準備去昨天的無花果園打包外賣。店主的兒子又在門口看門，看到我們就去叫媽媽。今天我們來的早，無花果還沒摘好，店主說現場給我們摘兩公斤新鮮的。

等待的時候，小朋友就和我們聊天，好像在幫他媽媽招待客人那樣，他不玩手機，給我們看了他的螞蚱。

 

他很高，我還以爲他讀小學，不過原來還沒。raku老師問他是不是喫無花果多了所以長的那麼高，他說應該是吧。他還告訴了我們他去過的地方。還有，原來今天是他媽媽的生日呢。

 今天生日的媽媽。

我們的大無花果，每盒14粒，1000元1盒。

幾位賣無花果的女孩。


最後五點多發車返程。原計劃晚飯前回來的，最後9點到家(感謝有HY開車)。兩天一夜旅程時間雖短，不過經歷了很多，四個人都非常開心~ 今次我雖然不是第一次去海灘游泳，但卻是最開心的一次呢。
