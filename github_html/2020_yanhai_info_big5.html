2. 初秋東廣島巖海小資一天遊 - 相關行程信息

(所有價格2024年底更新)

我們的行程地圖(線路取自navitime)(看不到打開此)：
(紅:住宿   藍:景點    綠:喫飯    黃:廁所  紫:停車場)


三原旅遊官網（巖海所在地）
政府旅遊報告：(2019)三原市観光戦略プラン(観光課)

西條早餐店
西條除了コメダ(komeda)，早餐店我去過的有Ncafe(只限週末)及hana cafe，都在國道375的御薗宇東側。兩家都不錯，我喜歡Ncafe多點，我們這次也在這裏喫。

久井巖海
日本最大規模的巖海，我們去的假日不過人很少。當時那裏廁所沒水，蜘蛛也很多。

巖海那邊附近沒啥喫的，要提前找好。我在地圖標了兩家(綠色)。想喫好點可以去北面的世羅町。

公園高爾夫球(パークゴロフ)
有9個洞，每個洞打的距離不到100米。沒打過的話挺新鮮的玩意。我們當時想去的白龍湖観光農園 因爲開錯車沒去成。這門票1000，租用球具不知道多少。另外東廣島福富有政府場地（道の駅湖畔の裏那裏）- 東広島市福富パークゴルフ場(西條以北，開車約半小時),是公認コース,門票1040，另租用球具310。一圈9個洞,再玩要再給錢。你不懂得話職員會教你規矩。分數是誰打更少的杆進洞就贏，有計分表提供。這個地方經常有比賽，去的話最好先打電話。
