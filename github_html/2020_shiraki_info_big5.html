2.廣島百名山白木山 - 相關行程信息

我們的行程地圖(行車線路取自navitime，紅色登山線路取自yamap)(無法顯示點擊此)：


白木山簡介
yamaquest廣島縣排名第四。攀升800多米，我們原路往返，走短的路線，上下都是兩小時。這座山樓梯挺多，但不全是。不算最安全的路，但也不危險，對體能有要求，我們下來都很累，大家的腿也疼了好幾天，最好帶登山杖。

白木山登山線路及記錄參見yamap。

我原來規劃的登山線路如下（來自yamap)，但後來只走了黑色有圈的那段，從白木山駅附近有正式登山口上。另外，如果你坐電車，有條線路下山往上深川駅。

停車場：登山口附近沒看見正式停車場，我們停在大概1公里外的Shoji超市。
廁所：登山路上沒廁所的，還有標誌禁止隨地上，要上就在超市上。
