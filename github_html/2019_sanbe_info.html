大田旅游官网
我们今次去的地方属于岛根大田。石见银山和三瓶山是这里的大景点。市政府下属有石見銀山課，属教育部。

我在大田政府网站找到大田市新観光振興計画（2023-2026）(案)，我简介一下。主要想看相关数据参考大田市新観光振興計画（2023-2026）(資料編) 。

到2040年，大田人口大概比现在减少三分一（p2）。人口减少就产生一些地域课题，例如经济，自然环境没人保护，文化社区活动减少等，所以要发展旅游业吸引人来。 大田的旅游人数在2008年最高，190万。疫情前2019年100万，整个岛根而言算是游客少的地区（p10,最高为出云和松江，2018年各有1000多万）。2021年大田观光消费额25亿，雇佣创出效果381人。为了振兴旅游业，他们定下7个课题(p15)，包括人才培养，观光资源优化，增加过夜旅客，环境整备，宣传方式，广泛合作等。每个课题报告内都有具体政策，其中一些政策制定根据之前做的统计调查。

关于广泛合作方面(p18)，其中一个政策是和其他旅游机构联系帮忙宣传。我对这个深有感受，在日本相同类型的博物馆会互相帮忙宣传，你去某家博物馆就可以看到其他家的海报。西条办音乐会的场所くらら也找到附近地方音乐会的信息。旅游宣传方面，小地方住宿网站也能找到同一个地方其他住宿（及游玩）信息，不怕竞争（例子吴市上蒲刈民宿かつら亭)，奥出云观光协会的airbnb收录的民间住宿，我觉得也体现了一种官民合作。日本这些地方真的很值得学习。大小携手打造地方发挥更大力量。

政策制订他们也考虑到旅游使用者意见，具体不同地区实施计划在p19。日本很注重意见，参加任何活动（音乐会，イベント等），博物馆参观都有问卷(アンケート)，我想日本旅游那么吸引，不停听意见不停改善是原因。

下面我简单介绍一下三瓶山那边我研究过的地方：
石见银山
里面好些地方，我们停车在遗产センター(Center)，骑车到龍源寺間歩，两者距离2.3公里（不能开车)。我们去路线上一些不用门票的景点。以前有钱人的官邸也是景点，我们没去。

资料室400（外国人有优惠，带上护照），龍源寺間歩500（也是外国人有优惠）。另外有导游团大久保间步(费用几千)，这地方不能自己去的。除此之外，有一些银器制作体验。

以前矿工条件非常艰苦，龍源寺間歩是个充满血泪的地方，开采还死过人，当时一个队友说一个人不敢来。

租用自行车在おおもり会館，电动2小时700，非电动500。遗产センター(Center)到龙源寺路上有点坡，租电动好。

景区内有饭吃的，我们去了距离停车场1公里的地方群言堂吃。这里吃饭地方不多。

石见银山有导游协会，有不同路线导游团，还有英语服务，每团8000。

三瓶山景区信息
官网:https://www.sanbesan.jp/mountain

除了三瓶山登山线路，还有附近的琴引山・大万木山及石见银山的。还有住宿游玩公交车等信息。

三瓶山东南西北都有登山口的，登顶要注意不要随便选一条路下来。我们从官网北部1.姫逃池（ひめのがいけ）ルート上去，然后从2. 名号（みょうごう）ルート下来。

比较特别的景区是埋没林，80年代偶然发现的地下森林，成因和火山有关。

住宿
三瓶山北の原キャンプ場

我们住的地方，政府营地，能网上预约，10人栋ケビン(cabin)假日料金23000（我们2019去24000，我惊讶怎么会降价），可以扎营（帐篷寝具烧烤用品都能租用）,开车去姬逃池登山口几百米。

附近有天体观测馆，周六对散客开放。

三瓶温泉
温泉都位于三瓶山南边，我们去了最贵的さひめ野温泉，这是温泉酒店，日帰入浴500，租浴巾300。

大田温泉津也很有名，不过离我们远就没去。

超市
我们去的美乡町A coop（6点关门）,营地以南约10公里，靠近我们泡的温泉。

大田市有Aeon,good day,我都标在地图橙色。
