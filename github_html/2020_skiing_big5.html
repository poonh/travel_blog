2.疫情下爽極免費滑雪 - 遊記

日期：2020年2月21日（週五）

第一次滑雪，這個滑雪場非常棒！ 由於免費，我們請了教練呢。

我們要出發了！

2月21日週五翹班去，原來怕路上結冰，不敢開車，不過出發前一天facebook說普通輪胎沒問題，打的電話去確認後，我們就8點從大西條出發，沒到十點就到。停車後就去接待處，接待我們的大叔非常好~
這裏就是滑雪場門口， 遠看就看到滑雪道了。
之後我們就去選衣服了。很多顏色都很花哨很好看。 褲子對我來說要麼太小，要麼就很長，不過長點也沒關係。花ちゃん最後把褲子弄壞，不知道爲什麼破了.....
我們的衣服，lele幫我選了紫色的外衣，我找到一條褲子搭配
等到換鞋的時候就很痛苦了，我穿的38號鞋對應24釐米。第一次穿滑雪鞋，很難穿，由於要保護，非常硬，很難穿進去。穿上之後還很難走路。
大概11點我們完全搞定（即是我們挑衣服穿鞋子的時間得1小時），我們就坐索道上海拔1000的滑雪場起點了，如果你覺得餓，可以在山下換衣服附近的餐廳先喫，但爲了方便，一般都在1000米那家餐廳喫。再次提醒，記得帶好錢上去。
 
第一個索道，上去海拔1000
到了我們就看到滑雪學校。因爲人湊夠了，就馬上開班了，總共1個半小時。去滑雪場地，要先坐無蓋的索道上升200米。當時我們4個學生，兩個老師帶我們上去。和我一個索道的老師以前在香港教過音樂的呢。他會說幾句英語。
這個索道沿線是可以滑下去的，海拔差兩百米，然後再坐索道上來
 
老師先帶我們做熱身，教學內容主要是如何剎車，方法就是膝蓋彎曲，屁股往外，腳底的撬成八字。老師教的時候好好的，大家都能剎車。老師的態度就是典型的日式服務，掛着笑容，老是鼓勵，令你覺得很舒服。

上課完後我們就去喫飯了，就坐第二個索道下去。這裏的餐廳比一般貴一點，要1000多。小喫有咖啡冰淇淋等等。很驚喜這裏竟然有泰國的綠咖喱，連上飲料1500日元。
 
很好喫的綠咖喱雞腿飯
餐廳外面可以曬太陽
餐廳裏面
下幾圖：山頂我們滑雪的地方 
 
喫完飯後，我們就回去滑雪了。由於剛纔老師教得都好好，我們都安心滑了....第一次獨立滑雪全軍盡墨，全部摔倒（老師也交了如何摔的，要屁股朝下）。不過我後面就好了，我也漸漸掌握技術，到最後還會拐彎呢。老師也教了摔倒後如何起來，就是翻身，但我穿了那個鞋子完全翻不了，所以摔倒後我都是把鞋子給脫了的。花ちゃん的視頻我也拍了，不過就不公開吧。

滑雪的時候，雖然運動量不大，不過很容易渴。我們每次就滑幾十米，到一個平臺後就拿滑雪工具回來，這樣反反覆覆的走，東西也挺重的，所以第二天就手臂痛了。

我發現滑雪場內只有我們這樣來回反覆走，其他人都從頭滑到底，然後索道回來。玩到四點半左右，我覺得來回有十次左右吧，我們就下去了！
我們下來了！
脫鞋後大家都感覺解放了，坐車回去的路上還有穿着滑雪鞋子的感覺。一週後，我穿鞋子的痕跡還在
晚飯回家喫好的~
今天完了一整天，非常完美的結束。明年再來！
