3.安芸高田美容歷史文化之旅 - 遊記

日期：2022年1月10日（成人節）

引言：
11月的某個週六，我正網上查資料準備12月 《囲爐裏慶聖誕》的行程，查着查着不知道爲什麼查到安芸高田去，知道那裏有神樂看。我第一次知道神樂是11月初《深秋神石高原道後山紅葉之旅》，但當時也沒有多關注。但這次再接觸引起我的興趣，同時我發現安芸高田有酵素風呂及毛利元就古蹟，可以安排一天行程。於時VIP洗碗公主大駕光臨時，我們就進行了一次安芸高田日歸旅行了。

去之前兩個週日，和J姐聊到我們這次旅行。碰巧她那天不用帶孩子，就和我們一起去玩了。

行程簡介：
大家可以先看一下我的行程簡介視頻。


行程開始：
早上7：20接到神樂門前湯治村電話，說今天疫情取消公演（現在是疫情暴漲期）。雖然今次行程是神樂爲主導，但我們是一天的行程，不看神樂也有其他能玩。最後發現，其實看了神樂我們行程時間不夠，那就留到下次好了。

我們8：00大西條出發，加上路上去便利店及小段開錯路，我們9：27到森藏酵素風呂。店家 已經準備好等我們了。當時只有我們4個人，

森藏位於一個小山坡上，環境很好，在森林裏。外面有凳子，可以欣賞景色。

 一進去，人家就上發酵的黑加侖子果汁（ カシス）。店家真會做生意。果汁不是他們自己生產的，不過配合發酵的主題。我們喝着挺好喝的，走的時候大家都買了一瓶。這個果汁亞馬遜也有，價格和這裏賣的差不多。

嘗一下先
這裏也是木工工場，很多店家制品。

喝完我們就進去換衣服了，不用先洗澡的，直接換上店家給的衣服就行，玩完才洗澡。
好像防護服

玩完大家都非常餓，不過這裏很美，大家都拍完照才走。

我鄭重宣佈，下面這張是raku老師的相親照，有誰有意附上簡歷在博客留言裏。

 拍完照都11：30，大家都很餓，就直接去神樂門前湯治村午飯了。到達之前經過音樂公路メロヂーロード，放的神樂音樂。這個音樂比較陰沉，聽着不是很明顯。今天雖然沒神樂公演，但這裏人不少，停車場差不多停滿。

歡迎光臨神樂門前湯治村
我們去千兩萬兩喫飯。這裏很多菜都是其他地方找不到的，我們看到菜譜真是什麼都想喫
對面是權兵衛，主要喫烏冬的
看到每個菜都想喫的菜譜

我們上菜了。我們點的幾個菜的確都很好喫。

大家請慢用~

我們的豆腐湯，這個豆腐就是隔壁那家豆腐店福助トーフ做的。我們後來去買了幾塊晚上喫。晚飯我再詳細介紹。下面這個湯鍋今晚我們喫關東煮也用，喫了我才知道我那個關東煮保溫鍋該如何玩。
這個豆腐的質感和平時喫的完全不一樣，非常好喫
我的魚炊飯。右上是炸魚皮。第一次在日本喫炸魚皮

 下面這個面我們都一致推薦，好像韓國的石鍋拌飯不過換了麪條。面是脆脆的，價格也便宜（連稅700多）

大家聽聽麪條的聲音
竹炭鴨肉烏冬。湯有濃厚的鴨味
大蝦天婦羅

夜叉烏冬我們4個人分着喫。洗碗公主喫最後一口。喫完赫然發現湯竟然呈心形，她今年一定會有桃花運
洗碗公主知道自己要走桃花運笑得特別甜。大家注意下她的叉子是不是很大，不知道這是不是就是傳說中的夜叉...
喫完飯我們就去逛街了~先欣賞下這裏的街道
這是溫泉，不過現在不敢泡
おもちゃどんじゃら堂是賣懷舊玩具的。raku老師買了，一會兒去毛利元就玩
我們現在去參觀一下神樂博物館
這裏有圓頂劇場(ドーム)，不過只有特別大型公演纔開放的
神樂歷史介紹
既然不能看神樂，就買個夜叉上臺表演一下吧
博物館還有神樂口罩賣，現在口罩款式真是啥都有
這叫わら(稻草)工芸，照片我拍來研究用的
這是廁所門口，也貼了神樂的東西

歡迎參加神樂問答挑戰
raku老師對了多少題？

神樂看完了，我們現在要出發去下一站 - 戰國名將毛利元就的郡山城了
到了明年，這裏就500年曆史了
爲了讓大家看得更投入，我今天還專門請了導遊

據導遊介紹，郡山城位於吉川盤地，附近有可愛川。毛利元就在此出生及下葬。位於廣島市的廣島城就是他的孫子建的。

我們現在要向毛利元就參拜道出發了

正式出發前，大家請給我一點時間跑個
然後在入口領個導遊圖
這裏風景挺漂亮的，是古典森林風。我們先來欣賞風景
刮颱風不知道會不會塌下來

 路上的路挺好走，一般是這種樓梯。

看完風景我們就去看歷史吧。出發了
目的地是郡山城最高點 - 本丸

首先去墓地看。他家族的墓地都在此。這裏古木蒼天，看上去很氣派，的確很像偉人的墓地。

通往墓地的樓梯
毛利元就兒子的墓地
這是百萬一心碑。你看碑文，能猜出原來寫的是什麼嗎？原來寫的是“一日一力一心”，大概是大家同心的意思（編者按：上圖是後來的仿製品）
對歷史感興趣的可以看看
路上很多凳子，大家中途休息 一會
還有530米到目的地，大家加油~
在路上，raku老師拿了她的懷舊玩具出來，以前她怎麼吹也吹不來的。今天她很容易吹起來了，意味着她今年會有好運
水晶球
據導遊介紹，這裏有很幾個圈圈（日語爲"丸"）。中心是本丸，然後有二の丸，三の丸，都是防衛的
到了本丸最高點後，我們就返程了。由於上次七國見山的教訓，我們決定原路往返
這裏的樹有點亂
下山時有點滑，raku老師變了我御用的raku登山杖

走着走着，洗碗公主問這是不是我們走過的路。當時我和raku老師都堅信這是我們的原路，後來一看地圖發現不是。我們都心理作用認爲我們是原路往返，聰明的洗碗公主是對的~我們誤打誤撞，最後又走到計劃的路線上了。

走到下去，又有公園，我們又玩了一會。

raku老師對未來的憧憬
親密照
坐一會抓緊時間玩手機先~
這個時候陽光正好
最後到了三矢之碑，在人家的後花園，但可以開門進去

據raku副導遊介紹，廣島足球隊サンフレッチェ広島(廣島三箭)就是取自毛利元就的三矢之碑。三矢說的是他三個兒子，訓跡的內容是：

”一本なら簡単に折れてしまうが、三本束ねれば簡単には折ることはできない。三本の矢のように三人が力を合わせれば毛利は安泰なのだ “

(deepL翻譯：一支箭可以輕易折斷，但如果三支箭綁在一起，就不能輕易折斷。 就像三支箭一樣，如果我們三個人一起努力，毛利人就會安全。）

更多參考見此。

我忘了我們在這裏逛了多久了，下山反正快5點了。我們現在要去行程最後一站 - 晚飯，先去熟悉的Mr Max買菜，然後就回家喫關東煮了~

raku小廚再一次秀刀工，她今晚還有祕製醬汁要上演，下面我再介紹
開飯了，歡迎J姐第一次來我家作客
乾杯

我們在神樂門前湯治村買的豆腐，就是超市的厚揚  (大家記得我的網紅料理外脆內嫩烤厚揚嗎？）但這個炸的皮厚很多，好喫很多。白豆腐也特別結實，切下去的時候感覺就和一般豆腐不一樣。

結實的豆腐
我們的關東煮
這瓶就是raku老師的祕製醬汁，用的紅糖+醋+水+柚子皮。柚子皮切條就可以，很清香，可能是因爲加熱了，下次我用柚子皮也加熱試試
喫完飯，出張洗碗人洗碗公主又出場了~自從她去了高知之後，大家都沒見過她在我的遊記洗碗了
最後，兩個傻婆又來情深告白了
最後 ，洗碗公主結束廣島的旅行 ，回高知找她的四萬十王子了

今次行程開心結束~期待下次去聽神樂公演。
