金秋中山道二戰松代大本營之旅 - 相關行程信息

(所有價格爲當年)

日記：2017年11月

行程：
馬籠宿 - 妻龍宿徒步
松代大本營（二戰遺蹟)

松代-東京大巴: http://willerexpress.com/st/3/en/pc/,只有這家公司有車從松代返東京）

中山道部分
長野旅遊官網：http://www.go-nagano.net/
這個網站到了看紅葉季節還有預報的(http://www.nagano-tabi.net/modules/season/kouyoudayori.html)。想抓住最好的時間看紅葉，事前看看這個網站就行。
長野市看猴子泡溫泉（坐車好像不到一小時，我覺得這個地方很快看完，所以就不去了）：http://en.jigokudani-yaenkoen.co.jp/
中山道介紹：https://zh.wikipedia.org/wiki/%E4%B8%AD%E5%B1%B1%E9%81%93
中山道是古代通去京都的路，詳細我也沒太瞭解，我們走的是整個中山道其中一段。馬籠宿位於岐阜縣，最近城市是中津川，這裏離名古屋也不遠。妻籠宿南木曽則位於長野縣。
我們走的路有兩個馬籠宿妻籠宿兩個小鎮，中間有江戶時代建築的村子，還有古木參天的森林，竹林，楓葉等等，集文化及自然景色於一身，秋天去看紅葉實在是好時候。
妻籠宿旅遊官網：http://www.tumago.jp/english/
妻籠宿馬籠宿提供行李搬運服務，方便徒步人士，費用500日元，不是全年都有的，詳情：http://tumago.jp/event/201710/post-35.html
如果你是想走到南木曽的話，就一定要自己拿行李了。
旺季的時候妻籠宿馬籠宿有公交車提供的，詳情看官網。
南木曽旅遊官網：http://www.town.nagiso.nagano.jp/kankou/index.html
我們徒步的地圖妻籠宿官網有，中間有清晰路標，想要具體點的，wikiloc有很多，例如：https://www.wikiloc.com/wikiloc/view.do?id=6724765
這條路特別受西方人歡迎，我們徒步中遇到的人都是西方人。
中央到馬籠宿停車地圖（即新宿出發到中央道馬籠下車的地方）
http://www.jrbuskanto.co.jp/bus_stop/etc_map.cfm?code=210012
中津川的長途大巴(到名古屋，松本，長野)：
http://www.town.nagiso.nagano.jp/data/open/cnt/3/13700/1/20170326_en.pdf

松代大本營部分
松代大本營官網：
http://matsushiro.org/daihonei/index.htm
這是一個日本人不太知道，但又非常重要的二戰遺蹟。當時日軍垂死掙扎，在1944年11月準備把東京基地搬過來。這裏有10公里的地下隧道（用來放軍事用品），慰安所及皇室居所等。
長野-松代大本營公交（八十二代銀行下，約半小時車程）：
http://www.alpico.co.jp/access/nagano/matsushiro/#timetable
長野市-新宿大巴：http://www.alpico.co.jp/access/express/nagano_shinjuku/
