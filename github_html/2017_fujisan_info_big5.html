攜老帶嫩遊富士三湖 - 西湖河口湖山中湖 - 相關行程信息

所有價格爲當年價格

日記：2017年5月

交通：從東京過去，新宿有大巴直達，到河口湖單程票1750日圓，車上帶廁所。準時約1小時40分到，但估計必定會晚點。去程是週五上午，我們晚點一小時吧，返程是週日下午，晚點差不多兩小時。瘋狂晚點的原因是因爲沒有高速（個人認爲）。
買票網址： http://bus.fujikyu.co.jp/highway/fujisan
網上訂了票然後最晚開車前10分鐘到便利店付款就行。

公交票：
富士五湖公交：h ttp://bus.fujikyu.co.jp/rosen/shuyu
如果只去河口湖及西湖，就買1300的公交pass，也有貴一點包遊船和索道的。
我們因爲要去河口湖山中湖及西湖，買的2550日圓，用兩天的passport: http://bus.fujikyu.co.jp/otoku/fuji-passport
​河口湖車站售票處出來就是河口湖周遊和西湖周遊的車，去山中湖的在6號站臺。
富士五湖區域點對點公交查詢：http://transfer.navitime.biz/fujikyu/pc/map/Top
河口湖-山中湖時刻表： http://bus.fujikyu.co.jp/rosen/detail/id/3/#d_h01
河口湖周遊及西遊周遊巴士時刻表： http://bus.fujikyu.co.jp/pdf/rosen/fujigoko/20160502/omni_timetable.pdf
去山中湖的車半小時一趟，發車準時，但中間必定晚點，到山中湖役場前起碼晚十分鐘。這趟車可以用pasmo，河口湖周遊及西湖周遊不可以。

富士五湖簡介：最大及交通樞紐是河口湖，住的也多，交通網絡完善。第二大是山中湖，位於河口湖東南面，從河口湖過去要坐半小時的車，海拔1000米。西湖位於山中湖西面，也有開發。精進湖及本棲湖在西湖西面，兩個都很小，玩的不多。

河口湖旅遊官網（景點坐河口湖周遊巴士）：
http://www.fujisan.ne.jp/
那裏景點很多，很多博物館，我就簡介自己去過的。

大石公園（公交站：河口湖自然生活館）：
位於湖的北側，面對着富士山，不同季節開不同的花，拍照絕佳的地方，有小餐廳可以喝咖啡喫冰淇淋
自然生活館（公交站：河口湖自然生活館）（可以做果醬及摘野果）：http://www.seikatsukan.jp/
遊覽船（公交站：遊覽船）： http://www.fujigokokisen.jp/contents/ensoleille/
930日圓一位，航程半小時。
天鵝腳踏船：和遊覽船同一個地方，2000日圓半小時，坐三個成人，5月底時5點半還有得玩，最後不知道幾點結束。
另外沿湖還有好些博物館及神社，自行參考官網。我坐車經過淺見神社，外面看上去古墓蒼天的，覺得比日本一般神社有特色。此神社位於富士吉田市，從河口湖坐車去山中湖的巴士會經過。
溫泉（公交站：船津濱溫泉街）：河口湖東北側離車站大概1公里的位置有很多溫泉酒店的。
運動：
皮艇，釣魚及騎車：http://www.c-ls.jp/%E3%83%AC%E3%83%B3%E3%82%BF%E3%83%AB
划船（河口湖北側）：http://www.fujisan.ne.jp/spot/info.php?no=163
另外，河口湖車站附近有自行車店鋪的，百度一下該能查到。
其實河口湖西湖山中湖我覺得都不是特別適合騎車，因爲沒專門自行車道，機動車又多。
美食：餺飥 （當地特產麪食）
有兩家出名的連鎖，下面這個在河口湖車站對面就有。
http://www.houtou-fudou.jp/
我們去的這家，山中湖店:http://www.kosaku.co.jp/
麪條看上去像刀削麪，湯是素菜湯，挺鮮味，麪條分量正常，因爲還有南瓜土豆，喫的很飽，可以三個人喫兩份。而且價格也不低，1500左右一碗。

西湖（下面景點都是坐西湖周遊巴士）：
手作坊：
西湖的里根場：http://www.fujisan.ne.jp/iyashi/dantaitaikenichiran.html
官網有優惠券，原價門票350元減50元，6歲以下免票
這是一條仿古的村子，裏面有造紙，蠶絲，陶藝等手作工藝，都要電話預約。我試過打電話去造紙，人家不會英語，預約不成。
和服：
也是里根場的，火之見屋，1000日圓。里根場不大，雖然和服可以穿一天，但大概一兩小時就會逛完。

山中湖（下面景點都在河口湖6號站臺坐車，假日到站必定會晚點）：
紅山之湯溫泉（公交站：花之都公園入口）: http://www.benifuji.co.jp/
這個溫泉賣點是可以邊泡溫泉邊看富士山，門票800，網上打印優惠券便宜100。有1000日圓左右的簡餐，也有需要預定的名貴套餐（3500-5000日圓，價格包入場）
花之都公園（公交站：花之都公園）： http://www.hananomiyakokouen.jp/
看花的，要門票。
忍野八海（公交站：忍野八海，1小時1趟）： http://www.vill.oshino.yamanashi.jp/8lake.html
忍野八海是八個泉眼，世界文化遺產。我原來打算去，後來起來晚了時間不夠。
遊船（公交站：觀光船）：http://www.yamanakako-yuransen.jp/
有天鵝形遊船，也有天鵝腳踏船。

富士芝櫻：http://www.shibazakura.jp/
每年四五月份賞富士芝櫻，位於富士五湖最西面的本棲湖附近，場地能看富士山，要門票，展覽期間新宿有車直達，一天大概兩三趟。
（我原來打算去看這個，但新宿的車是8:45，帶着小朋友太早了，當然到了河口湖也能另外坐車）

住宿：這次住的airbnb，在河口湖附近
