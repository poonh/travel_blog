车票购买攻略

售票机买票操作：みどりの券売機で観光列車etSETOra（エトセトラ） 広島→尾道のきっぷを買う （我看过才去买的）

这趟列车只有两节车厢，周末挺不好买的，所以我们平时去。当日车上有空坐，不过海侧位置满了。提前一个月买票，我们2月24日的票，车站售票机是1月25日开始售卖。我选了两个最好的座位(预先看好官网座位表)，是第1节车厢10A及10D。我买的グリーン(green)券，即额外坐这车需要的票，1000日元。另外加上普通广岛-尾道车票（1520日元,2024 价格，可以坐车时才买）。坐这趟车总成本2520日元。

绿色是1000元事前买的グリーン(green)券

午饭
 这里半山吃午饭，可以看濑户内海的地方有两个：1)>cafe holoholo，网上看是开门时间不过当时我们去没开。2)cafe holoholo附近的酒店-尾道ビュウホテルセイザン吃泰国菜，只有周末开。 尾道駅附近的商店街也很多吃的。

千光寺公园
尾道駅北口出来走一百米左右就到登山口(公园西侧)，有好几个的。攀升100多米，半小时内到顶，全程有带扶手的楼梯，很简单的,然后从东边下来经过猫之细道及千光寺新道。

咖啡店下午茶
原计划去评价很好的高档咖啡店satie,不过又没开，最后去了尾道商店街的coyote，顺便逛尾道商店街及尾道商工会议所参观。尾道特色咖啡店很多。

平原ぽっぽの湯(温泉)
位于车站东北3公里的。这个温泉看网站玩的很多，特点是女士有汉方药草桑拿。去这需要在车站坐公交车，详情查询navitime或おのみちバス-->路線バス時刻表各路線-->ＪＡ尾道総合病院線、市民病院線。

温泉平日860円・土日祝日1000円(2024价格)

上面google地图都标了上面的点及其他信息。

注意事项: 这里不是大城市，小店不靠谱的。这次经历了两家店网上说开，实际都不开。下次去这种店最好先打个电话问问。
