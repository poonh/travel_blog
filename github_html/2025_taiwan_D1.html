3.携老带少游高雄 - 游记

D1:弘记肉臊饭 - 龙华市场 - 瑞丰夜市

引言：

这是2017年携老带嫩游富士三湖的续。当年的“嫩“已经升级为”少“。当年陪她玩低B天鹅船，现在可以陪我玩我想玩的水上自行车（当然我想玩更多）。之前她说想去台湾，后来没下文，某天突然来电说之后一周去（约10天后），还好我手上已经有高雄行程，住宿地点航班具体计划都有(前几个月做行程时我还做的没劲，心想自己又不去，现在看来当时是为自己这趟旅程做的呢)，只要补充公交信息就可以。我不可以进行说走就走的旅行，没有足够的知识我旅行获得少。当时一老嫌热，不想去。后来想着想着又去，不过发现自己护照过期，幸好来得及。决定去后马上办护照，出发前那天拿到，刚刚好。

今次行程，特别鸣谢某尊某公子某兄klook赞助，省了我们台湾在地超过三分一开支。

行程开始前（买手机卡）：
其实我是不需要用网的，因为行程每一步都规划好，有离线地图也不需要用网看地图。但需要用网叫车及可能和同行人临时分开，还是要手机卡。两老原来在鸭寮街买了卡，不用实名登记的。我回来一看包装说明不清楚，我之前参考klook的手机卡有份插卡要连续用，也有分开用，天数是00:00开始算，还是插卡开始算，这些包装要说明的。我看这个没有，上网站看，发现网站不存在。及后查到原来这卡是鸭寮街某个店自己出的，网上评价也差，顿时不敢用。我们去鸭寮街换卡，换了两家大公司的。一家是happy，一家数码通（都要实名认证，但我感觉很敷衍，随便输入姓名及身份证号就可以，即时开通的。。。）happy包装给我的印象较好，有whatsapp客服，另一家没。不知道为什么，到了台湾，其他人手机都没问题，我的试过两张卡都没反应（我联系happy客服给我语音回信息，这水平不行）。最后用了一老去大陆用的lucky2，还好我带上这张卡备用。我记录此事，最重要记得lucky2最可靠。

当时一老还担心鸭寮街老板不给换卡，不过人家很容易就给换了。我们顺便去鸭寮街买了些水果，见识了“少”很喜欢的“西九龙”，水果比起我乡下便宜好多。。。

今天行程：
主要在酒店附近（捷运巨蛋站)活动，主要为吃。中午吃米芝莲肉臊饭，晚饭吃夜市。

飞机晚了起飞，不过飞行时间没有说的那么长（说的是1小时40分，实际只有1小时）。所以最后我们还是按时10:15到达。高雄机场没想到那么小，下机时只有我们一趟航班。当我们乘搭机场扶梯时，感觉很慢，好像那些省电扶梯没有人的时候那么慢。这种速度增加安全性，也特别方便老人，尤其机场有行李慢点较好。

下机后首先去游台湾金福气柜台抽奖。我们四个人都没中。在我们后面的两位香港人，女士抽中了。她身旁的男士说他们之前去台北，她也中了。有些人就是有抽奖运。抽不到奖，我们就去再买两张悠游卡（我一开始没在klook买所有人的卡就是以防我们中奖）。话说，我看悠游卡网站说65岁以上可以用优待卡，网站没说是否限于本地长者。我曾经咨询官网客服说台湾外的也可以用，但现场买说只限本地，而且他们也没优待卡卖。这点不一致性会给旅客造成麻烦的。。。

当时因为手机卡问题折腾我到17:15，然后我一个人去左营大巴站探路，确保第二天去小琉球行程顺利。话说这源于当年携老带嫩去富士山的经验，当时在新宿站坐车。早上的新宿站好多人，下车后很不顺利才去到大巴站，由于节奏太慢，刚好错过了车，幸好可以免费改。今次也是同样行程，我就先自己去一趟，把握好时间，并且看好捷运左营站列车开门方向，下捷运后扶梯电梯位置，确保我们迅速到达目的地。话说，我们经过过的车站（从左营R12到机场R4），全程都在同一侧开门。下过车的几个捷运站，扶梯电梯都在同一个位置，实在很方便赶时间的人。另外一点我喜欢的，就是每个站都有编号，首尔和釜山的地铁也是有编号的，对于外地人而言，只要记住坐车方向往大号码还是小号码就可以，不用记着站名，毕竟站名不好记。如果坐错了方向，有号码会更容易发现。

标了号码的车站，往南终点是小港，往小号码方向坐

去左营车站过程非常顺利，我不到6点返回，然后去瑞丰夜市晚饭。从酒店步行过去约6，7分钟。经过一家职业学校。youtube关于夜市视频很多，之前看过一个关于瑞丰的美食很吸引，不过我是重环境多于味道的人，所以夜市去一次就够。有些店有座位，不过很窄。有些只能买完站着吃。如果你喝饮料，一只手拿着饮料，另一只手拿食物，还要用筷子，还要拍照，实在无法吃。整体而言，夜市我觉得炸的食物挺多。不同食物种类也多，除了吃，还有很多玩的（详看下面照片）。如果酒店有足够空间，买外卖回去吃能吃的舒服点。

特别提醒，去夜市记得带个垃圾袋装垃圾。夜市垃圾桶不多，主要装需要边走边吃食品的垃圾。

去夜市前经过职业学校，我D3去了里面的操场巡视。据我们小琉球水上自行车教练阿彬说，台湾重视技术，所以技术学校很多。的确我们此行见过好几间。

晚上返回酒店经过董师傅 ，看门面还以为是连锁店，其实是个小店。门口有麻薯制作道具让人拍照。这个店开在酒店隔壁，注定有生意。我们走的时候买了好些东西。"少"今晚买了麻薯雪糕。董师傅有两家门市，这家只有一个人工作。没客人的时候现场制作麻薯，有人就停下来收钱。虽然店不大，但看网站有自己独家产品。他们也有凤梨酥，贵的420一盒8个（百分百凤梨馅），想试尝可以买单个50元（买一盒没有优惠的,这点倒是觉得他家挺老实，不吸引你多买），便宜的150一盒（20%凤梨馅，加了东瓜）。新鲜麻薯20元一个，还有很多旅游礼盒装。我们在最后一天的红顶谷创也买了凤梨酥，我在文末再介绍我吃过凤梨酥的味道。总括而言，台湾很多大小品牌凤梨酥，不出名的不一定不好吃，有散装大可以买一颗先尝尝。

另外，关于麻薯，其实就和糯米糍一样的东西，不是台湾独有的，为什么台湾就能发展成旅游产品，香港就只能在街头买到？
