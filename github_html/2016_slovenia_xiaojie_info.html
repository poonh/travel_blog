游记

恋恋不忘斯洛文尼亚 - 行程相关信息

(所有价格为当年)

日期：2016年5月

行程(详细行程在游记)：
D1:到达Bohinj Bistrica
D2:全天登山
D3:上午Bohinj漂流，下午去Lake Bled湖边随逛。
D4:骑车去Vintgar Gorge,宿Ljubljana
D5:返程

游玩：今次行程主要在Lake Bled和Lake Bohinj,我只介绍这两个地方。这两个湖都在阿尔卑斯山脚下，风景很漂亮。玩的主要是户外活动。七八月份这里的人超级多，住宿要提前几个月预定。

Lake Bled
湖上可以划船，游泳，stand-up paddle（下文详述）
湖外可以骑车(这里骑车线路很多，下文详述)参加水上活动，报团游览阿尔卑斯山，玩滑翔伞等。
Pr1mo tours是当地一家旅行社，我去年参加过，tripadvisor拿了很多奖，详见我去年斯洛文尼亚游记，极力推荐！这里可以租自行车。
3glav adventures
非常多的户外活动

Lake Bohinj
 这个湖在Bled以西约30公里，想登山的话这里是基地（登山游记详述）。也有大量水上活动，骑车骑马等等。每年5月底6月初还有野花节，旅游局安排很多活动的。
Bohinj旅游局
Bohinj Guest card(http://www.bohinj.si /en/about_bohinj/bohinj_guest_card):超级划算的卡，不自驾的游客10欧，天数不限，只要你在Bohinj就能用， 买的时候会帮你写上有效日期的，你说几天就几天。坐船免费（原价往返10.5欧），本地公交免费（都1.8欧一程，即Bistrica,Ribcev Laz,Stara Fuzina及Zlatorog这几个地方），去Bled，Ljubljana这些地方半票。Bohinj景区瀑布景点也免费（例如Savica Waterfall)，水上活动8折。我们花了10欧，换来等值约25欧的东西。这个卡可能宣传力度不太够，坐车坐船时发现很多人也没有。

公交
 这一带主要是Alpetour,也有AVTO,但不多。公交车都是几十人的大巴车。
这里公交车晚点还挺多的，晚5分钟10分钟很正常。
特别注意：这里公交车不报站，而且车上是没电子显示的。不知道什么时候下车的话，就上车时先跟司机打招呼，你买票时也要告诉他去哪的。另外自己用电子地图跟踪。虽然镇跟镇之间距离不远，从一个到另一个也就10分钟左右，但车次不多的，事前查好，或者公交站有时刻可以看。

水上活动
看Bohinj旅游局官网，很多选择，我们报的sports4fun,因为可以最少2个成团，其他要最少3个人。但当然这些团是拼团，有人拼的话一个人也行的。这些水上活动有些很新奇刺激，例如canyoning。我们今次玩的漂流rafting。

登山
这 边山上住宿很多，可以安排两天或多天线路，背包登山。这是我原来所想的，但山上住宿一般都6月底才开，直到9月底，所以这次就一天往返了。这里最高峰 triglav,2800度米，需要登山装备的。我原来考虑过，后来咨询过pr1motours老板，说外国人出过很多意外，我又没登山经验，就不考虑 了。如果你想爬但又没足够经验应付，最好找导游。
另外，这里有些登山线路需要头盔的，会有石头掉下来。登山公司有装备出租的。
登山公司参考：http://www.vertical-adventures.si/
这家公司看网页都是滑雪，但其实有不同季节登山导游团的。
这里登山有指示牌的，加上maps.me手机地图就够，不用旅游中心买地图的。maps me话说这里比google要强，每个指示牌的位置都有。
Stara Fuzina是很多登山线路起点，想早出晚归的话，住这里不错的，就是餐厅选择少些。
斯洛文尼亚全国山上的Hut: http://www.slovenia.info/en/Mountain-lodges.htm?koca_planinski_dom=0&lng=2
如果想登山看湖景又不想步行登山，Vogel有索道的，位于湖的南侧。

饮食
主食面包，这里特别喜欢荞麦，很多荞麦做的主食。

饮用水
这里自来水是阿尔卑斯山泉水，水质特好。

红酒
Cvicek,reforsk,teran，斯洛文尼亚自己的本地红酒，这里的就都是100毫升的，大概1欧多一杯。

天气
初春：我去年4月初去的，这个时候山上的雪还没完全化，有点冷，但能骑车。在农村骑得时候，四周都是雪山，非常强烈“阿尔卑斯山”的感觉。但这个时候不能登山。Bled的Vintgar Gorge也没开，要大概5月初。这个时候也不能玩水上活动。
初夏：今年5月底去，雪山还能看到一点点。缺少雪山，阿尔卑斯山感觉没那么强烈，但还很漂亮。这个时候可以玩水上活动，但登山的话，山上住宿还没开，只能一天往返。
盛夏：什么都可以玩，就是人多
9月：这个时候人少，水上活动都有，而且山上住宿也还开放。
冬天：只能滑雪。

语言
这里的人英语不错的，菜谱一般有斯洛文尼亚语，英语，德语和意大利语，有些甚至有俄语。
