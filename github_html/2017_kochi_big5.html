第一次一個人跑馬旅行-四國高知農村美食5天之旅 - 遊記

引言：日本西南部高知，位於四國。四國是日本最沒有被開發的旅遊區。主要是本國人旅遊，這裏充滿日本農村鄉土景色。想看看原汁原味的日本，而不是現代化商業化的她，四國是個好選擇。這裏由於人較少，公交不發達，想玩好最好自己開車。我不會開車，但去5天也能玩得很開心。

2017年2月16日週四
D1:東京-高知到達-波川溫泉酒店
題外話，早到羽田機場，想買杯熱茶，就連室溫的瓶裝茶也找不到....日本真是好奇怪，就喜歡涼的。話說近期事情多，我今次也沒多花時間做攻略，出門前也沒太興奮，不過到了機場的時候還是從心裏笑了出來的.....
飛機早到了十分鐘，我很順利的出來坐上了3:30的機場大巴，票價720，半小時到高知火車站。這個機場很小，領行李就只有兩個道。
上車前先在售票機買票，下車給司機票，小心不要弄丟。
（下圖：售票機及機場大巴及車票）



沿途嘛，就是一些農村景，現在冬天，都是黃黃的，也不太好看，不過天倒是很藍。

到站了，也是現場售票機買票到波川，260元。之後就上站臺等16:18的車，20多分鐘到。等的時候發現麪包超人及其他可愛的火車。P.S.火車站有些小商店可以買高知特產，還有很多面包超人呢。



（下幾圖：高知火車站及站臺隨拍，高知市其實並不漂亮）



這裏的火車其實看上去有點國內綠皮車的感覺。一個設計得很不好的地方是，梯楷特別高，不方便輪椅和嬰兒車。


順利到了波川，看地圖到之前會經過仁澱川的，這是日本最乾淨的河流，一定要一看。由於酒店在個小山坡上，在火車上就能看到。
（下圖：火車上拍的仁澱川）

（下圖：一下火車就能看到酒店）

這是一個小火車站，下車時會有人手收票的。站臺有售票機，小車站沒有人手售票的。
（下圖:波川車站）

雖然酒店近在眼前，但由於在路的另一面要繞一下才能到。我看電子地圖，經車路上去，但其實是有條行人小路可以上的，就在車路的旁邊。當你從火車站去酒店的路上，走到下圖一個小店的時候（沿途只有一個小店），左邊就是車路，右邊就是行人小路。我下來的時候才發現這條路。但有點凹凸不平，而且挺陡，雖然只有最多兩三百米，有箱子的會不太好走。

話說，我由於想看仁澱川美景，之前考慮過這家酒店和明天去玩工藝的土佐和紙工芸村的酒店。原來看照片覺得後者更漂亮一點，不過現場看還是前者好。土佐和紙工芸村酒店就在大馬路邊上，我現在住的這家不是，而且房間全部有陽臺，全部朝河，環境很好。
（下圖：酒店）

（下幾圖：酒店陽臺拍攝的，對面是伊野）




到了大概5點，約了7:30喫飯，放下行李馬上泡溫泉，先換上浴衣。現在是淡季，溫泉的人很少。室內的有一個大浴池，有按摩池，也有一個叫“寢湯”，是讓你躺下來，有石制枕頭的，很是享受。這裏也有露天風呂。之前在和歌山泡過溫泉，不過是夏天，露天沒有太大感覺。當時溫度大概十度左右，太陽已經下山。我去泡露天風呂，頭是涼的，頭部以下是熱的，覺得特別爽。在室外就是少了室內的悶。要是天氣好的話，晚上泡還可以看星星呢。我當時在外面覺得泡了最多十多二十分鐘吧，起來時覺得非常頭暈。由於有了上次空腹喝酒起來馬上暈倒的經驗，我就知道自己不能勉強，及時蹲下來就緩過來了。後來慢慢起來就沒事。各位童鞋注意，如果你起身覺得頭暈必須要及時蹲下，不然真的會暈倒的。
我後來分析了一下自己覺得頭暈的原因。（本人非生物專業）我覺得是因爲我身體是熱的，血管擴張，但頭是冷的，血管收縮。當我起來時，出於保護機制，血管不能馬上收縮，於是頭部的血就額外的往下身輸送，造成頭部一下子供血過少，就頭暈了。

好了，溫泉泡完，回去洗澡喫飯。和上次在和歌山一樣，穿着浴袍去喫，看見這麼多美食，坐的位置還要看到河景，真是非常享受。我強烈推薦，如果你想感受日本風情，想好好享受，第一次去日本住一次溫泉酒店，喫一次懷石料理。

我喫的是伊勢海老懷石料理（海老日語是蝦的意思），總共10個菜，另外加了一個高知土產燒鯖魚壽司。這裏做這個壽司和其他地方不一樣，米飯裏面加了芝麻和蔥，喫起來特別香呢。
至於懷石料理，做的很精緻全部都很好喫。今天我就喫到傳說的鰹魚，沾的橙汁醬油，夾着蒜片和蔥，第一次這樣喫壽司呢。

（下圖：這個米飯是現場蒸的,蔬菜是蕨菜）

雖然很豐富，還加了一個壽司，但我除了蝦蒸飯外還是喫完了的，強！
回去房間，想喝熱水，發現暖水壺，以爲是用來保溫的，但原來裏面全部是冰....不過也很欣賞他們的貼心，冰水熱水都提供了。房間內還有一些按摩保健用品可以試用，合適就去商店買。另外，他們竟然還有脂肪測量器！我試過，準確度有懷疑的說.....



總結：第二次住溫泉酒店，依然覺得是非常享受。真的，去過玩過喫過的地方，就住溫泉酒店喫懷石料理最享受的了。這家酒店還有陽臺，環境很棒！！！

2017年2月17日週五D2: 土佐和紙工芸村玩紡織和造紙（公交站：巖村）-高知市區-土佐山民宿
造紙的地方看開首網址。
早上喫完早飯，坐8:28從波川到伊野的車，到站是河的另一端，要過橋的。如果你想走路過去伊野是可行的，給汽車用的橋有行人路的，路程大概兩三公里。
話說，酒店的自助早餐不過不失，不過早餐也有鰹魚片，我又多喫了幾片。退房的時候，是昨天我入住時接待我的mm幫我退房,當時我在波川站沒看到售票機，問她如何買票，她當時告訴我要在伊野買。不過後來她發現錯誤，今天告訴我了，而且還用中文打出來，其實站臺有售票機的。這個mm的貼心令我很感動呢。
三分鐘火車就到伊野，票價160。下車後往南走一段，走到Lawson便利店的時候往左，就可以到伊野前公交站。我一開始走錯，走到電車那裏去，其實不到那麼遠的。另外，注意公交站站牌是不顯示行車方向的，我一開始還去了錯的方向，後來有司機指點纔去了正確方向。應該行車方向是往西的，往東的是去高知的。

下車的地方就是酒店門前，並沒有站牌的，不過對面回去的車站有站牌。這裏就是一條馬路，一面是山，另一面是仁澱川。現在還是冬天，景色不是特別好。
看他們官網提示，造紙不用預約，他們一天都有人在那裏，但有時候會有團體包場的，所以最好預約。另外，紡織一定要預約的，不預約的話，老師可能不在的。
下幾圖，造紙紡織的酒店及附近環境。



我9:20就到，到了前臺後，他們就先帶我去造紙。造紙的老師只會一些英語單詞，不過跟着造就行了，當時我一個學生。老師教我造紙，另一個mm幫我拍照，貴賓享受。
做明信片的話，600元8張，是最便宜的。另外可以造紙扇，價格忘記，官網有寫。我當時玩了大概40分鐘。

之後用剪刀把真的花花草草放上去，最後等30分鐘弄乾。幹了之後顏色變淡的。

在靜待30分鐘的時間，我就去玩紡織了。又是我一個人，專人服務。這位老師會英語的，她媽媽會這個，家裏有縫紉機，自小就會，她還會用紡織機做和服呢。她說大概需要十天。
這裏的線都是絲綢做的，我做了30cm的，2100元，60cm的3000，如果要造圍巾的話，需要120cm。我花了一小時，老師說我學的很快，技巧很好~
上幾張那裏的圖，那裏很傳統工作坊的感覺，很漂亮，還有地方休息，可以喝茶和喫糖，看看河景。要是有幾個朋友和家庭，非常愜意。

下圖：這裏喝茶環境非常好，不另外收費的。
你可以選擇顏色，編的時候也可以隨意變換。我挑了粉紅系列的。
線是上下兩層的，左腳和右腳獨立操作。開始你左腳是往下踩的，兩層線分開，此時你把線先從左邊放到右邊，右腳踩到和左腳平衡，這樣兩層線就合一了。這個時候拍兩下剛加上去的線，把它們拍嚴實，之後右腳往下踩到最低，兩層線又分開，再把線從右邊放到左邊，動作完成。老師當時幫我拍了視頻的呢。

喫過午飯，隨便逛逛，之後就去火車站看麪包超人，我買了擦手的毛巾給老媽和貝恩，約650一條。

之後我在商業街的K's cafe坐了一會，這裏環境不錯的說 。
最後坐上末班車去土佐山Masa及Tomoko家裏。下車的時候那裏太黑，我找不到他們的房子，其實就只有一兩百米的距離，後來Tomoko打電話給我，Masa出來接我了。這晚第一次喫日式家庭晚餐。他們是每人一份的，像西餐那樣。
Masa和Tomoko一家五口，三個兒子，還有一個女兒下個月出生。最大兒子十四歲，最小的5歲，那個還在上幼兒園的孩子能自己獨立喫飯，不用人喂。洗澡的時候大哥哥和小弟弟一起，不用父母操心。第二天早上起來時，看見大兒子在廚房切洋蔥炒雞蛋做早飯給自己喫，當時覺得，國內十四歲的孩子，大部分可能連碗也沒洗過，他就會做飯，還能照顧弟弟了。這不單只是家庭教育，在日本的學校，學生每頓午飯都要幫忙，分碗筷，飯菜，收拾等等。他們是訓練有素的了，這就叫“素質教育”吧。
飯後和Masa聊天，原來他是個漫畫家。幾年前一家從東京附近搬過來。他們住的房子當時沒有人住，好幾十年了，有點破。剛搬來時，天花地板也沒弄好，一半自己弄，另一半找人來用。舊房子還要找人來做防震加固。這種老房子，客廳，浴室，睡房都不在同一個樓，就在隔壁，但下雨還是有影響的。這裏大概治安太好，他們門都沒有鎖的。不是有鎖不上，是直接沒有安裝鎖。
我也和Tomoko聊天，日本女人懷孕飲食沒那麼多禁忌，咖啡照樣喝，生魚片照樣喫。日本老人好像外國一樣，也是如果住在孩子附近，纔會幫忙帶孫兒的，不然不會。她前幾年，已經是三個孩子的母親，還一個人去香港旅行呢。
冬天的時候，他們用的舊式暖爐，要木頭的（但當然這裏溫度不低，不用好像我去斯洛文尼亞阿爾卑斯山那裏的人那樣，要大量伐木）。我看Masa就帶着兒子去伐木。我感覺他們一家從大城市搬到鄉村，從新建立自己的家很不容易。同時我也感覺日本給人更大的生存空間，更大追求自己理想的空間。他們夫妻都是家裏工作，收入該不太多。但是有自己的理想，在城市經濟成本大，可以搬到小地方郊區，安安靜靜生活，做自己喜歡做的事情。而在國內呢？獨生子女有很大的養父母壓力，即使沒有，要放下世俗的眼光去做掙不了錢的事情不容易。即使你不介意別人的眼光搬到農村去，拖家帶口的，誰又願意自己的孩子接受農村教育？外在的枷鎖確實令人失去追求自己的機會。
Masa跟我分享他的漫畫，他其實沒有學過，不過媽媽也是設計師，自小受到薰陶吧。

另外，由於Masa是搞藝術的，會拍照，在他們家時，他幫我拍了很多生活照呢。
現在高科技發達，也可以直接畫在電腦上，用的下圖工具，專門電腦畫畫用的。
既然是漫畫家，家裏也是充滿特色，所有他們需要跟客人說明的事情，都會給上配圖，很可愛的說。
Tomoko還跟我說，他們有家咖啡店，就在房子十米以外的地方。一家很溫馨的小店，原來是個車房，現在變成能看書的咖啡店。週二下午他們經營，週六下午另外一個女孩經營。我當時心裏想，一週只經營半天如何掙錢？聽Tomoko說，客人也不太多，多數是在高知市遊覽的客人，開車上來玩的。週六下午那天我去了咖啡店，和女孩聊天，別有一番感覺，下文詳談。
