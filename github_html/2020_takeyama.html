3.广岛百名山岳山 - 游记

日期：5月1日（周五）
地点：广岛县府中市岳山（广岛县百名山）

前言：这次登山行程其实换来换去，一开始想去广岛县第一峰恐罗汉山，但去那要走高速，不走要接近三小时车程，后来看到三仓岳很吸引，但有一定危险性。我们之前一周爬了白木山，花了不少体力，有人还没恢复，所以今周玩个简单的吧。我看到世罗高原那边有冰淇淋店，心动想去吃，碰巧隔壁有男女鹿山，就是太简单了点，大概和西条第一名山镜山差不多，后来不知道哪看到附近的岳山，而且是广岛县百名山，看官网一小时就能登顶，攀升才300多米，我就决定去这里了（但事实证明不是那么简单的）。

行程开始：
早上9点多出发，到登山口前去世罗町的超市上厕所，具体地点我在地图标了，有很多的。我们到斗升登山口大概11点半（其实从西条开车1个半小时就到）。
就在路边的空地停车就行

另一角度看，左边是停车场，照片差不多正中心位置是登山口前的牌子（下面有）

我们正式起步前我要先记洗碗王子及公主一功。洗碗公主今次全程为我们背着移动 ティッシュ(tissue)，方便我们随时用。这个新颖的袋子设计，叫它做“先端纸巾袋”。
继承上次的付出，洗碗王子今次继续为我们全程背垃圾袋
我们登山了。起点有填手帐的地方，写的一手好字的洗碗王子负责写。
起点拍照留念。

路一开始就是分岔口，我们走的短的那条上去。正式踏在路上，我们发现原来这是野山，地上全是树叶，当时真庆幸买了登山杖，不然太滑了。路上一般就是下面那样，不太难的。
偶尔也遇到过绳子，有一段很抖。不过原来以为我们走另一条路下来，那条路因为长点，会平坦点，但其实不是......后面再说
路上的标记
 在没啥惊险之下，1小时15分之后我们登顶了！这里不像白木山360度景色，山顶没啥看的，我们就吃吃东西聊聊天，洗碗公主和onion学长还从手机翻出自己的老照片给我们看他们当年瘦的样子.....
这是山顶，我们在这里吃饭
 登顶留念
 
好了，想象不到的挑战在前头等着我们。上山的路上有点岩石，不过没有下山那条路好看。下山时有时候路不很清晰，不过看树上的丝带标记就行。
下山遇到这个岩石，我们跨过它的后面下去
路有点窄的，不过我们都能穿过
之后就很多扶绳子的路，都已经忘记了几次了
这段该是最陡的了。曾经有次另一侧就是会翻下去那种.....洗碗公主一直要洗碗王子扶着。小z也一直扶我，或者在后面拉我
路上其实还有很多大岩石的，有些是特别景点，更多像这种
后来我们走到一个岩石平台，那里看风景很棒！
 
经历过一段艰辛后，我们终于完成了绳索路，不过最后下山那段感觉也走了很久，有古木苍天的森林，也有野草路，也有要跨越小河流，很漂亮呢
好了，终于下来了，拍照留念先~

这个山大家都觉得很惊险，但又很好玩，就来一次吧，下次不去三仓岳了。花了两个半小时下来，由于有上周的训练，这次大家都觉得没什么。下山后事件还没完，时间还够去世罗冰淇淋店的（开车时快四点半了，店五点关门）。
快到冰淇淋店那条路，啥都没有，真的很难想象在这荒凉郊外会有冰淇淋店，不过地图是没错的，确实有！ 这个小店好漂亮呢，单球好像370，双球400多，我还买了冰咖啡。这里的酸奶也特别好喝，无添加的，190一杯。 
 
这里的厕所特别可爱，放了牛牛
这里还有马场，放羊及斑点狗
图偏左边有只羊在悬崖边走路

吃完刚好五点，人家也要关门了，我们就回去继续吃火锅晚餐了。第二天起来，大家都完全不痛，证明上次白木山后大家水平都提升，训练出来了。我们两周后就挑战广岛县第一峰 - 恐罗汉山。
