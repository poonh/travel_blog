2.廣島百名山嶽山 - 相關行程信息

我們的行程地圖(行車線路取自navitime)(無法顯示點擊此)
（藍:景點    綠:喫飯   黃:廁所  紫:停車場)）
徒步線路走的是紅色線路（東側要放大看，線路取自yamap），去程1.4公里，1小時15分到頂。返程2.3公里，花了2個半小時（純屬技術問題）。



提示一下，這山沒有正規停車場，具體地點看地圖紫色，在登山口那條路再往前走幾十米一塊可以隨便停車的空地，那地還是斜坡，當時難停死我了。那塊地不大，我覺得應該停不了十輛車的。這裏附近有公交車站的，下車後走一公里可以到斗升登山口，即我們的登山口。

嶽山一帶旅遊信息參考：
府中町旅遊官網（嶽山所在地）
世羅市旅遊官網（我們喫冰淇淋的地方）
嶽山登山者提供的登山線路及記錄參考：yamap的嶽山

政府旅遊報告：
2023年《第2期世羅町観光振興基本計畫》(商工観光課)

嶽山是廣島百名山，很好聽吧，日本有“全國百名山”(還有全國兩百名山，三百名山，四百名山，花百名山）。全國百名山廣島縣沒有在內的（中國地區只有大山）。我們去過其他的七國見山，火山，白木山都是廣島百名山。我沒寫過遊記位於東廣島的龍王山也是。這山比西條第一名山鏡山還要簡單，登頂可以俯視西條景色。

嶽山簡介：
這個山的賣點是岩石，的確挺好看，我們中間還在岩石縫中穿過。去之前我不知道這是半野山，完全沒樓梯的，走這個山有登山杖會好很多，不然容易滑。路上很多樹葉，剛下完雨不要去，因爲沒樓梯下完雨太滑了。上山有些地方比較陡，要拉繩子，下山那段路繩子更多，爬完這個山後大家有點怕，覺得危險（我還可以接受），就決定下次不去三倉嶽，去恐羅漢了。
世羅高原農場就在附近，有時間可以去玩，冰淇淋店有冰淇淋，黃油製作，騎馬體驗。不過當時因爲疫情沒開，但還好冰淇淋店在等我們呢。

世羅高原ファームランド ジェラート工房（世羅冰淇淋店）
