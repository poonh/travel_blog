戀戀不忘斯洛文尼亞 - 行程相關信息

(所有價格爲當年)

日期：2016年5月

行程(詳細行程在遊記)：
D1:到達Bohinj Bistrica
D2:全天登山
D3:上午Bohinj漂流，下午去Lake Bled湖邊隨逛。
D4:騎車去Vintgar Gorge,宿Ljubljana
D5:返程

遊玩：今次行程主要在Lake Bled和Lake Bohinj,我只介紹這兩個地方。這兩個湖都在阿爾卑斯山腳下，風景很漂亮。玩的主要是戶外活動。七八月份這裏的人超級多，住宿要提前幾個月預定。

Lake Bled
湖上可以划船，游泳，stand-up paddle（下文詳述）
湖外可以騎車(這裏騎車線路很多，下文詳述)參加水上活動，報團遊覽阿爾卑斯山，玩滑翔傘等。
Pr1mo tours是當地一家旅行社，我去年參加過，tripadvisor拿了很多獎，詳見我去年斯洛文尼亞遊記，極力推薦！這裏可以租自行車。
3glav adventures
非常多的戶外活動

Lake Bohinj
 這個湖在Bled以西約30公里，想登山的話這裏是基地（登山遊記詳述）。也有大量水上活動，騎車騎馬等等。每年5月底6月初還有野花節，旅遊局安排很多活動的。
Bohinj旅遊局
Bohinj Guest card(http://www.bohinj.si /en/about_bohinj/bohinj_guest_card):超級划算的卡，不自駕的遊客10歐，天數不限，只要你在Bohinj就能用， 買的時候會幫你寫上有效日期的，你說幾天就幾天。坐船免費（原價往返10.5歐），本地公交免費（都1.8歐一程，即Bistrica,Ribcev Laz,Stara Fuzina及Zlatorog這幾個地方），去Bled，Ljubljana這些地方半票。Bohinj景區瀑布景點也免費（例如Savica Waterfall)，水上活動8折。我們花了10歐，換來等值約25歐的東西。這個卡可能宣傳力度不太夠，坐車坐船時發現很多人也沒有。

公交
 這一帶主要是Alpetour,也有AVTO,但不多。公交車都是幾十人的大巴車。
這裏公交車晚點還挺多的，晚5分鐘10分鐘很正常。
特別注意：這裏公交車不報站，而且車上是沒電子顯示的。不知道什麼時候下車的話，就上車時先跟司機打招呼，你買票時也要告訴他去哪的。另外自己用電子地圖跟蹤。雖然鎮跟鎮之間距離不遠，從一個到另一個也就10分鐘左右，但車次不多的，事前查好，或者公交站有時刻可以看。

水上活動
看Bohinj旅遊局官網，很多選擇，我們報的sports4fun,因爲可以最少2個成團，其他要最少3個人。但當然這些團是拼團，有人拼的話一個人也行的。這些水上活動有些很新奇刺激，例如canyoning。我們今次玩的漂流rafting。

登山
這 邊山上住宿很多，可以安排兩天或多天線路，揹包登山。這是我原來所想的，但山上住宿一般都6月底纔開，直到9月底，所以這次就一天往返了。這裏最高峯 triglav,2800度米，需要登山裝備的。我原來考慮過，後來諮詢過pr1motours老闆，說外國人出過很多意外，我又沒登山經驗，就不考慮 了。如果你想爬但又沒足夠經驗應付，最好找導遊。
另外，這裏有些登山線路需要頭盔的，會有石頭掉下來。登山公司有裝備出租的。
登山公司參考：http://www.vertical-adventures.si/
這家公司看網頁都是滑雪，但其實有不同季節登山導遊團的。
這裏登山有指示牌的，加上maps.me手機地圖就夠，不用旅遊中心買地圖的。maps me話說這裏比google要強，每個指示牌的位置都有。
Stara Fuzina是很多登山線路起點，想早出晚歸的話，住這裏不錯的，就是餐廳選擇少些。
斯洛文尼亞全國山上的Hut: http://www.slovenia.info/en/Mountain-lodges.htm?koca_planinski_dom=0&lng=2
如果想登山看湖景又不想步行登山，Vogel有索道的，位於湖的南側。

飲食
主食麪包，這裏特別喜歡蕎麥，很多蕎麥做的主食。

飲用水
這裏自來水是阿爾卑斯山泉水，水質特好。

紅酒
Cvicek,reforsk,teran，斯洛文尼亞自己的本地紅酒，這裏的就都是100毫升的，大概1歐多一杯。

天氣
初春：我去年4月初去的，這個時候山上的雪還沒完全化，有點冷，但能騎車。在農村騎得時候，四周都是雪山，非常強烈“阿爾卑斯山”的感覺。但這個時候不能登山。Bled的Vintgar Gorge也沒開，要大概5月初。這個時候也不能玩水上活動。
初夏：今年5月底去，雪山還能看到一點點。缺少雪山，阿爾卑斯山感覺沒那麼強烈，但還很漂亮。這個時候可以玩水上活動，但登山的話，山上住宿還沒開，只能一天往返。
盛夏：什麼都可以玩，就是人多
9月：這個時候人少，水上活動都有，而且山上住宿也還開放。
冬天：只能滑雪。

語言
這裏的人英語不錯的，菜譜一般有斯洛文尼亞語，英語，德語和意大利語，有些甚至有俄語。
