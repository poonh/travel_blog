3.四万十寻花之旅 - 游记

前言：这是为了去四万十寻花的一次旅程，她去了四万十差不多一年了，她没去之前就说要去找她的。

今次我准备了40多分钟视频，记录了我们行程点滴，里面还有一些剧本成分。

全程开支:
D1:吃饭：~2200 (住宿省略)
D2:西内家一泊三食+岸边钓鱼浮潜套餐+单程私人游艇 ~18000
D3:晚饭：~1000    船票：1350     冰淇淋：50(宿花宅没花钱)
D4:早餐：550    午餐：~300（便利店）    温泉：980     晚餐:~1500
全程油费及高速：每人5500
杂费：每人1200
合共不到35000。
另外，我们两名司机买了开车保险，同样条款Lawson比7-11便宜。

行程开始：

D1:9月22日周三

早上七点出发 ，原计划去喀斯特（カルスト），不过中途下雨，所以就直接去梼原了.路上去了来岛海峡大桥拍照，

梼原町中心只有一个小超市，菜挺贵，选择也不多，可能交通不便原因。这个地方给我感觉是挺有钱。中心地段像个古镇。最棒的是有几个日本建筑大师隈研吾(Kuma Kengo)的建筑。我们去到的时候学生放学，骑在自行车上看到我们都打招呼，对这个地方感觉太好了。

豪宅从梼原町中心去开车约20多分钟，中间一小段是单程路，晚上开路上会很黑。 房子实际上是比我想像中更豪的。我们先看外表。

 我们今天原计划从梼原直接去宿毛坐船去鵜来島，但后来uguru box老板说可以来接我们，这样我们就多了一两个小时的时间，临时加插行程去高茂岬了。

 比起直接去宿毛，去高茂岬需要绕一下去个半岛。半岛是下坡的，最后8公里都是单程路，当时我时速大概20公里，还好没遇到对头车。半岛上开车时间大概不到1小时（可能40分钟左右，具体我忘了）。高茂岬游步道大概200米，很短的。连上拍照时间我们大概逗留了半小时。

鵜来島这个只有20多个村民的地方，全部都是老人。现在住的大概是最后一批了。岛上一点日本的气息都没有，连自动贩卖机都没有。唯一能花钱的地方，就是每天码头售船票时会同时售冰淇淋。岛上好些猫，都很乖。房子屋顶都是蓝色，后来问原因，是因为只有蓝色的漆。虽然岛很小，但也有山，有战迹可以看。


我现在带你们看看鵜来島。上面的路都很窄的，房子楼底很低。

我特别喜欢这个新鲜做的 炙り生鱼片，一上来已经闻到烟熏的味道。吃下去烟熏味很重，吃过这个之后再也不想吃寿司店的 炙り寿司了。盘子里面的装饰不是石头，而是陶艺制品，西内自己做的。另外配菜洋葱吃不完第二天做炒饭，所以西内配菜不用一般的白萝卜丝。

 做好饭，西内来和我们一起吃，一同聊天。他说自己只会做几个客人喜欢吃的菜，要是客人多待几天就会重复吃到一样的菜。另外，我们原来订的是海钓，他建议我们可以玩岸边钓鱼+浮潜套餐，说我们玩潜水更开心。事后我完全赞同。而且海钓他说平均每个人能钓到三四条，我们5个人钓那么多也吃不完。


来个自拍。

我们一开始在岸边练习。西内说，以前问日本人会不会游泳，都很谦虚的说不会，其实是会。我们5个人4个不会，当我们说不会的时候他没想到我们那么的不会。哎,这就是中日教育的差别。我们在练习的时候，onion学长特别不会，有次他趴下来脸朝海里，我们怎么叫他都没反应，当时还以为他有事，之后HY来了一个英雄救美...

潜水中途，raku老师的泳镜突然不能呼吸。遇到这种情况拍一下管子就可以。她也游到累了，先上船休息，当时HY拖她上去的。我也想去帮忙，不过她们实在太快...

第一次浮潜，真是超级好玩，和看电视完全不一样。那种兴奋感真是难以形容。小鱼都是一群一群的，有时候在你肚子下游过，有时候在你身边擦过。我总是很坏的想去抓它们。我们前后去了两个地方，第一个地方珊瑚多，小鱼多。第二个地方较深，看到很多大鱼，还有乌龟。我们潜水的时候，西内就打鱼给我们做午饭吃，我看他有时候潜到六七米深，他的肺活量肯定很好。 

我在海上的时候，突然听到洗碗公主抱着一条中等大小的鱼来求救，我当时还以为她那么厉害可以徒手抓鱼。原来是西内打到给她的，要是能拍照这张肯定很经典。她想找我帮忙，但鱼最后在我手里逃脱了...

经441看四万十川，直到最后上高速，虽然路不是很长，不过我觉得我们全程走了有两小时，最后我们快5点才到利乐温泉。

 raku老师给我们介绍利乐温泉。 去年我们爱媛之旅去白猪瀑布的路上就看到它。因为特殊原因才会注意到并且记住它，所以今年就来了。


这里果然是西日本最大级，实在是对得起这个名有余。我们看看酒店内部，很高级的。


温泉不能拍照，不过我记下来有什么玩了。玩的是我去过的温泉最多的。

1.樽风吕

温泉内所有池的水都去铁，唯有樽风吕没有。这里有三个樽，其中两个是砥部焼。砥部是松山南面的一个町，他们第一次做那么大的浴缸。浴缸因为有铁所以生锈。另外一个樽是高野槇。这是一种植物，其液体可以治疗牙周病。

2.山洞里面泡温泉，有特殊灯光

3.大露天风吕

4.带有玩具鸭子的汤

5.水中步道，有石头按摩脚底

6.冰水浴

7.寝汤，可以全身按摩

8.釜桑拿，温度特别高，里面有淋浴

9.普通桑拿 ，带电视

10.按摩池，可以按摩脚底，腰部及肩按摩，水力特别大

11.室内大浴池

我们旅程最后去了利乐温泉，实在一洗路上几天的疲劳。
