上一章

前言：Ahr Valley位于波恩西南约20公里，不是那种国际级很出名的旅游景点。这里有条河，叫River Ahr,属于莱茵河支流，四周很多山，所以叫Ahr Valley。这条河比较窄，估计十多二十米宽，水位最多只及脚。不能玩水上活动的。这里是个很大的葡萄园，可以泡温泉，骑车，徒步，尝美酒。这里没有大 城市，都是小镇。游客不会超级多，很安静，人很友善，又可以做运动，是个短途旅游好地方。由于我们三天首先游泳，之后骑车，最后徒步，很好的被誉为我们去 玩“伪铁三”。
话说五月的时候去了海德堡以西几十公里的German Weinstrasse徒步(Weinstrasse= wine street,这次徒步其实有另一段毕生难忘的故事，不过这里不多说了），查资料时无意中发现Red Wine Hiking Trail,再接着查就找到Ahr Valley这个地方了。当时看到很想去，不过没人一起，最后找到两个mm陪我，三个都玩的很开心。
另外，这个游记最后一天是海德堡周边骑车，其实不是和上面同一个旅程不过我放在一起。

德国葡萄园其实很多，整体一览表参考： http://www.germany.travel/en/towns-cities-culture/gemuetlichkeit/wine-country/winegrowing-regions/winegrowing-regions.html

景点介绍：
Ahr Valley有很多小镇，就是80公里骑车线路Ahr Cycle Route所经过的。这个线路从东北离波恩最近的小镇Remagen开始（这也是Ahr Valley最大的小镇，在莱茵河畔），直到西端Blankenheim，这个小镇就是Ahr River的尽头，有泉水。这个线路中间经过很多小镇，没感受过德国小镇的话这条路线可以让你好好感受，镇和镇之间是农田。虽然之前我去过德国小镇，但却 没有去过这里那么小的。小的连商店也没有，西端Ahrbrueck以西的小镇都没火车，公交车一天就一两趟。我第一次去德国这种地方。
8月底9月初葡萄收成的时候这里有活动，不过我们去这几天碰巧没碰上。

Ahr Cycle Route具体经过这些小镇(从东到西)：
Remagen-Bad Neuenahr-Ahrweiler-Dernau-Mayschloss-Altenahr-Ahrbrueck-Hoenninga-Liers-Insul-Schuld-Fuchscofen-Antweiler-Muesch-Ahrdorf-Ahrhuette-Blankenheim
由于这里是葡萄园，酒肯定不少，这里的酒可是获奖的呢。这里红酒较出名，有兴趣可以去卖酒的地方让人介绍。价格也不贵，很多不到10欧就有一瓶。我自己比较喜欢有汽的白酒Secco，带点葡萄甜味。这里还有威士忌，是附近一个山生产的，叫Eifel Whisky。

下图是Ahr cycle route能坐公交车的地图，取自guest card的pdf文件(此卡下文详述）里的免费公交地图再修改。公交车能到差不多Ahr cycle route的尽头，但Ahrbrueck之后就没火车了，公交车基本就一天一两趟，骑车我们骑到Liers,之后返程骑到Ahrbrueck坐火车的。

几个主要小镇介绍：
这几个地方都是住宿多一些的,但住Bad Neuenahr或Ahrweiler有Guest card送，凭卡坐本地公交车/火车免费，一些地方会打折。

Guest card:https://www.ahrtal.de/service/gaestebeitrag-in-bad-neuenahr-ahrweiler/
Remagen:Ahr Valley最大小镇，交通枢纽，去Ahr Valley其他小镇都从这里倒车，一般在5号站台发车，北上就是波恩。此镇位于莱茵河畔，我们返程时在火车站附近的河边喝了令我这次旅程完美的冰咖啡 (Eiskaffee)，一边欣赏对岸小镇美景，真是好像一幅画那么美。
Bad Neuenahr:和下面的Ahrweiler连在一起的，两个火车站相隔大概3公里，车程2分钟。这是个温泉小镇。
Ahrweiler:这里卖酒较多，据说老城漂亮，不过我们当时时间不够没有去，就骑车时在城门拍个照。这里是Red Wine Hiking Trail的起点。
Altenahr:一个漂亮小镇，Red Wine Hikng Trail终点。这里以后就是农村了。我们骑车那天在这吃午饭的。
除了Remagen,Bad Neuenahr和Ahrweiler游人多些外，其他小镇人都很小，商店不开比开的要多，不知道是不是周末才会开。

2016年8月25日周四:全天骑车Ahr Cycle Route
之 前查资料知道这条路不太难，有点坡，不过还是很没追求的租了电动车。Seta的电动车真不错，电池可以撑200公里，我们Ahr Cycle Route往返不成问题，但当然我们骑不到那么远。事后我觉得用电动车真爽，以前遇到上坡就要停下来推车，电动的话一上坡就开电源，轻轻松松就上去了，完 全不用停下来。整条路大部分是泊油路，小部分有小石子，一般都是专门自行车路，标记尚算清晰，但还是最好有电子地图辅助。这段路经常看到有人骑车，但人不 会多的构成阻碍。可能这段路太简单，遇到的骑友都是老年人居多。

2016年8月26日周五：Bad Neuenahr-Remagen沿河徒步

今天的路线可以参考上面Ahr Cycle route,又或者看着电子地图走就可以了。
开 始那段就是昨天大概两旁有高树的大公园，很舒服又不晒，那条路很适合跑步的说。后来我们上了葡萄园逛逛，之后再下来回到河边。看到苹果树还摘来吃，不过这 是野苹果，没有施肥，比较涩。后来走了一段路很窄，很多矮的草，中间偶尔有人遛狗，还看见狗在河里游泳呢，这条路还经过一个动物园。再走到后面我们又回到 自行车路了，这里人明显较多。今天走的这条路和昨天后面的农村路一样，大概秋天来看红叶最美，但现在绿油油一片的也很漂亮。

快12点时还有好几公里才到河流交汇处Kripp,我们要赶着坐12:50从Remagen发车的火车回去Bad Neuenahr，所以就改变行程，在Kripp Lederfabrik那里坐公交车去火车站了，还好当时眼前就有车，我们加快脚步赶过去。
公 交车来又见识了，这个小镇的公交车，原来不是专门的公交车，就是普通的20座小巴车，根本没公交车标志，就是前面用A4纸帖了个车的路线，车上也没有电子 显示的。虽然这里是个大镇但车上只有我们三人，沿途也经过了几个站也没有人上车。我不知道这里如何买票，车站没售票机，司机看他也没有零钱可以找，我们是 用guest card不用买票。
灰姑娘想在莱茵河拍照，我们就在车站前一站Kreissparkasse下车了。这里去河边就两百米，有轮渡可以坐，还有河边餐厅，环境很好。当时拍照只有几分钟，不过后来我们又改变了行程，回去Bad Neuenahr午饭后又回来河边餐厅喝冰咖啡了。
（下图：Remagen公交车）

回去Bad Neuenahr我们去吃意大利菜(Ristorante Milano da Gianni)，果然比德国菜精致好多，虽然分量不大。这是我们最后一顿饭，干杯一下。三个菜+饮料及一份甜品，埋单总共53欧。






:
吃饭对面的地方就是卖酒的，品种好多好多，还有德国威士忌，我第一次知道。店员很热情的招待我们，灰姑娘买了一瓶secco,还给包装得很漂亮。
（下图：德国威士忌）

（下图：灰姑娘买的Secco,8.9欧，他家葡萄园离Bad Neuenahr约3公里）

吃 完饭，我们坐3点的车又去Remagen了。原来应该是3：30的，但因为坐这趟车的话，倒车时间太短，我怕万一晚点了接不上下一趟会很麻烦，刚才在那看 莱茵河那边景色很好，就坐早一趟车去喝冰咖啡吧。我每次跑完步都要喝的冰咖啡，昨天一直想喝又没喝到，最后在这个景色如画的莱茵河畔喝到，而且是我行程最 后一站，真是令我行程完美了。
（下图：喝冰咖啡的地方面对对面风景如画的小镇）

 （下两图：在这里喝的冰咖啡）

（下两图：莱茵河随拍）


喝完我们就坐火车回去海德堡结束我们Ahr Valley三天两夜的行程了。这几天天气特好，晴天的几乎没云。这个地方虽然不出名，不过我们玩的超级开心，而且人很友善，各处服务态度都很好。我们都是抱着做运动的心态而来。对于我们这种没有什么文化的人，去旅行做做运动更容易投入，更容易玩的开心。

2016年7月16日：海德堡-Neckargemund-Hirschhorn骑车
具体路线看开首的gpx地图。
这是之前的一次骑行，虽然风景没Ahr Valley漂亮但也很不错的。基本全程都是沿河骑车，路线很好找，标记也清晰，但还是最好有电子地图。
话 说海德堡这个市政府的公用自行车Next bike租用有点不太好用。站点是很多，说的是每个站点都有个电子显示屏，可以在那里出租，但我们去的那个有问题，一直不能连网，最后是用手机租的。我们 租了两辆，但他们系统有问题，只收了我一辆的钱，但我觉得钱能少收就能多收的。
（下两图：Next Bike）


这段路大部分是自行车路，有靠近马路的，有远离的，有路段现在开了花很漂亮，靠近Neckargemund那边有城堡，中间有野果树，也有很大的玉米田。海 德堡到Hirschhorn约22公里，中间有些坡，骑了约3小时到（比我跑步还慢），有能力可以继续往东骑到其他小镇，之后坐火车回来。这段路也是很适 合看红叶，但盛夏也很好。
上几张图，不分先后顺序。










骑到Hirschhorn我们就吃午饭，之后原路返回海德堡，在主街1号(Hauptstrasse 1)的Eiscafe Venizia喝冰咖啡和吃Apfelstrudel。话说这里的Apfelstrudel是我吃过最好的， 皮特别薄。

(下图：我们的下午茶）
